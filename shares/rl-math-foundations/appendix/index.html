Appendix · Book pp. 237-272

附录:数学预备 (Mathematical Preliminaries)

关于翻译策略:本系列正文使用中文 + 英文术语保留,但附录的数学预备(概率论 / 线性代数 / 凸优化基础)不在本系列翻译范围内——这些是工科常识性内容,中文优秀教材已经很多。本页只列附录的子主题与原书页码,需要回顾时直接翻原 PDF。

附录子主题(原书 pp. 237-272)

  1. Probability theory basics(概率论基础)— 教材 §A.1,pp. 237-244。覆盖事件 / 条件概率 / 期望 / 方差 / 大数定律 / 中心极限定理。
  2. Stochastic processes(随机过程)— 教材 §A.2,pp. 245-251。重点是 Markov chain 的基本性质、平稳分布、收敛速度——本书 ch.6 (RM) 与 ch.7 (TD) 多次回引。
  3. Linear algebra essentials(线性代数要点)— 教材 §A.3,pp. 252-260。矩阵特征值 / 谱半径 / 范数等价 / 收缩映射定理(Banach Fixed-Point Theorem)——这是 ch.3 BOE 收敛证明的关键引理。
  4. Optimization preliminaries(优化基础)— 教材 §A.4,pp. 261-268。凸函数 / 一阶/二阶条件 / 梯度下降收敛性——ch.6/ch.9 反复用到。
  5. Useful inequalities(常用不等式)— 教材 §A.5,pp. 269-272。Jensen / Hoeffding / Markov / Chebyshev——ch.5 (MC) 的样本复杂度分析依赖这些。

什么时候需要回这里

正文大多数推导都是自包含的,但以下几个高密度数学环节强烈建议先回查附录:

  • 读 ch.3 BOE 的 压缩映射存在唯一性证明 之前,补 §A.3 收缩映射定理。
  • 读 ch.6 Robbins-Monro 的 步长条件 \(\sum \alpha_t = \infty,\ \sum \alpha_t^2 < \infty\) 之前,补 §A.2 随机过程收敛。
  • 读 ch.9 策略梯度定理 的 对数导数技巧 之前,补 §A.1 期望与梯度的可交换条件。
