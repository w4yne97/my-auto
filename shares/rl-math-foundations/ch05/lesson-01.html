L1: Mean Estimation as the Seed of MC (蒙特卡罗动机:均值估计)

§1 Why this lesson

Ch.4 L3 §6 已埋下伏笔:VI / PI / Truncated PI 三个算法都假设模型已知 —— \(p(r\mid s, a),\ p(s'\mid s, a)\) 显式给定。但真实 RL 问题里(Atari、机器人、推荐系统)模型几乎永远不可见。没有 \(p\),\(\mathbb E_\pi[\cdot]\) 怎么算?本节(教材 §5.1,pp. 77-79)从最朴素的均值估计 问题切入,给出整个 Ch.5 的灵魂答案 —— "采样多次取平均"。

关键洞察(教材 §5.1 + Box 5.1):(a) 哲学(p. 77 末段)"If we do not have a model, we must have some data; if we do not have data, we must have a model" —— RL 算法只可能在这两个端点之间;模型已知 → DP(Ch.4),数据已知 → MC(Ch.5)。(b) 两条估期望路线:model-based \(\mathbb E[X] = \sum_x p(x) x\) 用 PMF 闭式求;model-free \(\mathbb E[X] \approx \bar x = \frac{1}{n}\sum x_j\) 用样本均值。(c) 大数定律 (Box 5.1,p. 80):i.i.d. 样本均值是无偏 \(\mathbb E[\bar x] = \mathbb E[X]\),方差 \(\text{var}(\bar x) = \text{var}(X) / n\) 随 \(n\) 减到零。(d) 关键纽带:state value 与 action value 本身就是 return 的期望(\(v_\pi(s) = \mathbb E[G_t \mid S_t = s]\)),所以"估值" = "估期望" = "采样均值"。这把 Ch.4 的"规划" 切到 Ch.5 的"学习",model-free RL 由此诞生。

§2 核心定义(直觉先行)

  • 蒙特卡罗 (Monte Carlo, MC):用随机采样 解估计问题 的一大类技术。RL 里特指"用采样到的 episode 估计 \(v_\pi, q_\pi\)"。命名源自摩纳哥蒙特卡罗赌场(随机性的代名词)。
  • Episode (轨迹):agent 跟着策略 \(\pi\) 与环境交互产生的有限序列 \((s_0, a_0, r_1, s_1, a_1, r_2, \dots, s_T)\)。MC 要求 episode 跑到终止,这样才能算出完整 return \(G_t = r_{t+1} + \gamma r_{t+2} + \dots\)。
  • Sample mean (样本均值):\(\bar x = \frac{1}{n}\sum_{j=1}^n x_j\)。用来近似 \(\mathbb E[X]\) 的最朴素估计。
  • 大数定律 (Law of large numbers,Box 5.1,p. 80):i.i.d. 时 \(\mathbb E[\bar x] = \mathbb E[X]\)(无偏),\(\text{var}(\bar x) = \text{var}(X)/n\)(方差随 \(n\) 衰减为零)。这是 MC 收敛的数学根。
  • i.i.d. (独立同分布):样本之间不能相关(独立),也必须来自同一个分布。教材末段警告 —— 如果样本相关,样本均值可能永远 估不准(极端例子:全部 \(x_j\) 都等于第一个)。这是 MC 实操里 episode 之间要重新采样的根本原因。
  • Model-based vs model-free:前者依赖 \(p(\cdot)\) 解析求期望(Ch.4);后者只看 \(\{x_1, \dots, x_n\}\) 这一坨样本,什么 PMF 都不知道(Ch.5 起)。核心切换:planning → learning。
  • "为什么 RL 关心 mean estimation"(教材 §5.1 第二段):state/action value 本质就是return 的期望,\(v_\pi(s) = \mathbb E[G_t \mid S_t = s]\)。估值 = 估期望 = MC 估计。所以本节看似在讲投硬币,其实是把 RL 的核心算子("\(\mathbb E\)")交给样本。

Step 1. 抛硬币例子(教材 p. 79)。\(X = +1\)(正面)、\(X = -1\)(反面),\(p(+1) = p(-1) = 0.5\)。Model-based:\(\mathbb E[X] = 0.5 \cdot 1 + 0.5 \cdot (-1) = 0\)。Model-free:抛 \(n\) 次,记下 \(\{x_1, \dots, x_n\}\),算 \(\bar x\)。教材 Figure 5.2:\(n = 5\) 时 \(\bar x\) 还在 ±0.4 抖,\(n = 200\) 时已经几乎贴在 0 上。这就是大数定律的可视化。
Step 2. "为什么 RL 关心这个" 的桥接。回忆 \(v_\pi(s) = \mathbb E[G_t \mid S_t = s]\)、\(q_\pi(s, a) = \mathbb E[G_t \mid S_t = s, A_t = a]\)。两者都是return 的期望。如果我们能采样 \(n\) 条 episode 拿到 \(n\) 个 \(G_t^{(i)}\),那么 \(\bar G = \frac{1}{n}\sum G_t^{(i)} \approx v_\pi(s)\)。"估期望" 就是"估值"。这把 mean estimation 直接搬进 RL —— Ch.5 L2 的 MC Basic 公式 (5.2) 就是这一搬。
Step 3. i.i.d. 的实操含义。教材末段:若样本不独立,样本均值可能永远偏。RL 里 episode 内部的 \((s_t, a_t)\) 是有相关性 的(同一条 trajectory 上 \(s_{t+1}\) 由 \(s_t\) 决定)。所以 MC 用"多条 独立 episode 的 return" 而不是"一条 episode 内部多次抽样" 当作 i.i.d. 样本。这一点在 L4 Exploring Starts 与 L5 ε-greedy 里会重新遇到 —— "怎么让 (s, a) 都被独立访问" 是整章后半的主线。

§4 Grid-world:同一策略下的多条采样轨迹

本节没有"已知最优策略 π* 然后画箭头" 的需求 —— 关键直觉是"同一策略,跑两次会得到不同轨迹",这是采样的本质。我们用一个简单初始策略 \(\pi_0\)(部分箭头,未优化),展示三条 不同的采样轨迹。注意三条 trajectory 都不同 —— 这正是"样本" 的味道。

Episode 1:从 (0,0) 起,沿左侧绕一圈到 target (3,2)。\(G_0\) = 0 + 0γ + 0γ² + 0γ³ + 0γ⁴ + 1·γ⁵ = γ⁵ ≈ 0.59(γ=0.9)。

Episode 2:从 (0,0) 起,沿顶部和右侧绕,在 (3,3) forbidden 上踩了一脚 \(r = -10\),再到 target。\(G_0\) ≈ -10·γ⁸ + 1·γ⁹ ≈ -3.87。可以看出: 同一起点、同一策略族,两条 trajectory 给出差异巨大 的 \(G_0\)。这就是"方差" 的来源。

Episode 3:从 (0,0) 起,沿左侧+底部绕到 target。\(G_0 = γ^7 ≈ 0.48\)。

样本均值估计 \(v_\pi(s_0)\):三条 trajectory 给 \(G_0 \in \{0.59, -3.87, 0.48\}\)。\(\bar G = -0.93\)。三条样本 显然不够 —— 真正的 \(v_\pi(s_0)\) 需要把 forbidden 踩入概率、随机性都吃透,通常需要几百到几千条。但这就是"用样本估期望" 的雏形。下面给出 \(\bar G\) 在 \(n\) 上的演化(示意),与教材 Figure 5.2 同款"抖→稳" 形态。

§5 常见误解 / 易混概念

很多人以为:"既然样本均值收敛到期望,那 MC 算 \(v_\pi\) 是免费 的 —— 跑几条 episode 取平均就好。"
实际上:i.i.d. 假设有强约束,而且方差 衰减只有 \(O(1/\sqrt n)\)(标准误)。要把 \(\hat v\) 误差从 0.1 降到 0.01,样本数要乘 100。且 "跑一条 episode" ≠ "n 个独立样本" —— episode 内部 \((s_t, a_t)\) 序列高度相关。MC 实操里要么"多条独立 episode",要么"一条很长的 episode + soft 策略"(L5 ε-greedy)。
因为:大数定律的细则被 RL 实践严格要求。Ch.6 RM/SGD 会把"(1/n) 这个权" 推广为一般步长 \(\alpha_k\),并给出收敛条件 \(\sum \alpha_k = \infty,\ \sum \alpha_k^2 < \infty\)。这是 MC 与 TD 收敛证明的共同根。
很多人以为:"MC 完全抛弃了 \(p(s'\mid s, a)\),所以 MC 比 DP 普适。"
实际上:MC 仍然要求能采样,即环境本身 必须可交互(simulator 或真实环境)。否则连 \(\{x_1, \dots, x_n\}\) 都拿不到。"Data 没有,model 没有" 的极端情况谁也救不了。教材 p. 77 末段直说:"If we have neither, then we are not able to find optimal policies"。
因为:"model-based" 与 "model-free" 不是优劣对比,而是两条互补 的信息来源。Ch.5-Ch.10 开始基本走 model-free 路线,但 Ch.10 末尾会回到"能不能学一个模型再用"(model-based RL 系列,本书未深入,但是研究热点)。

§6 自检 + 视频 + 下一节

Q1. 公式 \(\mathbb E[X] \approx \bar x = \frac{1}{n}\sum x_j\) 中,如果样本不是 i.i.d.,会有什么后果?用一个极端例子说明。

A. 教材末段直接给:若所有样本完全相关(例如硬币每次都抛出和第一次一模一样的结果),那么 \(\bar x\) 永远 = 第一个样本,与 \(\mathbb E[X]\) 可以相差任意大(若第一次抛到 +1,\(\bar x = 1 \ne 0\))。这告诉我们:"样本数多" 不等价于"信息多",只有"独立样本数多" 才有大数定律保证。RL 实操含义:同一条 episode 内部相邻 step 高度相关,所以 MC 偏好"多条独立 episode";一条超长 episode 也能用,但要靠"策略 soft 让其自相关弱"(L5 ε-greedy)。

Q2. 大数定律里 \(\text{var}[\bar x] = \text{var}[X]/n\)。这告诉我们 \(\bar x\) 的标准误(标准差)是 \(\sigma / \sqrt n\)。要把估计精度从 0.1 提到 0.01,需要把 \(n\) 乘多少倍?

A. 100 倍。标准误 \(\sigma / \sqrt n\) 缩 10 倍 ⇒ \(\sqrt n\) 涨 10 倍 ⇒ \(n\) 涨 100 倍。这就是 MC 在 RL 里"方差大、收敛慢" 的根 —— 估计每多一位准度,样本数 ×100。Ch.7 TD 用 bootstrap 把单步样本换成"当前估值的部分信息",大幅减少方差,这是 TD 比 MC 高效的关键。

Q3. RL 里 \(v_\pi(s) = \mathbb E[G_t \mid S_t = s]\) 的"期望" 是对什么随机性求的?为什么从同一个 \(s\) 出发跑两次会得到不同 \(G_t\)?

A. 期望是对整条 trajectory 的随机性求的:动作 \(a_t \sim \pi(\cdot \mid s_t)\)(若 \(\pi\) 随机)+ 下一状态 \(s_{t+1} \sim p(\cdot \mid s_t, a_t)\)(若环境随机)+ 奖励 \(r_{t+1} \sim p(\cdot \mid s_t, a_t)\)。从同一 \(s_0\) 出发的两次 rollout,即使策略相同,因 \(p\) 与 \(\pi\) 的随机抽样,\(s_1, s_2, \dots\) 的具体值就不同 ⇒ \(G_0\) 不同。MC 正是抓住"一条 trajectory 上的 \(G_t\) 是随机变量",用样本均值近似它的期望。
