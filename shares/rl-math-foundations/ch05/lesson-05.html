L5: ε-Greedy Policy Family (ε-贪心策略 + 介绍)

§1 Why this lesson

L4 的 MC Exploring Starts 必须 "从每个 (s, a) 起跑充分多 episode",这条假设在真实环境(机器人、推荐、自动驾驶)几乎不成立。本节(教材 §5.4.1-5.4.2,pp. 89-91)给出根本性替换 —— ε-greedy 策略:让 \(\pi(a \mid s) > 0\) 对所有 a,这样一条充分长 episode 就能 visit 每个 \((s, a)\) 多次,不再需要 exploring starts。

关键洞察(教材 §5.4.1-5.4.2):(a) Soft policy 概念(p. 89):任何 \(\pi(a \mid s) > 0\) 对所有 \((s, a)\) 都成立的策略叫 soft;反之是 deterministic / greedy。Soft 策略的关键性质:一条充分长 episode 能 visit 所有 \((s, a)\),所以"多条 exploring-start 短 episode" 可换成"一条 soft 长 episode"。(b) ε-greedy 公式(p. 89 末):\(\pi(a \mid s) = 1 - \frac{|\mathcal A(s)| - 1}{|\mathcal A(s)|}\varepsilon\) 给 greedy action,\(\frac{\varepsilon}{|\mathcal A(s)|}\) 给其他 action。关键:greedy action 总比其他 action 概率更高(\(1 - \varepsilon + \varepsilon/|\mathcal A| \ge \varepsilon/|\mathcal A|\) 对 \(\varepsilon \in [0, 1]\) 成立)。(c) ε 端点:\(\varepsilon = 0\) ⇒ deterministic greedy(回到 L4 的强假设依赖);\(\varepsilon = 1\) ⇒ uniform random(\(\pi(a \mid s) = 1/|\mathcal A|\) 对所有 a),最强探索但完全不利用估值;\(\varepsilon \in (0, 1)\) 是连续光谱。(d) 实操采样(p. 90 中段):生成随机数 \(x \sim U[0, 1]\),若 \(x \ge \varepsilon\) 选 greedy(概率 \(1 - \varepsilon\)),否则均匀随机选(再次以 \(1/|\mathcal A|\) 选,可能再次撞中 greedy)。这是工程上常见的两步实现。(e) 整合到 MC(§5.4.2):把"improvement step 用 \(\arg\max_a q\)" 改成"用 ε-greedy 改"。policy improvement 的求解空间从"所有 \(\Pi\)" 缩到"ε-greedy 子集 \(\Pi_\varepsilon\)"。本节讲分布形态与 policy improvement 公式;L6 落地完整算法 + 全章总结。

§2 核心定义(直觉先行)

  • Soft policy (软策略,p. 89):对所有 \((s, a)\),\(\pi(a \mid s) > 0\)。即每个 action 都有正概率被选。性质:一条充分长 episode 能 visit 所有 \((s, a)\),省掉 exploring starts。
  • ε-greedy 策略(p. 89 末):soft 策略的特例。\(\pi(a \mid s) = 1 - \frac{|\mathcal A(s)| - 1}{|\mathcal A(s)|}\varepsilon\) 给 greedy(\(a = a^*\)),\(\frac{\varepsilon}{|\mathcal A(s)|}\) 给其他。\(\varepsilon \in [0, 1]\) 控制"探索 vs 利用" 平衡。
  • ε 的端点意义:\(\varepsilon = 0 \Rightarrow\) greedy(L4 的 \(\arg\max\));\(\varepsilon = 1 \Rightarrow\) uniform 随机(\(1/|\mathcal A|\) 对所有 a),完全不利用估值。实操:\(\varepsilon \in [0.05, 0.3]\) 是常见取值,深度学习里还有 ε-decay 调度。
  • "greedy 概率 ≥ 其他概率" 的不等式(p. 90 顶):\(1 - \varepsilon + \varepsilon/|\mathcal A| \ge \varepsilon/|\mathcal A|\) 对 \(\varepsilon \in [0, 1]\) 恒成立。直觉:ε-greedy 总是 偏向 greedy action,绝不退化为 uniform 之外的形式 —— ε=1 时恰好 等于 uniform(此时 greedy 与其他都是 \(1/|\mathcal A|\))。
  • 采样实现(p. 90 中段):抽 \(x \sim U[0, 1]\)。若 \(x \ge \varepsilon\) 直接选 greedy 动作 \(a^*\);若 \(x < \varepsilon\) 在 \(\mathcal A\) 上均匀抽(可能再撞 \(a^*\))。等价于上面分布公式。
  • 策略空间 \(\Pi\) vs \(\Pi_\varepsilon\):\(\Pi\) = 所有合法策略(含 deterministic 与 stochastic);\(\Pi_\varepsilon\) = 满足 ε-greedy 形式的策略子集(给定 \(\varepsilon\))。注:\(\Pi_\varepsilon \subset \Pi\),且 \(\varepsilon \to 0\) 时 \(\Pi_\varepsilon\) 退化到 deterministic 策略集。
  • "policy improvement 的搜索空间收窄"(§5.4.2):原 PI 的 improvement 解 \(\arg\max_{\pi \in \Pi}\)(任意策略空间);ε-greedy MC 改解 \(\arg\max_{\pi \in \Pi_\varepsilon}\)。代价:最优解 \(\pi^*_\varepsilon \in \Pi_\varepsilon\) 比真 \(\pi^* \in \Pi\) 略差;补偿:不再需要 exploring starts。

公式 2:ε-greedy 在 \(\Pi_\varepsilon\) 内的 improvement(教材 (5.5),§5.4.2)

Step 1. 验证 \(\sum_a \pi(a \mid s) = 1\)。\(|\mathcal A|\) 个动作,1 个 greedy + \((|\mathcal A| - 1)\) 个其他。\((1 - \frac{|\mathcal A| - 1}{|\mathcal A|}\varepsilon) + (|\mathcal A| - 1) \cdot \frac{\varepsilon}{|\mathcal A|} = 1 - \frac{(|\mathcal A| - 1)\varepsilon}{|\mathcal A|} + \frac{(|\mathcal A| - 1)\varepsilon}{|\mathcal A|} = 1\)。✓
Step 2. 验证 \(\varepsilon = 0\) 与 \(\varepsilon = 1\) 端点。\(\varepsilon = 0\):greedy 概率 = 1,其他 = 0 ⇒ 纯 greedy(deterministic)。\(\varepsilon = 1\):greedy 概率 = \(1 - (|\mathcal A| - 1)/|\mathcal A| = 1/|\mathcal A|\),其他 = \(1/|\mathcal A|\) ⇒ uniform。即 \(\Pi_{\varepsilon = 0}\) 是 deterministic 策略集,\(\Pi_{\varepsilon = 1}\) 只含 uniform 策略。\(\varepsilon \in (0, 1)\) 是中间。
Step 3. (5.5) 解的形式推导。在 \(\Pi_\varepsilon\) 内最大化 \(\sum_a \pi(a \mid s) q(s, a)\)。已知 ε-greedy 必须给所有 \(a \ne a^*\) 同概率 \(\varepsilon/|\mathcal A|\),自由变量只有"哪个 a 当 a*"。\(\sum_a \pi(a \mid s) q(s, a) = (1 - \frac{|\mathcal A| - 1}{|\mathcal A|}\varepsilon) q(s, a^*) + \frac{\varepsilon}{|\mathcal A|} \sum_{a \ne a^*} q(s, a)\)。第一项最大化要求 \(a^* = \arg\max_a q(s, a)\)(因为 \(1 - \varepsilon + \varepsilon/|\mathcal A| \ge \varepsilon/|\mathcal A|\),把权重最大的位置给 q 最大的动作)。第二项是常数(不依赖 \(a^*\) 的具体选择,只看是否包含每个动作一次)。所以最优 ε-greedy 策略仍由 \(\arg\max_a q\) 决定 greedy 动作 —— 与原 PI 一致,只是不再"独占 100%"。

§4 ε-greedy 分布形态:从 greedy 到 uniform

用 distribution-bar 展示同一个 state \(s\)、5 个动作 \(\{↑, →, ↓, ←, \cdot\}\)(\(|\mathcal A| = 5\))在不同 ε 下的概率分布。设 greedy action \(a^* = ↓\)。

关键观察:四档分布从左到右展现"greedy 概率衰减、其他概率上升" 的连续过渡。(i) ε=0:greedy 独占 1.0,其他 0;(ii) ε=0.1:greedy 0.92(\(1 - 0.1 + 0.1/5 = 0.92\)),其他 0.02(\(0.1/5\));(iii) ε=0.5:greedy 0.6,其他 0.1;(iv) ε=1:全部 0.2 = uniform。这就是教材 §5.4.1 的"探索程度 连续光谱" —— 工程师选 ε 就是选"多探还是多利"。

样本量 vs 估计精度的另一视角。同一 state 上 ε 越大、被采到 greedy 之外动作的概率越大 ⇒ 那些 \(q(s, a)\) 估计样本越多 ⇒ 估计误差越小。下面的 convergence-plot 草图给出"非贪心动作 q 估计误差 vs 样本数 n" 在不同 ε 下的衰减:

数值含义:ε=0 时非贪心动作永远 不被采样,误差永远 = 1(无信息)。ε=0.1 时被采样概率 \(\sim 0.02\) per step,误差按 \(1/\sqrt{n \cdot 0.02}\) 衰减。ε=1 时被采样概率 \(0.2\) per step,误差按 \(1/\sqrt{n \cdot 0.2}\) 衰减,最快但greedy 利用度 最低。

§5 常见误解 / 易混概念

很多人以为:"ε-greedy 就是uniform 加权 的策略 —— ε 概率均匀抽,1-ε 概率贪心抽。"
实际上:ε-greedy 不是 uniform 与 greedy 的简单加权。教材 (p. 89 末) 公式:greedy 动作概率是 \(1 - \frac{|\mathcal A| - 1}{|\mathcal A|}\varepsilon = 1 - \varepsilon + \varepsilon/|\mathcal A|\),不 是 \(1 - \varepsilon\)。其他动作概率是 \(\varepsilon/|\mathcal A|\)。差异关键:在 ε 概率"抽随机" 时,仍可能 抽到 greedy 动作(\(1/|\mathcal A|\) 概率),所以 greedy 动作的总 概率是 \((1 - \varepsilon) + \varepsilon \cdot 1/|\mathcal A| = 1 - \varepsilon + \varepsilon/|\mathcal A|\)。这就是教材的公式。很多人写错的版本:"\(\pi(a^*) = 1 - \varepsilon, \pi(\text{其他}) = \varepsilon / (|\mathcal A| - 1)\)"。这也是 一种 soft 策略,但不是 教材定义的 ε-greedy。
因为:定义差异源于"是否允许第二次抽到 greedy"。教材的实现(p. 90 中段)\(x \sim U[0, 1]\) → \(x \ge \varepsilon\) 选 greedy,\(x < \varepsilon\) 在所有 5 个动作里 uniform 抽 —— 第二步可能再次撞 greedy。这是一个 implementation 细节,但导致两种数学公式略有差异。本书坚持教材定义,所以 ε-greedy 是 \(\Pi_\varepsilon\) 中"greedy 概率最大" 的形式。
很多人以为:"ε 应该选小一些(如 0.05),让策略近乎 greedy 最优。"
实际上:ε 太小会让"非贪心动作 q 估计" 永远不准。看 §4 convergence-plot:ε=0.05 时非贪心动作每 100 步只被采到 \(\sim 1\) 次,误差衰减极慢。如果初始 \(q\) 估计有偏,greedy 动作选错,小 ε 让"纠错信号" 来得太慢。这是探索不足(under-exploration)的典型陷阱 —— 算法会"自信地走错路"。实操:ε 通常从 \(\sim 0.5\) 起,按 episode 数衰减到 \(\sim 0.05\)("ε-decay 调度,L6 §6 提)。
因为:"探索 vs 利用" 的最优解依赖初始 q 估计的不确定性。早期不确定性大 ⇒ 多探;后期 q 已稳 ⇒ 多利用。固定 ε 是粗暴折衷;ε-decay 让算法自适应。Ch.7 Q-learning 默认搭配 ε-decay,Ch.8 DQN(\(\varepsilon = 1.0 \to 0.1\) 退火)是经典实现。
很多人以为:"ε-greedy 在 \(\Pi_\varepsilon\) 内最优,所以等价于 PI 在 \(\Pi\) 内最优。"
实际上:ε-greedy 最优 ≠ 真最优(教材 p. 91 末段)。\(\Pi_\varepsilon \subset \Pi\),所以 \(\pi^*_\varepsilon \in \Pi_\varepsilon\) 一般不等于 \(\pi^* \in \Pi\)。具体差距:\(v_{\pi^*_\varepsilon}(s) \le v_{\pi^*}(s)\) 严格成立,等号只在 ε=0 时成立。L6 §4 会用具体数值(教材 Figure 5.6:ε=0 vs ε=0.5 的 v 值差异)展示这个差距。结论:ε-greedy 是"放弃 strict optimality 换 explorability" 的有意识 trade-off,不是"近似 最优"。
因为:strict optimal 策略一定 deterministic(BOE 定理,Ch.3 L2),但 deterministic 策略 \(\notin \Pi_\varepsilon\) for \(\varepsilon > 0\)。所以 ε-greedy 注定在 \(\Pi_\varepsilon\) 内是受限最优。Ch.10 Actor-Critic 用 stochastic policy + 直接梯度优化,不再受这个限制。

§6 自检 + 视频 + 下一节

Q1. ε=0.5、\(|\mathcal A| = 4\) 时,greedy 动作的概率 = ?其他三个动作各自的概率 = ?

A. Greedy 概率 = \(1 - \frac{|\mathcal A| - 1}{|\mathcal A|}\varepsilon = 1 - \frac{3}{4} \cdot 0.5 = 1 - 0.375 = 0.625\)。等价计算:\(1 - \varepsilon + \varepsilon/|\mathcal A| = 1 - 0.5 + 0.125 = 0.625\)。其他每个动作的概率 = \(\varepsilon / |\mathcal A| = 0.5 / 4 = 0.125\)。验证:\(0.625 + 3 \cdot 0.125 = 0.625 + 0.375 = 1\)。✓

Q2. 为什么 ε=1 时 ε-greedy 退化为 uniform 而不是 "给非贪心动作 100% 概率"?

A. 看公式:ε=1 时 greedy 概率 = \(1 - \frac{|\mathcal A| - 1}{|\mathcal A|} = \frac{1}{|\mathcal A|}\),其他每个动作概率 = \(\frac{1}{|\mathcal A|}\)。所有 动作同概率 ⇒ uniform。直觉解释:实现层面 ε=1 意味着"100% 概率执行随机抽",随机抽包括 greedy 动作 —— 所以 greedy 仍有 \(1/|\mathcal A|\) 概率被选,与其他动作平等。这就是为什么教材 ε-greedy 公式在 ε=1 时退化为 uniform 而不是"反贪心"。

Q3. ε-greedy 是 soft 策略族里的一个特例。其他常见 soft 策略形式有哪些?

A. 三类常见:(1) Boltzmann / softmax 策略:\(\pi(a \mid s) = \exp(q(s, a) / \tau) / \sum_b \exp(q(s, b) / \tau)\),温度 \(\tau\) 控制探索。\(\tau \to 0\) 退化 greedy,\(\tau \to \infty\) 退化 uniform。比 ε-greedy 更光滑,概率随 q 值连续变化,但需要 q 数值排序与差距 都准确(对 noise 敏感)。(2) Stochastic policy with entropy bonus:Ch.10 SAC 等算法用 \(J(\pi) + \beta H(\pi)\) 形式,鼓励高熵(强探索)。(3) UCB (Upper Confidence Bound):bandit 文献的经典,\(\arg\max_a [q(s,a) + c\sqrt{\log n / N(s,a)}]\),给"访问少" 的动作 bonus。RL 里 MCTS(AlphaGo)用 PUCT 变种。本书选 ε-greedy 是因为它最简单,且足以让所有 (s, a) 被覆盖 —— 收敛理论清晰,Ch.7 Q-learning 直接复用。
