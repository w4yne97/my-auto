L3: Bellman Optimality Equation (P3 - More on BOE) (贝尔曼最优方程(P3):压缩映射与求解)

§1 Why this lesson

L2 给出了 BOE \(v = f(v) = \max_\pi (r_\pi + \gamma P_\pi v)\),并指出它是非线性不动点方程。但还差三个根本问题:(1) 解存在吗?(2) 解唯一吗?(3) 怎么算?本节(教材 §3.3.3-3.4,pp. 40-49)用一个工具一次性回答全部 —— 压缩映射定理(Banach fixed-point theorem)。

策略是这样的:先证 \(f(v)\) 在最大范数 \(\|\cdot\|_\infty\) 下是 \(\gamma\)-Lipschitz,即 \(\|f(v_1) - f(v_2)\|_\infty \le \gamma \|v_1 - v_2\|_\infty\)(教材 Theorem 3.2)。这一性质就叫"contraction"(收缩)。然后调用压缩映射定理(教材 Theorem 3.1,严格证明在附录 §A.3):任何收缩映射都有唯一不动点 \(v^*\),且迭代 \(v_{k+1} = f(v_k)\) 从任意 \(v_0\) 出发指数收敛到 \(v^*\),收敛率刚好等于 \(\gamma\)。再加上 §3.4 的 Theorem 3.4(\(v^* \ge v_\pi\) 对所有 \(\pi\) 成立)与 Theorem 3.5(确定性贪心 \(\pi^*(s) = \arg\max_a q^*(s, a)\) 是最优策略),L1 / L2 留下的所有问题就都被堵住:\(v^*\) 唯一存在;\(\pi^*\) 一定存在但未必唯一;一定有确定性贪心版本的 \(\pi^*\);算法就是 \(v_{k+1} = f(v_k)\)(下章会叫它value iteration)。本节是 Ch.3 的数学密度峰值:压缩证明(\(\gamma\)-Lipschitz)、Banach 三个结论、Theorem 3.4 的 \(\gamma^n P_\pi^n \to 0\) 极限论证 —— 一步都不能跳。读懂之后,Ch.4 VI / Ch.7 TD 的"为什么收敛" 都迎刃而解。

§2 核心定义(直觉先行)

  • 不动点 (fixed point,教材 §3.3.3,p. 41):对函数 \(f: \mathbb{R}^d \to \mathbb{R}^d\),若存在 \(x^* \in \mathbb{R}^d\) 使 \(f(x^*) = x^*\),则称 \(x^*\) 是 \(f\) 的不动点。"\(f\) 把 \(x^*\) 映成自己" —— 这就是"不动"的字面含义。
  • 压缩映射 (contraction mapping,教材 §3.3.3,p. 41):函数 \(f: \mathbb{R}^d \to \mathbb{R}^d\) 是压缩映射,若存在 \(\gamma \in (0, 1)\) 使 \(\|f(x_1) - f(x_2)\| \le \gamma \|x_1 - x_2\|\) 对所有 \(x_1, x_2\) 成立。关键:必须 \(\gamma\) 严格小于 1(\(\gamma < 1\),非 \(\gamma \le 1\)),否则定理不成立。
  • 压缩映射定理 (contraction mapping theorem / Banach fixed-point theorem,Theorem 3.1,p. 41):若 \(f\) 是压缩映射,则:(a) 存在性:存在不动点 \(x^*\) 满足 \(f(x^*) = x^*\);(b) 唯一性:\(x^*\) 唯一;(c) 算法 + 指数收敛:迭代 \(x_{k+1} = f(x_k)\) 从任意 \(x_0\) 出发,\(x_k \to x^*\) 指数级,\(\|x_k - x^*\| \le \gamma^k \|x_0 - x^*\|\) (实际证明给出更紧界 \(\|x^* - x_n\| \le \dfrac{\gamma^n}{1-\gamma} \|x_1 - x_0\|\))。这个定理本身的严格证明详见附录 §A.3 —— 用 Cauchy 列收敛性 + 完备空间的标准分析论证。
  • BOE 右端的压缩性(Theorem 3.2,p. 44):\(f(v) \doteq \max_\pi (r_\pi + \gamma P_\pi v)\) 对任意 \(v_1, v_2 \in \mathbb{R}^{|\mathcal{S}|}\) 满足 \(\|f(v_1) - f(v_2)\|_\infty \le \gamma \|v_1 - v_2\|_\infty\)。这里 \(\|\cdot\|_\infty\) 是最大范数 \(\|v\|_\infty = \max_i |v_i|\)。证明思路:利用 \(P_\pi\) 是行随机矩阵(每行概率分布求和为 1),\(\|P_\pi w\|_\infty \le \|w\|_\infty\) 对任意向量 \(w\) 成立,再处理 \(\max_\pi\) 算子。
  • BOE 解的存在唯一性(Theorem 3.3,p. 46):把 Theorem 3.2 代入 Theorem 3.1 即得:BOE \(v = f(v)\) 有唯一解 \(v^*\),且迭代 \(v_{k+1} = \max_\pi (r_\pi + \gamma P_\pi v_k)\) 从任意 \(v_0\) 出发指数收敛到 \(v^*\)。这个迭代过程在 Ch.4 叫作 value iteration,算法实现细节延后到 Ch.4。
  • 从 \(v^*\) 算 \(\pi^*\)(Theorem 3.5,p. 47-48):一旦 \(v^*\) 已知,定义 \(q^*(s, a) \doteq \sum_r p(r\mid s, a) r + \gamma \sum_{s'} p(s'\mid s, a) v^*(s')\),则确定性贪心策略 \(\pi^*(a\mid s) = 1\) 当 \(a = \arg\max_a q^*(s, a)\)、否则为 0,是 BOE 的最优策略。关键:\(\pi^*\) 不必唯一(若多个 \(a\) 并列 \(\arg\max\)),但确定性版本一定存在 —— 这是 Theorem 3.5 的内容。
  • \(v^*\) 是真正的"最优状态值"(Theorem 3.4,p. 47):解出来的 \(v^*\) 满足 \(v^*(s) \ge v_\pi(s)\) 对所有 \(s\) 与所有 \(\pi\) 成立 —— 即 \(\pi^* = \arg\max_\pi (r_\pi + \gamma P_\pi v^*)\) 真的就是L2 Definition 3.1 意义下的最优策略。证明用 \(v^* - v_\pi \ge \gamma P_\pi (v^* - v_\pi) \ge \gamma^n P_\pi^n (v^* - v_\pi) \to 0\)(\(\gamma^n \to 0\) 且 \(P_\pi^n\) 有界)。

Step 1. 把 \(f(v_1) - f(v_2)\) 用 \(\arg\max\) 写出(Box 3.2 关键一步)。设 \(\pi_1^* \doteq \arg\max_\pi (r_\pi + \gamma P_\pi v_1)\)、\(\pi_2^* \doteq \arg\max_\pi (r_\pi + \gamma P_\pi v_2)\)(分别是 \(v_1, v_2\) 上贪心策略)。则 \(f(v_1) = r_{\pi_1^*} + \gamma P_{\pi_1^*} v_1\) 且 \(f(v_2) = r_{\pi_2^*} + \gamma P_{\pi_2^*} v_2\)。关键:由 \(\arg\max\) 的定义,\(f(v_1) \ge r_{\pi_2^*} + \gamma P_{\pi_2^*} v_1\)(因为 \(\pi_2^*\) 是 \(v_2\) 上的贪心,在 \(v_1\) 上不一定更优),同理 \(f(v_2) \ge r_{\pi_1^*} + \gamma P_{\pi_1^*} v_2\)。两式相减得:\(f(v_1) - f(v_2) \le (r_{\pi_1^*} + \gamma P_{\pi_1^*} v_1) - (r_{\pi_1^*} + \gamma P_{\pi_1^*} v_2) = \gamma P_{\pi_1^*}(v_1 - v_2)\)。同理 \(f(v_2) - f(v_1) \le \gamma P_{\pi_2^*}(v_2 - v_1)\)。两式合起来得到逐元素夹逼 \(\gamma P_{\pi_2^*}(v_1 - v_2) \le f(v_1) - f(v_2) \le \gamma P_{\pi_1^*}(v_1 - v_2)\)(注意符号翻转),即 \(|f(v_1) - f(v_2)| \le \max\{|\gamma P_{\pi_1^*}(v_1 - v_2)|, |\gamma P_{\pi_2^*}(v_1 - v_2)|\}\) 逐元素成立。这是 Box 3.2 的核心代数操作。
Step 2. 用 \(P_\pi\) 行随机性把 \(\|P_\pi w\|_\infty \le \|w\|_\infty\) 取出。对任意行 \(p_i^\top\)(\(P_\pi\) 第 \(i\) 行,\(p_i \ge 0\) 且 \(\sum_j [p_i]_j = 1\)),有 \(|p_i^\top w| = |\sum_j [p_i]_j w_j| \le \sum_j [p_i]_j |w_j| \le \sum_j [p_i]_j \|w\|_\infty = \|w\|_\infty\)(第一步三角不等式;第二步用 \(|w_j| \le \|w\|_\infty\);第三步用行和为 1)。所以 \(\|P_\pi w\|_\infty = \max_i |p_i^\top w| \le \|w\|_\infty\)。这是 Ch.2 L4 §3 那个"行随机矩阵谱半径 \(\le 1\)" 论证的"\(\|\cdot\|_\infty\) 版本",更直接、不需要谱半径。
Step 3. 合成出 Theorem 3.2 的结论。把 Step 1 与 Step 2 接起来:\(\|f(v_1) - f(v_2)\|_\infty \le \max\{\|\gamma P_{\pi_1^*}(v_1 - v_2)\|_\infty, \|\gamma P_{\pi_2^*}(v_1 - v_2)\|_\infty\} \le \gamma \|v_1 - v_2\|_\infty\)(用 Step 2 把 \(\|P_\pi (v_1 - v_2)\|_\infty\) 上界压住)。结论:\(f\) 在 \(\|\cdot\|_\infty\) 下是 \(\gamma\)-Lipschitz,即压缩映射。注意 \(\gamma\) 这个收缩因子就是 MDP 的折扣率 —— 这把"为什么 RL 需要 \(\gamma < 1\)" 的真相讲明白了:没有 \(\gamma < 1\),BOE 就不再是压缩映射,Banach 定理整套理论塌陷。这也是 Ch.1 把 \(\gamma \in (0, 1)\) 写进 MDP 五元组的根本动机。
Step 4. 用 Banach 定理(教材 Theorem 3.1,严格证明在附录 §A.3)收尾。Banach 定理告诉我们:任何 \(\gamma\)-Lipschitz contraction \(f\) 在完备度量空间 \(\mathbb{R}^{|\mathcal{S}|}\)(配最大范数)上 (a) 存在不动点 \(v^*\)、(b) 唯一、(c) 迭代 \(v_{k+1} = f(v_k)\) 从任意 \(v_0\) 出发都指数收敛 \(\|v^* - v_n\|_\infty \le \dfrac{\gamma^n}{1-\gamma} \|v_1 - v_0\|_\infty\)(教材 (3.4))。具体证明的核心是把 \(\{v_k\}\) 证成 Cauchy 列(用 Step 1-3 给的 \(\gamma\)-Lipschitz 性质),再用 \(\mathbb{R}^{|\mathcal{S}|}\) 的完备性得极限。\(\mathbb{R}^{|\mathcal{S}|}\) 在最大范数下是 Banach 空间(完备赋范向量空间),所以前提满足,结论照单全收。详细完整证明请见附录 §A.3,本节聚焦"为什么 \(f\) 满足该定理的前提"。

§5 常见误解 / 易混概念

很多人以为:"压缩就是\(\|f(x_1) - f(x_2)\| < \|x_1 - x_2\|\),严格小于即可。"
实际上:需要存在统一的 \(\gamma < 1\) 让上界对所有 \((x_1, x_2)\) 都成立 —— 不是逐对都"小于" 就行。逐对严格小于但 \(\sup_{x_1 \ne x_2} \dfrac{\|f(x_1) - f(x_2)\|}{\|x_1 - x_2\|} = 1\) 的映射(non-expansive 但非压缩)不满足 Banach 定理:它的不动点可能不存在或不唯一,迭代可能不收敛。
因为:Banach 证明里需要把 \(\|x_{m} - x_n\| \le \dfrac{\gamma^n}{1-\gamma} \|x_1 - x_0\|\) 这条几何级数收敛性建立起来,这要求 \(\gamma\) 严格小于 1 的统一上界。"逐对严格小于"的版本叫contractive on each pair,是更弱的条件,Banach 定理在其上失效。本节例 \(f(v) = \max_\pi (r_\pi + \gamma P_\pi v)\) 满足的是统一 \(\gamma\)-Lipschitz —— 这比"严格小于" 强,所以 Banach 定理可用。
很多人以为:"BOE 既然有唯一解 \(v^*\),那 \(\pi^*\) 也就唯一了。"
实际上:\(v^*\) 唯一,\(\pi^*\) 未必唯一(教材 §3.4 末段明确指出)。具体来说,若某 \(s\) 上 \(\arg\max_a q^*(s, a)\) 有多个动作并列(\(q^*(s, a_1) = q^*(s, a_2) = \max\)),则任何在 \(\{a_1, a_2\}\) 上分布的策略都是最优的 —— 包括 \(\pi^*(a_1\mid s) = 1\)、\(\pi^*(a_2\mid s) = 1\)、以及 \(\pi^*(a_1\mid s) = \pi^*(a_2\mid s) = 0.5\) 等。教材 Figure 3.3 给了这种"多 \(\pi^*\)" 的可视化例子。但 Theorem 3.5 保证确定性贪心版本一定存在,所以"\(\pi^*\) 未必唯一,但确定性 \(\pi^*\) 一定存在"。
因为:\(v^*\) 是 \(f\) 的不动点(数值表),\(\pi^*\) 是 \(\arg\max\) 的argument 表;一个数值表对应可能多个 \(\arg\max\) 选择 —— 这两层的"唯一性"区别不能模糊。Ch.4 PI 实现里"如何在 tie 情况下选 \(\pi^*\)"是一个常见 detail。
很多人以为:"\(\gamma\) 越大,\(v^*\) 越大;那 \(\gamma \to 1\) 应该让 \(v^*\) 最大化,这才是'最优决策'。"
实际上:\(\gamma \to 1\) 时:(a) BOE 不再是严格压缩映射(\(\gamma = 1\) 是 non-expansive 的边界),Banach 定理失效,\(v^*\) 可能不存在或不唯一;(b) 即使存在,\(v^*\) 量级会爆炸(\(R_\text{max}/(1-\gamma) \to \infty\)),数值计算困难;(c) \(\gamma\) 是"远视程度",不是"算法目标" —— 它是 MDP 五元组的一部分,跟 reward 一起定义了优化目标,不是越大越好。
因为:RL 想优化的目标 \(\max_\pi \mathbb{E}_\pi[\sum_t \gamma^t r_t]\) 由 \(\gamma\) 与 \(r\) 共同决定;调 \(\gamma\) 等于换问题(L4 会用 reward shifting / γ rescaling 把这个观察形式化)。\(\gamma = 1\) 时叫"undiscounted"问题,需要不同的数学工具(平均奖励 / 终止状态),本书 Ch.10 之后才会零散涉及。Ch.3-Ch.10 的所有理论默认 \(\gamma \in (0, 1)\)。

§6 自检 + 视频 + 下一节

Q1. 为什么 BOE 的压缩性证明必须用最大范数 \(\|\cdot\|_\infty\),不能用 \(\|\cdot\|_2\) 或 \(\|\cdot\|_1\)?

A. 关键不等式 \(\|P_\pi w\|_\infty \le \|w\|_\infty\) 用到了 \(P_\pi\) 行随机性(每行非负、和为 1)。在 \(\|\cdot\|_2\) 范数下,\(\|P_\pi w\|_2 \le \|P_\pi\|_2 \|w\|_2\),其中 \(\|P_\pi\|_2 = \sigma_\text{max}(P_\pi)\) 是最大奇异值,可能 \(> 1\)(行随机矩阵在 2-范数下不必是 contractive 的)。例子:\(P = \begin{pmatrix} 1 & 0 \\ 0.5 & 0.5 \end{pmatrix}\) 行随机,但 \(\|P\|_2 \approx 1.118 > 1\)。所以2-范数下 BOE 不必是压缩映射。1-范数下 \(\|P_\pi^\top w\|_1 \le \|w\|_1\) 对列随机矩阵成立,但 \(P_\pi\) 是行随机的,所以 1-范数也不直接适用。最大范数是 BOE 的"天然"度量,这是它在所有 RL 收敛证明里反复出现的根本原因(Ch.4 VI、Ch.7 TD 都用 \(\|\cdot\|_\infty\))。

Q2. 假设 BOE 迭代 \(v_{k+1} = f(v_k)\) 跑了 100 轮,误差 \(\|v_{100} - v^*\|_\infty < 0.01\)。\(\gamma = 0.9\) 与 \(\gamma = 0.99\) 对应的"\(\|v_0 - v^*\|_\infty\) 起始误差" 上界各是多少?

A. 由 \(\|v_k - v^*\|_\infty \le \gamma^k \|v_0 - v^*\|_\infty\) 反推:\(\|v_0 - v^*\|_\infty \le 0.01 / \gamma^{100}\)。\(\gamma = 0.9\):\(0.9^{100} \approx 2.66 \times 10^{-5}\),起始误差上界 \(\approx 376\)。\(\gamma = 0.99\):\(0.99^{100} \approx 0.366\),起始误差上界 \(\approx 0.0273\)。解读:\(\gamma = 0.9\) 的"100 轮把误差降到 0.01" 容许起始误差非常大(376),非常宽容;\(\gamma = 0.99\) 的"100 轮降到 0.01" 只允许起始误差 0.027,几乎要求初值就接近真值。实践含义:\(\gamma\) 接近 1 时迭代法效率骤降,需要更多轮次或更好的初值估计 —— 这就是warm start(L4 §5 说过)的工程意义。

Q3. Theorem 3.4 用了 \(v^* - v_\pi \ge \gamma^n P_\pi^n (v^* - v_\pi) \to 0\) 这个递推。为什么 \(\lim_{n\to\infty} \gamma^n P_\pi^n (v^* - v_\pi) = 0\)?\(P_\pi^n\) 这个序列本身会发散吗?

A. \(P_\pi^n\) 是行随机矩阵的幂,仍然是行随机矩阵 —— 它的元素都在 \([0, 1]\) 范围内,不会发散。极限 \(\lim_n P_\pi^n\) 存在(若 \(P_\pi\) 有极限)且行和仍为 1,但具体值不重要 —— 关键是 \(\|P_\pi^n w\|_\infty \le \|w\|_\infty\) 一致有界。所以 \(\|\gamma^n P_\pi^n w\|_\infty \le \gamma^n \|w\|_\infty \to 0\)(因 \(\gamma < 1\))。整套论证的真正核心是 \(\gamma^n \to 0\),\(P_\pi^n\) 的"有界" 是背景条件而非"魔法"。这跟 Ch.2 L4 收敛证明 \(\delta_k = \gamma^k P_\pi^k \delta_0\) 几何衰减是同一套机器,只是这里用在了"\(v^*\) 比 \(v_\pi\) 大"上而不是"迭代误差"上。

下一节预告

这一节我们用 Banach 定理把"BOE 一定有唯一解、迭代一定收敛、收敛速率 \(\gamma\)"全部钉死。但 Ch.3 还差最后一块拼图:\(v^*\) 与 \(\pi^*\) 在更换 MDP 参数时怎么变?具体而言,如果把 \(\gamma\) 调小(变近视),\(\pi^*\) 会怎么变?如果把所有 reward 加一个常数(shifting)或乘一个正数(scaling),\(\pi^*\) 会变吗?\(v^*\) 会怎么变?下一节(L4,Book §3.5-3.7)用Theorem 3.6 (optimal policy invariance under affine transformations) 系统回答这些问题:仿射变换 \(r \to \alpha r + \beta\)(\(\alpha > 0\))保持 \(\pi^*\) 不变,\(v^*\) 按 \(\alpha v^* + \beta/(1-\gamma) \cdot \mathbf{1}\) 变换;而调 \(\gamma\) 是真换问题,\(\pi^*\) 一般会变(短视化)。L4 会用 5×5 grid + γ 表 + 实例数据把这两件事区分清楚 —— 这是后面 Ch.4-Ch.10 调参时不踩坑的基础。
