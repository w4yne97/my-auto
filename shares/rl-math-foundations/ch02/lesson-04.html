L4: Bellman Equation (P4 - Matrix-vector form and solution) (贝尔曼方程(P4):矩阵-向量形式与求解)

§1 Why this lesson

上一节(L3)我们得到了贝尔曼方程的逐元素形式(elementwise form):\(\forall s \in \mathcal{S}\),都有 \(v_\pi(s) = \sum_a \pi(a\mid s)[\sum_r p(r\mid s,a)r + \gamma \sum_{s'} p(s'\mid s,a) v_\pi(s')]\)。问题在于:\(|\mathcal{S}| = 25\) 已经要写 25 条联立方程,真实问题 \(|\mathcal{S}|\) 可达上千上万,逐条写不现实。

本节(教材 §2.6-2.7,pp. 25-30)做两件事:(1) §2.6 把 (2.7) 打包成一行矩阵-向量等式:\(v_\pi = R_\pi + \gamma P_\pi v_\pi\),其中 \(v_\pi, R_\pi\) 是 \(|\mathcal{S}|\) 维向量、\(P_\pi\) 是 \(|\mathcal{S}| \times |\mathcal{S}|\) 概率转移矩阵。这不是新方程,只是记号上的合并。(2) §2.7 给两条求解路径:闭式 \(v_\pi = (I - \gamma P_\pi)^{-1} R_\pi\)(理论用),和迭代 \(v_{k+1} = R_\pi + \gamma P_\pi v_k\)(实践用,Ch.4 价值迭代的雏形)。本节不教线性代数 —— 我们只把矩阵形式当一个干净的"压缩记号"来看,把存在性、可逆性、收敛性这些技术细节扼要带过即可。

§2 核心定义(直觉先行)

  • 策略下的奖励向量 (reward vector under \(\pi\)) \(R_\pi\)(教材 §2.6 公式 (2.8) 下方):\([R_\pi]_i \doteq r_\pi(s_i) = \sum_a \pi(a \mid s_i) \sum_r p(r \mid s_i, a) \, r\)。翻译:在状态 \(s_i\) 上、按策略 \(\pi\) 选动作后,期望即时 reward。这就是 (2.7) 方括号里第一项,被打包成向量的第 \(i\) 个分量。
  • 策略下的状态转移矩阵 (state transition matrix under \(\pi\)) \(P_\pi\)(教材 §2.6 公式 (2.8) 下方):\([P_\pi]_{ij} \doteq p_\pi(s_j \mid s_i) = \sum_a \pi(a \mid s_i) \, p(s_j \mid s_i, a)\)。翻译:从 \(s_i\) 出发、按 \(\pi\) 选动作后,环境把 agent 送到 \(s_j\) 的总概率。这是 (2.7) 方括号里第二项的核心。注意维度:\(P_\pi \in \mathbb{R}^{|\mathcal{S}| \times |\mathcal{S}|}\),行 \(i\) 对应 "从 \(s_i\) 出发",列 \(j\) 对应 "到 \(s_j\)"。
  • 状态值向量 (state value vector) \(v_\pi\):\(v_\pi \doteq [v_\pi(s_1), \dots, v_\pi(s_n)]^\top \in \mathbb{R}^n\),其中 \(n = |\mathcal{S}|\)。这是 25 个数堆叠成的向量,L4 的所有式子都把它当作单一对象操作。
  • 矩阵-向量形式 (matrix-vector form,教材 §2.6 公式 (2.10)):\(v_\pi = R_\pi + \gamma P_\pi v_\pi\)。这就是把 (2.7) 在 \(\forall s\) 上"竖着堆叠"得到的等式:每一行就是 (2.7) 在某个 \(s_i\) 上的展开。注意 \(v_\pi\) 同时出现在两边 —— bootstrap 的标志。
  • 闭式解 (closed-form solution,教材 §2.7.1 公式 (2.11) 上方):\(v_\pi = (I - \gamma P_\pi)^{-1} R_\pi\)。前提是 \(I - \gamma P_\pi\) 可逆 —— L4 §3 用 Gershgorin 圆盘定理给一个直观证明。
  • 迭代解 (iterative solution,教材 §2.7.2 公式 (2.11)):\(v_{k+1} = R_\pi + \gamma P_\pi v_k, \quad k = 0, 1, 2, \dots\),从任意 \(v_0\)(典型取 \(v_0 = 0\))出发。可证 \(v_k \to v_\pi\) 当 \(k \to \infty\)。这是 Ch.4 价值迭代算法的"政策评估"子步骤,本节先预演。
  • \(P_\pi\) 的两个性质:(a) \(P_\pi \ge 0\)(逐元素非负 —— 概率); (b) \(P_\pi \mathbf{1} = \mathbf{1}\)(每行求和为 1 —— 行随机矩阵)。这两条性质保证了所有特征值的绝对值 \(\le 1\),进而保证 \(I - \gamma P_\pi\) 可逆与迭代收敛。

Step 1. 从 (2.10) 到 (2.11) 的可逆性证明(直观版)。把 (2.10) 移项 \((I - \gamma P_\pi) v_\pi = R_\pi\),只要 \(I - \gamma P_\pi\) 可逆就能闭式解。\(P_\pi\) 是行随机矩阵,Gershgorin 圆盘定理告诉你它的所有特征值落在以 \([P_\pi]_{ii}\) 为圆心、\(\sum_{j \ne i} [P_\pi]_{ij}\) 为半径的圆盘并集里。每个圆盘 = 圆心 + 半径 = 行的对角元 + 非对角和 = 该行总和 = 1。所以特征值的绝对值 \(\le 1\),从而 \(\gamma P_\pi\) 的特征值绝对值 \(\le \gamma < 1\),\(I - \gamma P_\pi\) 没有 0 特征值,可逆。教材 §2.7.1 用同一个论证更细。
Step 2. 从 (2.11) 看收敛性(对应教材 Box 2.1)。设 \(\delta_k = v_k - v_\pi\),代入 (2.11) 与 (2.10) 相减得 \(\delta_{k+1} = \gamma P_\pi \delta_k\),从而 \(\delta_k = \gamma^k P_\pi^k \delta_0\)。因 \(\|P_\pi^k\|_\infty \le 1\)(行随机矩阵的幂仍是行随机矩阵),\(\|\delta_k\|_\infty \le \gamma^k \|\delta_0\|_\infty\) —— 误差几何衰减,收敛速率 = \(\gamma\)。\(\gamma = 0.9\) 时每次迭代误差减小 10%,大约 60-70 次迭代误差就低于 \(10^{-3}\)。这是 Ch.4 VI 收敛分析的预演。

§4 Grid-world 实例化

用教材 §2.6 图 2.6 / §2.5 图 2.5 那个 2×2 例子(随机策略),手工写出 \(v_\pi = R_\pi + \gamma P_\pi v_\pi\) 的具体形状。target = \(s_4\)、forbidden = \(s_2\)、\(s_1\) 上 0.5 概率向右(撞 \(s_2\) 拿 \(-1\))/0.5 概率向下(走 \(s_3\) 拿 \(0\)),\(s_2, s_3\) 都直接走到 \(s_4\) 拿 \(+1\),\(s_4\) 自环每步 \(+1\):

  • \(R_\pi = [\,0.5 \cdot 0 + 0.5 \cdot (-1),\ 1,\ 1,\ 1\,]^\top = [-0.5,\ 1,\ 1,\ 1]^\top\)
  • \(P_\pi = \begin{pmatrix} 0 & 0.5 & 0.5 & 0 \\ 0 & 0 & 0 & 1 \\ 0 & 0 & 0 & 1 \\ 0 & 0 & 0 & 1 \end{pmatrix}\):第 1 行是 \(s_1\) 出发的转移分布(0.5→\(s_2\)、0.5→\(s_3\));第 2、3、4 行都是 \(\to s_4\) 的确定转移。每行和都是 1 ✓。
  • \(\gamma = 0.9\) 代入 (2.10) 解出(也可手工逐项):\(v_\pi(s_4) = 1/(1-0.9) = 10\),\(v_\pi(s_3) = v_\pi(s_2) = 1 + 0.9 \cdot 10 = 10\),\(v_\pi(s_1) = -0.5 + 0.9 \cdot (0.5 \cdot 10 + 0.5 \cdot 10) = -0.5 + 9 = 8.5\)。与 L3 §4 例 2 完全一致 —— 矩阵形式没引入新的物理,只是记号压缩。

怎么读这个动画?每按一次"下一步",信息从 target 向外扩散一层(像水波),距离 target 越近的格子越早收到非零值。\(\gamma = 0.9\) 决定衰减系数:每多走一步,值多乘一次 0.9。这就是 Ch.4 价值迭代的内核 —— "每次只用上一轮的 \(v_k\) 算下一轮 \(v_{k+1}\)",简单且只需 \(O(n^2)\) 矩阵-向量乘,适合大状态空间。

§5 常见误解 / 易混概念

很多人以为:"既然有闭式解 \(v_\pi = (I - \gamma P_\pi)^{-1} R_\pi\),为什么还要迭代?直接套公式不就完了?"
实际上:闭式解只在理论分析时有用。实际计算 \((I - \gamma P_\pi)^{-1}\) 需要 \(O(n^3)\) flops 和 \(O(n^2)\) 内存(还得做 LU 分解);而迭代法每步 \(O(n^2)\),空间 \(O(n)\),且 \(\gamma\) 不太大时几十轮就收敛。一旦 \(n\) 大于 1000(grid 32×32 起),闭式解就明显吃力。
因为:实际 RL 关心可扩展性。Ch.4 VI、Ch.7 TD、Ch.8 DQN 全都用迭代/采样规避矩阵求逆 —— 闭式解此后基本不再出现,但它的存在是正确性的背书:迭代法收敛到的就是这个闭式解。
很多人以为:"\(P_\pi\) 是状态转移矩阵 —— 等等,这跟 MDP 五元组里的 \(p(s'\mid s, a)\) 不是一回事吗?"
实际上:差一个策略层。\(p(s'\mid s, a)\) 是环境给定 \((s, a)\) 后的转移;\(P_\pi\) 是把策略 \(\pi\) 也卷入后的"\(s \to s'\)"转移:\([P_\pi]_{ij} = \sum_a \pi(a \mid s_i) \, p(s_j \mid s_i, a)\)。前者是 MDP 自带的,后者依赖策略 —— 不同 \(\pi\) 给出不同的 \(P_\pi\),\(v_\pi\) 自然不同。
因为:Ch.1 L1 §5 那条"\(p\) 是环境的、\(\pi\) 是 agent 的"提醒,这里是在线性代数层面再演一次:\(P_\pi\) 这个 "strapped" 矩阵就是把两个属性卷起来的产物,后面 Ch.3 BOE 就把 max 套在这层卷积上做"最优"。
很多人以为:"\(v_0 = 0\) 是迭代的必须初值。"
实际上:任何 \(v_0 \in \mathbb{R}^n\) 都行 —— 误差几何衰减 \(\gamma^k\),起点只影响初期速率,不影响极限。\(v_0 = 0\) 只是最常用的"无信息先验",方便实现。如果你已有一个粗糙估计(比如上轮 PI 留下的 \(v\)),从那个 \(v_0\) 开始可能收敛更快(warm start,Ch.4 PI 会用)。
因为:压缩映射(\(\gamma\)-Lipschitz)的不动点不依赖起点,只依赖映射本身。Ch.3 BOE 收敛证明会用到同样的 Banach 不动点定理,本节先在线性情形预演一次。

§6 自检 + 视频 + 下一节

Q1. 在 §4 那个 4 状态例子里,把 \(\gamma\) 从 0.9 改到 0.5,迭代收敛大约需要多少次才能让 \(\|\delta_k\|_\infty < 10^{-3}\)?

A. \(\|\delta_k\|_\infty \le \gamma^k \|\delta_0\|_\infty\)。设 \(\|\delta_0\|_\infty \approx 10\)(初值 0,真值上限大约是 10),则需要 \(0.5^k \cdot 10 < 10^{-3}\) 即 \(0.5^k < 10^{-4}\),\(k > 4 / \log_{10} 2 \approx 13.3\)。大约 14 次。对比 \(\gamma = 0.9\) 时需要 \(0.9^k < 10^{-4}\),\(k > 87\) —— 差了 6 倍。这正是γ 越小、收敛越快的直觉佐证(短视的 agent 不需要看远,迭代少几轮就够);代价是 γ 小了"最优策略"也变了,这是上一节 (R2) 的延续。

Q2. 为什么 \(P_\pi\) 一定是行随机(每行和为 1)而不是列随机?

A. 因为 \([P_\pi]_{ij}\) 编码"从 \(s_i\) 出发(行下标)到 \(s_j\)(列下标)的概率",同一行的所有元素就是"从 \(s_i\) 出发,所有可能下一状态的概率分布",自然加和为 1。如果定义反了(行=终点、列=起点),那它就是列随机矩阵 —— 数学上等价,只是约定不同。本书统一用"行 = from、列 = to"的约定,后面 Ch.4-Ch.10 都跟着用,改成另一种会让 \(v_\pi = R_\pi + \gamma P_\pi v_\pi\) 变成 \(v_\pi = R_\pi + \gamma P_\pi^\top v_\pi\),记号会乱。

Q3. "迭代法收敛到 \(v_\pi\)"和"\(v_\pi = (I - \gamma P_\pi)^{-1} R_\pi\)"两件事,谁更基础?如果 \(I - \gamma P_\pi\) 不可逆(假设 \(\gamma = 1\) 又有自环),迭代法还会收敛吗?

A. 闭式解的"存在性"依赖于"\(I - \gamma P_\pi\) 可逆";迭代法的收敛性也依赖同一件事(因为收敛极限就是闭式解)。如果 \(I - \gamma P_\pi\) 不可逆(典型场景:\(\gamma = 1\) + 状态空间有环),\(v_\pi\) 可能不存在或不唯一,迭代会发散或循环。所以 \(\gamma < 1\) 是整个理论的根 —— 不是技术细节,是 RL 这个数学问题能成立的前提。这也呼应 Ch.1 L2 把 γ ∈ [0, 1) 写进 MDP 五元组的原因。

下一节预告

这一节我们用 \(v_\pi(s)\) 评估"整个状态"。但 RL 真正想做的是:找出每个状态下最值得执行的那个动作。要回答"动作好坏",需要一个比 state value 更细的量 —— action value \(q_\pi(s, a) \doteq \mathbb{E}_\pi[G_t \mid S_t = s, A_t = a]\):"在 \(s\) 下强行执行 \(a\)、之后再按 \(\pi\) 走"的折现 return 期望。下一节(L5,Book §2.8)会引入它,推出 \(v_\pi(s) = \sum_a \pi(a\mid s) q_\pi(s, a)\) 这个干净的关系,以及 \(q_\pi(s, a) = \sum_r p(r\mid s, a) r + \gamma \sum_{s'} p(s'\mid s, a) v_\pi(s')\) —— 它是后续 Q-learning(Ch.7)、DQN(Ch.8)、Policy Gradient(Ch.9)的核心对象。
