L3: Bellman Equation (P3 - Derivation) (贝尔曼方程(P3):推导)

§1 Why this lesson

上一节(L2)我们给了 \(v_\pi(s)\) 的语义定义:从 \(s\) 出发的折现 return 期望。但定义只告诉我们这个量"是什么",没告诉我们"怎么算"。直接按定义去算等于把所有 trajectory 枚举一遍 —— 真随机系统下不可行。本节就来补上"怎么算"。

本节是整章最关键的一节(教材 §2.4-2.5,pp. 20-25)。我们要从 L2 的定义出发,通过三步得到大名鼎鼎的贝尔曼方程:(1) 用 \(G_t = R_{t+1} + \gamma G_{t+1}\) 把 return 自己拆成"即时 + 未来";(2) 两边对 \(S_t = s\) 取条件期望,期望线性把它变成两项 \(\mathbb{E}[R_{t+1} \mid S_t=s] + \gamma \mathbb{E}[G_{t+1} \mid S_t=s]\);(3) 用 Markov 性把第二项中的 \(\mathbb{E}[G_{t+1} \mid S_t=s]\) 替换成 \(\sum_{s'} v_\pi(s') \, p(s' \mid s)\) —— 关键就是这一步用 \(v_\pi(s')\) 表达了 \(v_\pi(s)\),自洽闭环。最终得到的就是公式 (2.7) —— 一组以 \(\{v_\pi(s)\}_{s \in \mathcal{S}}\) 为未知量的线性方程组。下一节 L4 会把它写成矩阵形式直接求解。

§2 核心定义(直觉先行)

  • return 的递归形式 (recursive form of \(G_t\),教材 §2.4 公式 (2.4) 上方):\(G_t = R_{t+1} + \gamma R_{t+2} + \gamma^2 R_{t+3} + \dots = R_{t+1} + \gamma (R_{t+2} + \gamma R_{t+3} + \dots) = R_{t+1} + \gamma G_{t+1}\)。这就是"\(G_t\) = 一步即时 reward + γ 倍未来 return"。是 return自身的递归,不是期望。
  • 条件期望线性化(教材 §2.4 公式 (2.4)):\(v_\pi(s) = \mathbb{E}[R_{t+1} \mid S_t = s] + \gamma \, \mathbb{E}[G_{t+1} \mid S_t = s]\)。把"对 \(R_{t+1} + \gamma G_{t+1}\) 取期望"拆成两项分别取期望。
  • 第一项展开 — 即时 reward 的期望(教材 §2.4 公式 (2.5)):\(\mathbb{E}[R_{t+1} \mid S_t = s] = \sum_a \pi(a \mid s) \sum_r p(r \mid s, a) \, r\)。直觉:先按 \(\pi\) 选动作 \(a\),再按环境 \(p(r \mid s, a)\) 看会拿什么 reward,逐层加权。这是全期望公式(law of total expectation)的直接应用。
  • 第二项展开 — 未来 return 的期望(教材 §2.4 公式 (2.6)):\(\mathbb{E}[G_{t+1} \mid S_t = s] = \sum_{s'} v_\pi(s') \, p_\pi(s' \mid s)\),其中 \(p_\pi(s' \mid s) \doteq \sum_a \pi(a \mid s) \, p(s' \mid s, a)\)。关键一跳:用 Markov 性把 \(\mathbb{E}[G_{t+1} \mid S_t = s, S_{t+1} = s']\) 简化为 \(\mathbb{E}[G_{t+1} \mid S_{t+1} = s'] = v_\pi(s')\) —— 因为未来只依赖现在,与历史无关。
  • 合并 → 贝尔曼方程(教材 §2.4 公式 (2.7)):\(v_\pi(s) = \sum_a \pi(a \mid s) \bigl[ \sum_r p(r \mid s, a) \, r + \gamma \sum_{s'} p(s' \mid s, a) \, v_\pi(s') \bigr], \forall s \in \mathcal{S}\)。一句话:从 \(s\) 出发的状态值 = 即时 reward 期望 + γ 倍下一状态值的期望。注意"\(\forall s\)"才是 Bellman 方程的真身 —— 它是 \(|\mathcal{S}|\) 条线性方程联立,不是单条等式。
  • 等价形式(教材 §2.4 末段):用 \(p(s', r \mid s, a)\) 联合分布合并两个 sum,有 \(v_\pi(s) = \sum_a \pi(a\mid s) \sum_{s', r} p(s', r \mid s, a) [r + \gamma v_\pi(s')]\);若 reward 仅依赖下一状态(\(r = r(s')\)),则进一步化简为 \(v_\pi(s) = \sum_a \pi(a\mid s) \sum_{s'} p(s' \mid s, a)[r(s') + \gamma v_\pi(s')]\) —— Sutton-Barto 用的就是这个形式。

§3 关键公式与拆解

本节的核心就是 公式 (2.7)。我们用 4 个 callout 把它推完(每个 callout 对应一步关键变换),全程不需要超出"全期望公式 + Markov 性"。

公式:贝尔曼方程(教材 §2.4 公式 (2.7),p. 21)

Step 1. 从 \(G_t\) 的递归到 \(v_\pi\) 的递归(对应教材公式 (2.4))。从 L2 的 \(G_t = R_{t+1} + \gamma R_{t+2} + \dots\) 出发,把第一项分出来:\(G_t = R_{t+1} + \gamma (R_{t+2} + \gamma R_{t+3} + \dots) = R_{t+1} + \gamma G_{t+1}\) —— 注意这是 RV 等式,两边都是随机变量。两边对 \(S_t = s\) 取条件期望、用期望的线性,得 \(v_\pi(s) = \mathbb{E}[R_{t+1} \mid S_t = s] + \gamma \, \mathbb{E}[G_{t+1} \mid S_t = s]\)。这一步只用到一个事实:常数(\(\gamma\))可以拎出期望算子,期望线性。它对任意 RV 都成立,跟 Markov 性无关。
Step 2. 第一项展开 \(\mathbb{E}[R_{t+1} \mid S_t = s]\)(对应教材公式 (2.5))。用全期望公式按 \(A_t\) 分层:\(\mathbb{E}[R_{t+1} \mid S_t = s] = \sum_a \pi(a \mid s) \, \mathbb{E}[R_{t+1} \mid S_t = s, A_t = a]\)。再按 reward 分布展开 \(\mathbb{E}[R_{t+1} \mid S_t = s, A_t = a] = \sum_r p(r \mid s, a) \, r\)。合并得到 \(\mathbb{E}[R_{t+1} \mid S_t = s] = \sum_a \pi(a \mid s) \sum_r p(r \mid s, a) \, r\)。记号小心:这里出现了 \(p(r \mid s, a)\),它是 reward 的概率分布(MDP 五元组中的 \(R\) 部分),与状态转移 \(p(s' \mid s, a)\) 是不同的两个分布。
Step 3. 第二项展开 \(\mathbb{E}[G_{t+1} \mid S_t = s]\)(对应教材公式 (2.6))—— 这是整个推导的"魔术"步。同样用全期望公式按 \(S_{t+1}\) 分层:\(\mathbb{E}[G_{t+1} \mid S_t = s] = \sum_{s'} \mathbb{E}[G_{t+1} \mid S_t = s, S_{t+1} = s'] \cdot p(s' \mid s)\)。关键一跳:用 Markov 性(教材 §1.7 写过)把 \(\mathbb{E}[G_{t+1} \mid S_t = s, S_{t+1} = s']\) 简化为 \(\mathbb{E}[G_{t+1} \mid S_{t+1} = s']\) —— 因为给定 \(S_{t+1}\),未来 \(G_{t+1}\) 与更早的 \(S_t\) 条件独立。但 \(\mathbb{E}[G_{t+1} \mid S_{t+1} = s']\) 又恰好就是 \(v_\pi(s')\) 的定义!所以 \(\mathbb{E}[G_{t+1} \mid S_t = s] = \sum_{s'} v_\pi(s') \, p(s' \mid s)\)。最后用 \(p(s' \mid s) = \sum_a \pi(a \mid s) \, p(s' \mid s, a)\) 把策略分出来,得 \(\mathbb{E}[G_{t+1} \mid S_t = s] = \sum_a \pi(a \mid s) \sum_{s'} p(s' \mid s, a) v_\pi(s')\)。这一步是 Bellman 方程"自洽"的根 ——用 \(v_\pi(s')\) 表达 \(v_\pi(s)\),Markov 性是它能存在的充分条件。
Step 4. 合并 → 公式 (2.7)。把 Step 2、Step 3 的结果代回 Step 1,得 \(v_\pi(s) = \sum_a \pi(a \mid s) \sum_r p(r\mid s,a) r + \gamma \sum_a \pi(a \mid s) \sum_{s'} p(s' \mid s, a) v_\pi(s')\)。把外层 \(\sum_a \pi(a \mid s)\) 提出公因子,就是公式 (2.7) 的最终形式。结构上读它:外层 \(\sum_a \pi(\cdot)\) 是策略带来的随机性平均,内层方括号 \([\,\,]\) 是给定动作 \(a\) 后的"即时 + 折扣未来"分解。Bellman 方程的整体形状就是 "策略层 ∘ 动作层(即时 + γ × 未来)"的两层结构,一层对应 \(\pi\)、一层对应 \(p\),把 MDP 的两个随机源都打包进去了。

§4 Grid-world 实例化

用教材 §2.5 的两个最小实例(2×2 网格)逐字写出贝尔曼方程,然后求解。target = \(s_4\)(蓝色,右下),forbidden = \(s_2\)(橙色,右上),\(r_\text{forbidden} = -1\)、\(r_\text{target} = +1\)、其他 = 0,\(\gamma = 0.9\)。我们仍把它放回我们贯穿全书的 5×5 框架里看(target = \((3,2)\)、forbidden 集合保持),只是仅展示 4 个被讨论的状态:

2×2 例 1(教材图 2.4,确定性策略)。把 4 个状态分别写贝尔曼方程,只代入策略给出的那一个动作(其他动作概率为 0,sum 退化):

  • \(s_1\):\(\pi(a_3 \mid s_1) = 1\),\(p(s_3 \mid s_1, a_3) = 1\),\(p(r=0 \mid s_1, a_3) = 1\)。代入 (2.7):\(v_\pi(s_1) = 0 + \gamma v_\pi(s_3)\)。
  • \(s_2\):\(v_\pi(s_2) = 1 + \gamma v_\pi(s_4)\)(动作下、reward = +1、转移到 \(s_4\))。
  • \(s_3\):\(v_\pi(s_3) = 1 + \gamma v_\pi(s_4)\)(动作右、reward = +1、转移到 \(s_4\))。
  • \(s_4\):\(v_\pi(s_4) = 1 + \gamma v_\pi(s_4)\)(target 自环)。

从最后一条解出 \(v_\pi(s_4) = 1/(1-\gamma) = 10\),回代得 \(v_\pi(s_2) = v_\pi(s_3) = 1 + 0.9 \cdot 10 = 10\),\(v_\pi(s_1) = 0 + 0.9 \cdot 10 = 9\)。这与教材 p. 23 的结果完全一致。注意:"贝尔曼方程"虽然写成一行公式,实际是 4 条联立 —— 求解过程是一组耦合方程的代数运算,不是单条递推。

2×2 例 2(教材图 2.5,随机策略)。这次 \(s_1\) 上策略变成 \(\pi(a_2 \mid s_1) = 0.5\)、\(\pi(a_3 \mid s_1) = 0.5\)(右走撞 \(s_2\) forbidden、下走到 \(s_3\))。代入 (2.7) 得 \(v_\pi(s_1) = 0.5 \, [-1 + \gamma v_\pi(s_2)] + 0.5 \, [0 + \gamma v_\pi(s_3)] = -0.5 + 0.5 \gamma v_\pi(s_2) + 0.5 \gamma v_\pi(s_3)\)。其他三个状态 \(s_2, s_3, s_4\) 的方程不变,解得 \(v_\pi(s_4) = v_\pi(s_3) = v_\pi(s_2) = 10\),\(v_\pi(s_1) = -0.5 + 0.5 \cdot 9 + 0.5 \cdot 9 = -0.5 + 9 = 8.5\)。对比例 1 的 \(v_\pi(s_1) = 9\):随机策略让 \(s_1\) 的 state value 跌了 0.5 —— 正是"50% 概率撞 forbidden 拿 \(-1\)"的折现期望损失,与直觉吻合。

§5 常见误解 / 易混概念

很多人以为:"\(v_\pi(s) = R_{t+1} + \gamma v_\pi(s')\) —— 直接把 \(\mathbb{E}\) 去掉就完事了。"
实际上:这是整章最常见的写错。\(v_\pi\) 是常数、\(R_{t+1}\) 是 RV、\(s'\) 是 RV 的取值 —— 类型根本对不上。必须保留 \(\mathbb{E}\),且把 \(\mathbb{E}[\cdot \mid S_t = s]\) 用 \(\sum_a \pi(a \mid s) \sum_{r,s'} p(\cdot)\) 显式展开,才是 (2.7) 的正确形式。
因为:没有 \(\mathbb{E}\) 的 \(v_\pi(s) = R_{t+1} + \gamma v_\pi(s')\) 是沿一条 trajectory 的"return 一步分解",只在 trajectory 已经实现后成立 —— 不是关于 \(\{v_\pi(s)\}\) 的方程。期望算子是把 RV 等式升级为标量等式的必要桥梁。
很多人以为:"贝尔曼方程是递推:有了 \(v_\pi(s_3)\) 就能算 \(v_\pi(s_1)\),依次推下去。"
实际上:它是联立线性方程组(\(|\mathcal{S}|\) 个未知数 \(v_\pi(s_1), \dots, v_\pi(s_n)\),\(|\mathcal{S}|\) 条等式)。"递推"那种感觉只在 trajectory 是无环 DAG 时才成立(比如本节例 1,target 自环 + 单向链);一旦有环(L1 §2.2 那个 4 状态循环),就没有"先后顺序",必须联立解。
因为:未知量 \(v_\pi(s)\) 同时出现在等式两边(\(v_\pi(s) = \dots + \gamma v_\pi(s')\),且 \(s'\) 可以指回 \(s\) 自己),这是方程而不是递推式。L4 会把这件事写成 \(v_\pi = R_\pi + \gamma P_\pi v_\pi\),并用 \((I - \gamma P_\pi)^{-1}\) 一锤求解。
很多人以为:"Step 3 的 Markov 性化简是数学技巧,如果不懂可以先跳过。"
实际上:这一步是 Bellman 方程能存在的根本原因。如果没有 Markov 性,\(\mathbb{E}[G_{t+1} \mid S_t = s, S_{t+1} = s']\) 仍然依赖 \(S_t = s\) 这个历史,就不等于 \(v_\pi(s')\) —— Bellman 方程根本写不出来。
因为:所有 RL 算法(策略评估、价值迭代、Q-learning、TD)都站在 Bellman 方程上,而 Bellman 方程站在 Markov 性上。Ch.1 L2 Q3 的"扩状态"修复正是在补这条性质 —— 一旦补不回来,RL 整套理论就垮了。

§6 自检 + 视频 + 下一节

Q1. 在公式 (2.7) 的推导里,Step 1 的"\(G_t = R_{t+1} + \gamma G_{t+1}\)"对任意 trajectory 都成立、不需要 Markov 性。但 Step 3 的 \(\mathbb{E}[G_{t+1} \mid S_t = s, S_{t+1} = s'] = v_\pi(s')\) 必须 Markov 性。为什么差别这么大?

A. Step 1 是 RV 之间的代数恒等式(把 \(\sum_{k=0}^\infty \gamma^k r_{t+k+1}\) 重新排列),跟概率结构无关。Step 3 是条件分布的化简:它说"给定 \(S_{t+1} = s'\),再多知道 \(S_t = s\) 不会改变 \(G_{t+1}\) 的分布"——这正是 Markov 性的定义。如果 \(R_{t+2}, R_{t+3}, \dots\) 还依赖 \(S_t\) 这个历史,这个化简就不成立,\(v_\pi(s')\) 跟出发的 \(s\) 有关,Bellman 方程的"\(v_\pi(s) = \dots + \gamma v_\pi(s')\)"那部分会变成 "\(\dots + \gamma v_\pi(s' \mid s)\)" —— 多了个条件,方程组就不再封闭。

Q2. 把 §4 的 2×2 例 2 中 \(s_1\) 的随机策略改成 \(\pi(a_2 \mid s_1) = 1\)(确定性向右,撞 \(s_2\)),其他不变。\(v_\pi(s_1)\) 是多少?

A. 代入 (2.7):\(v_\pi(s_1) = 1 \cdot [-1 + \gamma v_\pi(s_2)] = -1 + 0.9 \cdot 10 = 8\)。比 0.5/0.5 随机策略下的 \(v_\pi(s_1) = 8.5\) 还低 —— 这与"100% 撞 forbidden 比 50% 更亏"的直觉一致。教学点:同一个 (2.7) 公式,只要换 \(\pi(a \mid s_1)\),就直接得到不同的 \(v_\pi(s_1)\),无需重新推导 —— 这就是 Bellman 方程作为策略评估工具的力量。

Q3. 对一个有 \(|\mathcal{S}| = 25\) 状态的 grid-world,公式 (2.7) 对应多少条联立方程?多少个未知数?线性还是非线性?

A. 25 条方程、25 个未知数(\(v_\pi(s_1), \dots, v_\pi(s_{25})\))、线性。这是因为 \(\pi(a \mid s)\) 和 \(p(\cdot \mid s, a)\) 都是给定常数,等式中 \(v_\pi\) 的系数全是常数,左右两边对 \(v_\pi\) 都是一次。这正是 L4 能直接写成矩阵 \(v_\pi = R_\pi + \gamma P_\pi v_\pi\) 并闭式求解的原因 —— 如果对 \(\pi\) 也求 max(下章 BOE)就变非线性了。

下一节预告

这一节我们得到了贝尔曼方程的逐元素形式(elementwise form)—— \(|\mathcal{S}|\) 条联立方程。但 \(|\mathcal{S}| = 25\) 一个一个写已经很烦,真实问题里 \(|\mathcal{S}|\) 可达成千上万。下一节(L4,Book §2.6-2.7)就把这堆方程打包成矩阵:定义向量 \(v_\pi \in \mathbb{R}^{|\mathcal{S}|}\)、\(R_\pi \in \mathbb{R}^{|\mathcal{S}|}\) 和矩阵 \(P_\pi \in \mathbb{R}^{|\mathcal{S}| \times |\mathcal{S}|}\),把 (2.7) 写成 \(v_\pi = R_\pi + \gamma P_\pi v_\pi\),从而闭式解 \(v_\pi = (I - \gamma P_\pi)^{-1} R_\pi\)。这是 RL 第一次出现"线性代数"工具,也是后面 VI/PI/TD 都站在的地基。
