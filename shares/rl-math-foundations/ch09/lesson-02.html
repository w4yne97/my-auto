L2: PG (P2) — Metric 1: Average State Value (策略梯度 P2:指标一,平均状态值)

§1 Why this lesson

L1 立了"策略 = 函数 \(\pi_\theta\),目标 = 标量 \(J(\theta)\)" 的骨架,但没 给出具体 的 \(J\) 形式。本节(教材 §9.2 第一部分,pp. 193-197)给第一种 指标:平均状态值 (average value) \(\bar v_\pi = \sum_s d(s) v_\pi(s) = \mathbb E_{S \sim d}[v_\pi(S)]\)。这是 RL 文献最常用的指标(也是教材 (9.1) \(\mathbb E[\sum_{t=0}^\infty \gamma^t R_{t+1}]\) 的另一种写法)。三件事:(a) 给定义;(b) 解释加权 \(d(s)\) 的两种选择(独立于 \(\pi\) 的 \(d_0\) vs 依赖 \(\pi\) 的 \(d_\pi\));(c) 证三种等价表达(求和、期望、discounted return)。

核心 takeaway:(a) 定义:\(\bar v_\pi \doteq \sum_{s\in\mathcal S} d(s) v_\pi(s)\),其中 \(d(s) \ge 0\)、\(\sum_s d(s) = 1\) 是状态加权分布。(b) 两种 \(d\) 选法:(i) \(d = d_0\) 独立 于 \(\pi\)(均匀 \(1/|\mathcal S|\) 或固定起点 \(s_0\)),记 \(\bar v_\pi^0\);(ii) \(d = d_\pi\) 依赖 于 \(\pi\)(stationary distribution \(d_\pi^\top P_\pi = d_\pi^\top\)),记 \(\bar v_\pi\)(无上标)。(c) 三种等价:\(\bar v_\pi = \sum_s d(s) v_\pi(s) = \mathbb E_{S\sim d}[v_\pi(S)] = \lim_{n\to\infty}\mathbb E[\sum_{t=0}^n \gamma^t R_{t+1}]\)(教材 (9.1) 与 Table 9.2)。(d) 几何 = 内积:\(\bar v_\pi = d^\top v_\pi\),把 \(v_\pi \in \mathbb R^{|\mathcal S|}\) 与 \(d \in \mathbb R^{|\mathcal S|}\) 看成两个向量做内积 ⇒ L4 求梯度时用得上。(e) 桥到 L3:本节是 \(\gamma < 1\) 的discounted 视角;L3 给 \(\gamma = 1\) 的average reward 视角,Lemma 9.1 证两者在 \(\gamma < 1\) 下等价(\(\bar r_\pi = (1-\gamma)\bar v_\pi\))。

§2 核心定义(直觉先行)

  • 平均状态值 \(\bar v_\pi\):\(\bar v_\pi = \sum_{s\in\mathcal S} d(s) v_\pi(s)\)。"把每状态的价值按权 \(d\) 加权平均"。\(d\) 高的状态贡献大,\(d\) 低的状态贡献小。直观:策略常去 的状态价值高 ⇒ 策略整体 好。
  • 独立分布 \(d_0\)(教材 §9.2 第一案例):\(d_0\) 与 \(\pi\) 无关,常见两种:(i) 均匀 \(d_0(s) = 1/|\mathcal S|\) ⇒ "所有状态等权重重要";(ii) 固定起点 \(d_0(s_0) = 1, d_0(s\ne s_0) = 0\) ⇒ "只关心从 \(s_0\) 出发的总回报"。第二种特别有用:对应 RL 中"agent 总从同一起点开始" 的标准设定。
  • 策略相关分布 \(d_\pi\)(stationary distribution):\(d_\pi^\top P_\pi = d_\pi^\top\),即 \(\pi\) 诱导的马氏链不变分布。直观:策略长期运行 中状态 \(s\) 出现的频率。"常访问的 \(s\) 权大,罕访问的 \(s\) 权小" — 这与"策略实际表现" 一致。
  • 等价表达一:求和 vs 期望:\(\sum_s d(s) v_\pi(s) = \mathbb E_{S\sim d}[v_\pi(S)]\)。这是形式 改写,不改值。期望形式在 SGD 中更便利(单样本估)。
  • 等价表达二:discounted return:\(\mathbb E[\sum_{t=0}^\infty \gamma^t R_{t+1}] = \sum_s d(s) v_\pi(s) = \bar v_\pi\)(教材 (9.1) + 推导)。证明用全概率公式拆 \(R_{t+1}\) 的条件期望,把"从 \(s_0\) 出发的累积回报" 重写成"各 \(s\) 价值的加权和"。这一等价是 RL 文献 \(J = \mathbb E[\sum \gamma^t R]\) 与教材 \(J = \bar v_\pi\) 同构 的根。
  • 向量内积形式:\(\bar v_\pi = d^\top v_\pi\),其中 \(d, v_\pi \in \mathbb R^{|\mathcal S|}\)。这一形式在 L4 求 \(\nabla_\theta \bar v_\pi\) 时极其 重要 —— 把求梯度变成"对内积求导",可借线性代数工具。
  • "\(\theta\) 通过 \(\pi\) 影响 \(v_\pi\)":\(\bar v_\pi(\theta) = \sum_s d(s) v_\pi(s; \theta)\)。即使 \(d = d_0\) 固定,\(v_\pi(s)\) 仍依赖 \(\theta\)(策略 \(\pi_\theta\) 决定贝尔曼方程的解)。\(d = d_\pi\) 则双重 依赖 ⇒ 求梯度多一项 \(\nabla d_\pi\)(L4 处理)。

分布 \(d\) 的两种选择(直观对比):

观察:\(d_0\) 一律均权;\(d_\pi\) 反映"策略常去 target 附近" ⇒ target 状态权大。两种 \(d\) 给两种不同 的 \(\bar v_\pi\) 值,不同 的 \(\nabla_\theta \bar v_\pi\) 形式 —— L4 Theorem 9.2 处理 \(d_0\),Theorem 9.3 处理 \(d_\pi\)。

Step 1. "discounted return = 加权 \(v_\pi\)" 的全概率证。\(\mathbb E[\sum_{t=0}^\infty \gamma^t R_{t+1}]\) 中的外层 \(\mathbb E\) 是对初始 \(S_0\) + 整条轨迹 一起取期望。用全概率拆 \(S_0\):\(\mathbb E[\cdot] = \sum_s \Pr(S_0 = s) \cdot \mathbb E[\cdot | S_0 = s]\)。内层 \(\mathbb E[\sum \gamma^t R_{t+1} | S_0 = s] = v_\pi(s)\)(Ch.2 L2 状态值定义)。外层 \(\Pr(S_0 = s) = d(s)\)(初始分布)。代入 ⇒ \(\sum_s d(s) v_\pi(s) = \bar v_\pi\)。结论:\(J(\theta) = \mathbb E[\sum \gamma^t R]\) 不是新 目标,而是 \(\bar v_\pi\) 的路径展开 写法。
Step 2. "\(d_\pi\) 加权 = 策略表现的自然 度量"。\(d_\pi\) 由 \(\pi\) 自身决定 ⇒ \(\bar v_\pi = d_\pi^\top v_\pi\) 是 \(\pi\) "常访问位置上的平均价值"。两个策略 \(\pi_1, \pi_2\) 比较时,各自在自己 常访问的状态上各自 的 \(v_\pi\) 加权 ⇒ 公平。换 \(d_0\) 均匀加权 ⇒ "策略从未 访问的状态上的 \(v_\pi\)" 也算进总分,可能不公平(那些状态的 \(\pi\) 行为无关紧要)。但 \(d_\pi\) 依赖 \(\theta\) ⇒ L4 求梯度时多一项,数学上比 \(d_0\) 麻烦。教材两种都讲,实际工程多取 \(d_\pi\) 隐含(on-policy 样本天然来自 \(d_\pi\),Ch.8 L2 同样道理)。
Step 3. "内积 \(d^\top v_\pi\)" 的线性代数便利。把 \(\bar v_\pi\) 写成内积 ⇒ 求 \(\nabla_\theta\) 时:若 \(d\) 与 \(\theta\) 无关(\(d = d_0\)),\(\nabla_\theta \bar v_\pi = d^\top \nabla_\theta v_\pi\)(只对 \(v_\pi\) 求);若 \(d = d_\pi\) 依赖 \(\theta\),需展开 \(\nabla_\theta(d_\pi^\top v_\pi) = (\nabla_\theta d_\pi)^\top v_\pi + d_\pi^\top \nabla_\theta v_\pi\) 两项。L4 Theorem 9.2 / 9.3 各自处理,后者形式上略不同但同骨架(PG 定理)。关键:线性代数把"策略改一点 \(\pi\) 怎么变" 转换成"矩阵-向量微分",省力。

§5 常见误解 / 易混概念

很多人以为:"\(\bar v_\pi = \mathbb E[\sum \gamma^t R]\) 这个目标显然 —— 直接对它求梯度做 SGD 不就行了?为什么还要写成 \(\sum_s d(s) v_\pi(s)\)?"
实际上:两种写法各有所长。\(\mathbb E[\sum \gamma^t R]\) 采样 友好(单条 episode 的 \(\sum \gamma^t r_t\) 是无偏样本),但求梯度 不便(要对随机轨迹的整条求 \(\nabla\));\(\sum_s d(s) v_\pi(s)\) 分析 友好(转成内积 + 矩阵-向量,L4 PG 定理直接出),但采样 不直观。Ch.9 用分析 形式推 PG 定理(L4),再翻回 期望形式做 SGD(L5 REINFORCE)。
因为:"推" 与"用" 的形式 需求不同。Ch.9 把这两种形式作为同一目标的两面,根据需要切换。这是 RL 数学论文中常见的"形式切换术",PG 定理本身就是切换的产物。
很多人以为:"\(d\) 选什么不重要 —— 反正都是加权平均,最终 \(\arg\max_\theta\) 应该差不多。"
实际上:\(d\) 选法显著 影响最优 \(\theta^*\)。\(d_0 = \delta_{s_0}\)(固定起点) ⇒ 优化"从 \(s_0\) 出发的总回报",在不可达 \(s\) 上的 \(\pi\) 行为不影响 \(J\) ⇒ 那些状态训练后无定义(任意值都行)。\(d_0\) 均匀 ⇒ 所有 \(s\) 上 \(\pi\) 都要训好,即使不可达 \(s\) 也要(奇怪:策略不去那也要好)。\(d_\pi\) ⇒ 与 on-policy 样本一致,工程默认。不同 \(d\) ≈ 不同任务定义。
因为:RL 的"最优策略" 本质是加权 优化的解,加权 \(d\) 改变 ⇒ 解改变。教材给两种 \(d\) 不是装饰,而是不同应用场景 下两种合理选择(固定起点 vs 长期运行)。L4 PG 定理对两种 \(d\) 各 给一形式(Theorem 9.2 vs 9.3)。
很多人以为:"既然 \(\bar v_\pi = d^\top v_\pi\) 是线性 内积,那 \(\nabla_\theta \bar v_\pi = d^\top \nabla_\theta v_\pi\) 应该显然 —— L4 何必整一章?"
实际上:\(\nabla_\theta v_\pi\) 本身不 显然。\(v_\pi(s) = \mathbb E[\sum \gamma^t R | S_0 = s]\) 隐式依赖 \(\pi_\theta\)(贝尔曼方程的解),求 \(\nabla_\theta v_\pi\) 要展开 \(v_\pi = \sum_a \pi(a|s; \theta) q_\pi(s, a)\) 再对 \(\theta\) 求导(Lemma 9.2)。结果含 \(\nabla q_\pi\) 项,\(q_\pi\) 又递归依赖 \(v_\pi\) ⇒ 递归方程。L4 用"matrix-vector + Kronecker" 解开这递归,得 \(\nabla v_\pi(s) = \sum_{s'}\Pr_\pi(s'|s) \sum_a \nabla \pi(a|s') q_\pi(s', a)\),最终给 PG 定理。
因为:RL 的价值 是 MDP 全局耦合的函数,改 \(\theta\) 一点 \(\Rightarrow\) 改所有 \(s\) 的 \(v_\pi\)(通过马氏链传播)。L4 的工作是把这耦合 形式解析 写出。这不是"显然",是 PG 定理的核心难点。

§6 自检 + 视频 + 下一节

Q1. 写出 \(\bar v_\pi\) 的三种 等价表达,并说明各自适合什么 用途。

A. (i) 求和 \(\sum_s d(s) v_\pi(s)\) ⇒ "原生" 定义,强调"每状态贡献"。(ii) 期望 \(\mathbb E_{S\sim d}[v_\pi(S)]\) ⇒ 采样 便利,SGD 单样本估;L5 REINFORCE 用此形式取 \(\hat v_\pi(s_t) \approx G_t\)。(iii) 内积 \(d^\top v_\pi\) ⇒ 线性代数 便利,L4 求 \(\nabla_\theta\) 时把"策略改一点" 转 \(\nabla v_\pi\) 矩阵-向量。(iv)(等价 disounted return)\(\mathbb E[\sum \gamma^t R_{t+1}]\) ⇒ 文献常见,与 RL 论文衔接。前三种数学严格等价,第四种通过全概率公式化简到第一种。

Q2. \(d_0\) 与 \(d_\pi\) 选法的语义 区别是什么?哪种与 on-policy SGD 更自然?

A. \(d_0\) 独立 于 \(\pi\):均匀 \(1/|\mathcal S|\)("所有 \(s\) 等重要")或 \(\delta_{s_0}\)("只关心固定起点")。\(d_\pi\) 依赖 \(\pi\):\(d_\pi^\top P_\pi = d_\pi^\top\),反映"\(\pi\) 长期访问频率"。on-policy 自然性:on-policy 样本 \(s_t \sim d_\pi\)(经长 episode 后近似 stationary),拿样本算 \(v_\pi(s_t)\) 平均 = \(\bar v_\pi^{d_\pi}\) 的无偏估,SGD 直接套。换 \(d_0\) ⇒ 需重要性采样修正(IS 比 \(d_0(s_t) / d_\pi(s_t)\),增加方差)。所以工程默认 \(d_\pi\)(教材 Theorem 9.3),理论分析两个都讲。

Q3. 推导 \(\mathbb E[\sum_{t=0}^\infty \gamma^t R_{t+1}] = \sum_s d(s) v_\pi(s)\)(用全概率公式)。

A. 由全概率公式拆初始 \(S_0\):\(\mathbb E[\sum \gamma^t R_{t+1}] = \sum_s \Pr(S_0 = s) \cdot \mathbb E[\sum \gamma^t R_{t+1} | S_0 = s]\)。内层:\(\mathbb E[\sum \gamma^t R_{t+1} | S_0 = s] = v_\pi(s)\)(Ch.2 L2 状态值定义,正是从 \(s\) 出发 的折扣累积回报期望)。外层:\(\Pr(S_0 = s) = d(s)\)(初始分布约定)。代入 ⇒ \(\sum_s d(s) v_\pi(s) = \bar v_\pi\)。意义:文献中"\(J = \mathbb E[\text{return}]\)" 与教材"\(J = \bar v_\pi\)" 同义,只是切角度不同。所有 RL 论文的 PG 都从这一等价开始。

下一节预告

本节给了第一种 指标 \(\bar v_\pi\)(discounted 视角,\(\gamma < 1\))。L3(同教材 §9.2 第二部分) 给第二种:平均奖励 (average reward) \(\bar r_\pi = \sum_s d_\pi(s) r_\pi(s) = \mathbb E_{S\sim d_\pi}[r_\pi(S)]\)。直观:每步 reward 的 long-run 平均,适合无折扣 (\(\gamma = 1\)) 或无穷 horizon 场景(机器人持续运行、网络流控)。L3 还会证Lemma 9.1: \(\bar r_\pi = (1-\gamma) \bar v_\pi\)(discounted 下两 metric 等价),并给 \(\bar r_\pi\) 的三种等价 表达(类比本节)。Ch.9 之所以同时讲两 metric:L4 推 PG 定理时both 都用,且无折扣 形式比有折扣 形式梯度更优雅(Theorem 9.5 strictly,9.3 是 approximation)。
