L3: PG (P3) — Metric 2: Average Reward (策略梯度 P3:指标二,平均奖励)

§1 Why this lesson

L2 给了第一种 指标 \(\bar v_\pi\)("平均状态值",discounted 视角,\(\gamma < 1\))。本节(教材 §9.2 第二部分,pp. 195-198)给第二种:平均奖励 (average reward) \(\bar r_\pi = \sum_s d_\pi(s) r_\pi(s) = \mathbb E_{S\sim d_\pi}[r_\pi(S)]\)。直观:策略每步获得的 reward 的 long-run 平均值。这一指标在无折扣(\(\gamma = 1\))或无穷 horizon 场景更自然 —— 机器人持续运行、网络流控、广告竞价等"没有终止" 的任务,\(\sum \gamma^t R\) 收敛性可疑(\(\gamma = 1\) 时和发散),但每步均值恒 有限。本节三件事:(a) 给定义;(b) 证三种等价表达(求和、期望、Cesaro 极限);(c) 证Lemma 9.1 \(\bar r_\pi = (1-\gamma)\bar v_\pi\) —— L2 与 L3 在 \(\gamma < 1\) 下同向最优。

核心 takeaway:(a) 定义:\(\bar r_\pi \doteq \sum_s d_\pi(s) r_\pi(s)\),其中 \(r_\pi(s) = \sum_a \pi(a|s; \theta) r(s, a) = \mathbb E_{A\sim\pi(s)}[r(s, A)]\) 是状态 \(s\) 上的期望即时 reward(教材 (9.3))。(b) 三等价:\(\bar r_\pi = \sum_s d_\pi(s) r_\pi(s) = \mathbb E_{S\sim d_\pi}[r_\pi(S)] = \lim_{n\to\infty}\frac{1}{n}\mathbb E[\sum_{t=0}^{n-1} R_{t+1}]\)(教材 (9.4)-(9.5),Box 9.1 用 Cesaro 平均证)。(c) Lemma 9.1:\(\bar r_\pi = (1-\gamma)\bar v_\pi\),证用贝尔曼方程 \(v_\pi = r_\pi + \gamma P_\pi v_\pi\) 左乘 \(d_\pi^\top\) 化简。(d) 含义:\(\gamma < 1\) 下两 metric 只差常数 因子 \((1-\gamma)\) ⇒ \(\arg\max_\theta \bar r_\pi = \arg\max_\theta \bar v_\pi\),用谁优化都一样。\(\gamma = 1\) 时 \(\bar v_\pi\) 不存在(\(v_\pi\) 发散),只 \(\bar r_\pi\) 可用。(e) 桥到 L4:L4 PG 定理对both metric 各推一遍 —— Theorem 9.3 处理 \(\bar v_\pi\)(approximation),Theorem 9.5 处理 \(\bar r_\pi\)(strict equality 在 \(\gamma = 1\),更优雅)。\(\bar r_\pi\) 是 PG 定理的最干净 形式。

§2 核心定义(直觉先行)

  • 状态期望 reward \(r_\pi(s)\)(教材 (9.3)):\(r_\pi(s) = \sum_a \pi(a|s; \theta) r(s, a) = \mathbb E_{A\sim\pi(s)}[r(s, A)]\)。"从 \(s\) 出发,按 \(\pi\) 取 \(a\),\(r(s, a)\) 的期望"。这是即时(one-step)reward,不含未来。\(r(s, a) = \mathbb E[R | s, a] = \sum_r r p(r|s, a)\)。
  • 平均奖励 \(\bar r_\pi\)(教材 (9.2)):\(\bar r_\pi = \sum_s d_\pi(s) r_\pi(s) = d_\pi^\top r_\pi\)(向量内积形式)。\(d_\pi\) 必须是策略相关 stationary distribution(L2 的 \(d_0\) 在此不 适用 —— 平均奖励本质是长期 行为度量,起点早被遗忘)。
  • Cesaro 平均(教材 Box 9.1):\(\lim_{n\to\infty}\frac{1}{n}\sum_{k=1}^n a_k = \lim_{k\to\infty} a_k\)(若后者存在)。"序列的平均极限 = 序列的极限"。这是 \(\bar r_\pi\) 与 discounted return \(\frac{1}{n}\mathbb E[\sum R_{t+1}]\) 等价的关键工具。
  • 三种等价(教材 Table 9.2):(i) 求和 \(\sum_s d_\pi(s) r_\pi(s)\);(ii) 期望 \(\mathbb E_{S\sim d_\pi}[r_\pi(S)]\);(iii) 长期平均 \(\lim_n \frac{1}{n}\mathbb E[\sum_{t=0}^{n-1} R_{t+1}]\)(教材 (9.4))。前两者形式改写,第三者用 Cesaro + stationary distribution 证(Box 9.1)。
  • Lemma 9.1: \(\bar r_\pi = (1-\gamma) \bar v_\pi\)(教材 §9.2 末):(i) 由贝尔曼 \(v_\pi = r_\pi + \gamma P_\pi v_\pi\) 左乘 \(d_\pi^\top\):\(d_\pi^\top v_\pi = d_\pi^\top r_\pi + \gamma d_\pi^\top P_\pi v_\pi\);(ii) 用 stationary 性质 \(d_\pi^\top P_\pi = d_\pi^\top\):右侧 = \(d_\pi^\top r_\pi + \gamma d_\pi^\top v_\pi\);(iii) 移项 \(\bar v_\pi - \gamma \bar v_\pi = \bar r_\pi\) ⇒ \(\bar r_\pi = (1-\gamma)\bar v_\pi\)。
  • "discounted vs undiscounted":\(\gamma < 1\) ⇒ 两 metric 只差常数,可互换;\(\gamma = 1\) ⇒ \(\bar v_\pi\) 发散(\(\sum 1 \cdot R\) 不收敛),只 \(\bar r_\pi\) 可定义。所以教材 §9.3.2 (L4 后半)专门 讲无折扣 情形 —— \(\bar r_\pi\) 是更通用 的指标。
  • 桥到 L4 PG 定理:Theorem 9.3 给 \(\bar r_\pi\) 在 \(\gamma < 1\) 下的梯度(approximation,丢一个量级 \(O(1)\) 项);Theorem 9.5 给 \(\bar r_\pi\) 在 \(\gamma = 1\) 下的梯度(strict equality,更优雅)。\(\bar r_\pi\) 这一指标在 PG 定理推导中最干净,这是教材教 \(\bar r_\pi\) 而非只 \(\bar v_\pi\) 的结构 原因。

公式 3:Lemma 9.1 — \(\bar r_\pi = (1-\gamma) \bar v_\pi\)(教材 §9.2 末)

Step 1. Cesaro 平均的具体 用法。Box 9.1 第一步证 \(\bar r_\pi = \lim_{n\to\infty}\frac{1}{n}\mathbb E[\sum_{t=0}^{n-1} R_{t+1} | S_0 = s_0]\)(任意起点 \(s_0\))。关键:\(\frac{1}{n}\sum_{t=0}^{n-1} \mathbb E[R_{t+1}|S_0]\) 的极限 由 Cesaro 平均 \(=\) \(\lim_{t\to\infty} \mathbb E[R_{t+1}|S_0]\)。后者 \(=\) \(\sum_s r_\pi(s)\lim_{t\to\infty} P^{(t)}(s|s_0) = \sum_s r_\pi(s) d_\pi(s) = \bar r_\pi\)(stationary 性质)。关键:Cesaro 把"每步期望的序列" 转成"稳定后的极限",从而起点 \(s_0\) 被遗忘。这是"average reward 不依赖起点" 的形式根。
Step 2. "\(\bar r_\pi\) 没有 \(\gamma\)" 的实际意义。机器人控制:agent 持续运行 24 小时不停,无"终止" 概念。\(\sum \gamma^t R\) 在 \(\gamma = 1\) 下发散,在 \(\gamma < 1\) 下"未来打折" 与"每步同等重要" 的任务直觉不符;\(\bar r_\pi\) 每步同权,直接对应"持续高 reward"。另一例:网络流控,目标是长期 吞吐率,正是"每步 reward 平均"。教材 §9.3.2 (L4 后半)在\(\gamma = 1\) 下推 PG 定理,得 Theorem 9.5(strict equality),数学上比 \(\bar v_\pi\) 的 Theorem 9.3(approximation)更 干净 —— 工程上反而 \(\bar r_\pi\) 是首选 形式。
Step 3. "\(\bar r_\pi = (1-\gamma)\bar v_\pi\)" 之外的"不同点"。Lemma 9.1 让人误以为两 metric "完全等价";但:(i) 求梯度 时形式不同:\(\nabla \bar r_\pi\) 在 \(\gamma = 1\) 严格 = \(\mathbb E[\nabla \ln \pi \cdot q_\pi]\)(Theorem 9.5);\(\nabla \bar v_\pi\) 含一项 \((1-\gamma)^{-1}\) 系数 + 高阶残项(Theorem 9.3 是 \(\gamma \to 1\) 极限的近似)。(ii) 用不同 状态分布 \(\eta\):\(\bar v_\pi^0\) 的 PG 定理用 \(\rho_\pi\)(discounted total probability,(9.19));\(\bar r_\pi\) 用 \(d_\pi\)(stationary)。(iii) 数值上"同向" 不等于"梯度同方向" —— 实际工程选哪个目标 ⇒ 用哪个梯度公式。结论:Lemma 9.1 给了"同最优",但怎么 走到最优(梯度路径)在两 metric 下细节不同。

§4 数值演示:\(\bar r_\pi\) 与 \(\bar v_\pi\) 的比例验证

用 L2 玩具 MDP:3 状态,\(v_\pi = [10, 5, 8]^\top\),\(d_\pi = [0.3, 0.4, 0.3]\)。设 \(r_\pi = [1.0, 0.5, 0.8]^\top\)(\(\pi\) 在三状态下 one-step 期望 reward),\(\gamma = 0.9\)。

观察:\(\bar v_\pi\) 与 \(\bar r_\pi\) 同号同向,只差 \((1-\gamma)\) 因子(本例 \(=0.1\))。但梯度 的形式不同(L4 Theorem 9.3 vs Theorem 9.5),且无折扣 (\(\gamma=1\)) 下只有 \(\bar r_\pi\) 可用。\(\bar r_\pi\) 是 PG 定理更通用 的入口。

§5 常见误解 / 易混概念

很多人以为:"既然 \(\bar r_\pi = (1-\gamma)\bar v_\pi\),Ch.9 何必两个 都讲?直接用 \(\bar v_\pi\) 一个就够了。"
实际上:\(\gamma = 1\) 时 \(\bar v_\pi\) 不存在。Lemma 9.1 只在 \(\gamma < 1\) 下成立(证明用 \(I - \gamma P_\pi\) 可逆,要求 \(\gamma < 1\))。\(\gamma = 1\) ⇒ \(v_\pi(s) = \mathbb E[\sum_{t=0}^\infty R_{t+1}|s]\) 在非吸收 链上发散 ⇒ \(\bar v_\pi\) 无定义。但 \(\bar r_\pi\) 在 \(\gamma = 1\) 仍有定义(Cesaro 平均收敛)。无折扣 RL 任务(机器人持续控制)必须 用 \(\bar r_\pi\)。
因为:"等价" 是条件(\(\gamma < 1\))下的等价。Ch.9 教both 是为覆盖 全场景:\(\gamma < 1\) 用 \(\bar v_\pi\)(传统 RL,episode 化任务),\(\gamma = 1\) 用 \(\bar r_\pi\)(continuing tasks)。L4 也分 9.3.1 (discounted) / 9.3.2 (undiscounted) 两小节推 PG 定理。
很多人以为:"\(\bar r_\pi\) 既然是每步 平均 reward,只看 immediate reward 即可,与未来 无关 ⇒ 应该比 \(\bar v_\pi\) 容易学。"
实际上:\(\bar r_\pi\) 仍隐含 长期 dynamics。\(\bar r_\pi = d_\pi^\top r_\pi\) 中 \(d_\pi\) 是长期 访问频率,由 \(\pi\) 与 \(P\) 共同 决定 ⇒ 改 \(\theta\) 改 \(\pi\) 改 \(d_\pi\)。即使 \(r_\pi\) 是 one-step,\(\bar r_\pi\) 通过 \(d_\pi\) 把"长期 dynamics" 引回。\(\nabla_\theta \bar r_\pi\) 仍含 \(\nabla_\theta d_\pi\) 项(L4 Theorem 9.5 推完,严格形式 \(\mathbb E[\nabla \ln \pi \cdot q_\pi]\) 中的 \(q_\pi\) 用 Poisson equation 定义,内涵长期)。
因为:RL 的长期 与即时 不可分。\(\bar r_\pi\) 把"长期" 藏到 \(d_\pi\) 中,\(\bar v_\pi\) 把"长期" 藏到 \(v_\pi\) 中,本质都是"策略 + 转移" 的耦合。两者难度相当,选哪个看形式便利 与任务设定。
很多人以为:"Lemma 9.1 证明左乘 \(d_\pi^\top\) 看起来像魔法 一步,有什么直觉?"
实际上:"左乘 \(d_\pi^\top\)" 是把贝尔曼方程的每行 按 \(d_\pi\) 加权求和。贝尔曼 \(v_\pi(s) = r_\pi(s) + \gamma \sum_{s'} P_\pi(s, s') v_\pi(s')\) 是每个 \(s\) 一行;左乘 \(d_\pi^\top\) 即对所有 \(s\) 按 \(d_\pi\) 加权累加,把逐行 等式合成一个 标量等式。这一"加权求和" 操作在 RL 推导中无处不在(Ch.8 L4 \(A \succ 0\) 证、Ch.9 L4 PG 定理),核心是\(d_\pi^\top P_\pi = d_\pi^\top\) 的 stationary 性质让 \(\sum_{s'}\) 项自身 化简。
因为:stationary distribution 是 \(P_\pi\) 的左特征向量(特征值 1),\(d_\pi^\top P_\pi = d_\pi^\top\) 让"加权求和后再传一步" 与"加权求和" 相等 ⇒ 任意"线性贝尔曼" 形式都可这一步消化 \(P_\pi\)。这是 RL 矩阵分析的万能 lever。

§6 自检 + 视频 + 下一节

Q1. 写出 \(\bar r_\pi\) 的三种 等价表达,并指出哪一种适合 \(\gamma = 1\) 场景。

A. (i) 求和 \(\bar r_\pi = \sum_s d_\pi(s) r_\pi(s)\)("原生" 定义);(ii) 期望 \(\bar r_\pi = \mathbb E_{S\sim d_\pi}[r_\pi(S)]\)(SGD 友好,样本估);(iii) Cesaro 长期平均 \(\bar r_\pi = \lim_n \frac{1}{n}\mathbb E[\sum_{t=0}^{n-1} R_{t+1}]\)(教材 (9.4),Box 9.1 用 Cesaro 平均证)。三者数学等价。哪种适合 \(\gamma = 1\):三种都 适合(\(\bar r_\pi\) 定义里没 \(\gamma\)),区别仅是形式。前两者数学严格,第三者更直观(表达"每步平均 reward" 的物理直觉)。注意 \(\bar v_\pi\) 在 \(\gamma = 1\) 下不存在(发散),所以 continuing tasks 必须 用 \(\bar r_\pi\)。

Q2. 用贝尔曼方程证 Lemma 9.1: \(\bar r_\pi = (1-\gamma)\bar v_\pi\)。

A. 第一步:贝尔曼方程 \(v_\pi = r_\pi + \gamma P_\pi v_\pi\)(矩阵-向量形式,Ch.2 L4)。第二步:左乘 \(d_\pi^\top\) ⇒ \(d_\pi^\top v_\pi = d_\pi^\top r_\pi + \gamma d_\pi^\top P_\pi v_\pi\)。第三步:用 stationary 性质 \(d_\pi^\top P_\pi = d_\pi^\top\) ⇒ 右侧第二项 = \(\gamma d_\pi^\top v_\pi\)。第四步:左侧 = \(\bar v_\pi\),右侧 = \(\bar r_\pi + \gamma \bar v_\pi\) ⇒ \(\bar v_\pi - \gamma \bar v_\pi = \bar r_\pi\) ⇒ \(\bar r_\pi = (1-\gamma)\bar v_\pi\)。含义:\(\gamma < 1\) 下两 metric 同向最优,任选一优化即可。\(\gamma = 1\) 下证明失效(\((1-\gamma) = 0\) 给 \(\bar r_\pi = 0\) 错误结论 —— 实际是 \(\bar v_\pi\) 发散导致 lim 不存在)。

Q3. 为什么"average reward" 任务的argmax 在 \(\gamma < 1\) 与 \(\gamma = 1\) 下可能不同?

A. \(\gamma < 1\) 下 \(\bar r_\pi = (1-\gamma)\bar v_\pi\),最大化 \(\bar r_\pi\) 等价于最大化 \(\bar v_\pi\),解的形式上同。但:\(\bar v_\pi\) 偏向 "近期高 reward"(\(\gamma\) 折扣后期);\(\bar r_\pi\) (\(\gamma = 1\)) 等权所有时刻。例:agent 选"立即拿 +1 然后 0" vs "10 步后拿 +10 持续"。\(\gamma = 0.9\) 下前者 \(\bar v \approx 1\)、后者 \(\bar v \approx 10 \cdot 0.9^{10}/(1-0.9) \approx 35\) ⇒ 后者优;\(\gamma = 0.5\) 下后者 \(\bar v \approx 10 \cdot 0.5^{10}/(1-0.5) \approx 0.02\) ⇒ 前者优。argmax 在同 \(\gamma\) 下两 metric 等价(Lemma 9.1),但不同 \(\gamma\) 给不同最优。\(\gamma = 1\) 是"无视折扣" 的极限 选择,适合 continuing tasks。

下一节预告

本节给了第二种 指标 \(\bar r_\pi\) 与 Lemma 9.1。L2 + L3 把两种 指标 \(J(\theta)\) 都定义 完毕。L4(教材 §9.3, pp. 198-209,40 分钟最长一节) 是 Ch.9 的核心定理:对两种 metric 求 \(\nabla_\theta J\)。policy gradient theorem 给出统一 形式 \(\nabla_\theta J(\theta) = \mathbb E_{S\sim\eta, A\sim\pi}[\nabla_\theta \ln\pi(A|S; \theta) \cdot q_\pi(S, A)]\)(教材 (9.9),Theorem 9.1)。L4 推 4 步:(i) 起 \(J(\theta) = \bar v_\pi\) 或 \(\bar r_\pi\);(ii) 取 \(\nabla_\theta\);(iii) 用 log-derivative trick \(\nabla \pi / \pi = \nabla \ln \pi\);(iv) 凑成 \(\mathbb E[\nabla \ln \pi \cdot q_\pi]\)。这是 PG 与 DQN 在形式 上的分水岭:DQN 学 \(\hat q\) 然后 max,PG 直接用 \(\nabla \ln \pi\) 不需 max。L5 把 \(q_\pi\) 用样本回报 \(G_t\) 替代,得 REINFORCE。
