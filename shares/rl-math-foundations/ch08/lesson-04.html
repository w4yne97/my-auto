L4: Illustrative Examples and Deadly Triad (示例分析 + 致命三联)

§1 Why this lesson

L3 给出 TD-FA 算法形式 \(w \leftarrow w + \alpha \delta_t \nabla_w \hat v(s; w)\),并指出"线性 FA + on-policy ⇒ 收敛到 MSPBE 极小" 这一保证。本节(教材 §8.2.4-8.2.5, pp. 164-179)做两件事:(a) 把算法跑在 5×5 grid-world 上,直观看线性 FA 学出来的 \(\hat v\) 长什么样 —— 多项式基(\(d=3, 6, 10\))与 Fourier 基(\(q=1,2,3\))给出 RMSE 随 episode 单调下降,但受逼近族容量限制,RMSE 不到 0;(b) 把 §8.2.5 的理论分析 摊开:确定型 \(w_{t+1} = w_t + \alpha(b - A w_t)\) 收敛到 \(w^* = A^{-1} b\),其中 \(A = \Phi^\top D(I - \gamma P_\pi)\Phi\) 正定 ⇒ \(\|I - \alpha A\|_2 < 1\) ⇒ 几何收敛;同时摆出"w* = MSPBE 极小" 与误差界 \(\|\Phi w^* - v_\pi\|_D \le \frac{1}{1-\gamma} \min_w \sqrt{J_E(w)}\) 这一实操可怕(\(\gamma\) 接近 1 时上界爆)的事实。

但本节的核心戏剧 在 deadly triad —— Sutton-Barto 命名的三联组合:函数近似 + bootstrap + off-policy。任两条独立都不出问题,三条同时出现可让 \(\|w_k\|\) 指数发散。Baird 反例(7-state star MDP)是纯线性 FA + Q-learning + 错误加权 ⇒ 权重发散 的经典构造:每条都"合法",凑齐三件就炸。本节把这条结构性 警告说严,顺便交代实操缓解(experience replay + target network + 小 stepsize)—— 这正是 L7 DQN 两大 trick 的诞生动因。

§2 核心定义(直觉先行)

  • 多项式特征 (polynomial features,教材 (8.15)-(8.17)):把 \((x, y) = (\text{col}, \text{row})\) 归一化到 \([-1, 1]\) 后,取 \(\phi(s) = [1, x, y]^\top\)(\(d=3\) 平面)、\([1, x, y, x^2, y^2, xy]^\top\)(\(d=6\) 二次面)、或 \(d=10\)(三次)。维度越高,\(\hat v\) 表达力越强,RMSE 下界越低。但 \(\phi\) 是线性 组合,真实 \(v_\pi\) 若非多项式则永远到不了 0。
  • Fourier 特征 (Fourier basis,教材 (8.18)):\(\phi(s) = [\cos(\pi(c_1 x + c_2 y))]\),\(c_1, c_2 \in \{0, \ldots, q\}\),维度 \((q+1)^2\)。\(q=1\) 给 \(d=4\),\(q=3\) 给 \(d=16\)。正交基,理论性能优于多项式。
  • (8.22) 确定型 update:\(w_{t+1} = w_t + \alpha (b - A w_t)\),\(A = \Phi^\top D(I - \gamma P_\pi)\Phi\),\(b = \Phi^\top D r_\pi\)。\(D\) 是稳态分布 \(d_\pi\) 的对角阵。不动点 \(w^* = A^{-1} b\) 即 MSPBE 极小。
  • \(A\) 正定 (positive definite):对任意非零 \(x\),\(x^\top A x > 0\)(教材 Box 8.4 证)。证法用strict diagonal dominance:\(M = D(I - \gamma P_\pi)\) 满足 \((M+M^\top) \mathbf 1 = 2(1-\gamma) d_\pi > 0\),且对角占优 ⇒ \(M + M^\top \succ 0\) ⇒ \(A \succ 0\)(\(\Phi\) 列满秩)。
  • 误差界 (8.32):\(\|\hat v(w^*) - v_\pi\|_D \le \frac{1}{1 - \gamma} \min_w \sqrt{J_E(w)}\)。"逼近族最优" 的 \((1-\gamma)^{-1}\) 倍 是 \(\hat v(w^*)\) 离 \(v_\pi\) 距离的松上界。\(\gamma = 0.99\) 时常数 \(=100\),量纲骇人 —— 实操 \(\hat v\) 远比这界紧,但worst case 警告深刻。
  • Deadly triad (致命三联,Sutton-Barto §11.3):\((1)\) 函数近似(共享 \(w\),不是表式);\((2)\) bootstrap(target 含 \(\hat v(s'; w)\),不是 MC);\((3)\) off-policy(数据加权不 是 \(d_\pi\),如 Q-learning 用 behavior policy 探索)。三者同时 在场 ⇒ \(A\) 不再保证正定 ⇒ 算法可发散(\(\|w_k\| \to \infty\))。
  • Baird 反例 (Baird's counterexample, 1995):7 状态 star MDP,2 个动作(solid 和 dashed),behavior policy 偏好 dashed,target policy 偏好 solid;线性 FA 维度 \(d=8\)。教材正文未细写,但结论简单:在纯线性 FA + Q-learning + off-policy 设置下,权重指数爆炸 —— 算法不仅不收敛,反而越跑越远。这是 deadly triad 不可绕 的实证。
  • 缓解策略 (mitigations):(i) Experience replay:从 buffer 均匀抽样 ⇒ 数据近似 满足"on-policy 加权"(L7 详谈)。(ii) Target network:用 frozen \(w^-\) 算 target ⇒ 截断 bootstrap 自反馈(L7)。(iii) 小 stepsize:让 \(\|I - \alpha A\| < 1\) 即使 \(A\) 近退化也保边际稳。(iv) Gradient TD (Sutton et al., 2009):Bellman residual 的真 梯度 + 第二组参数 ⇒ off-policy 稳收敛(理论上正,实操少见)。

Step 1. "三者缺一不发散" 的因果链。RL 收敛证明的核心引理 是 \(P_\pi^\top d_\pi = d_\pi\)(target policy 的稳态分布在 \(P_\pi\) 下不变),这条让 \(M = D(I - \gamma P_\pi)\) 严格对角占优(Box 8.4)。on-policy 用 \(D_\pi\),引理成立 ⇒ \(A \succ 0\) ⇒ 收敛。off-policy 用 \(D_b\),引理失效。同时 bootstrap 让 target 含 \(P_\pi\),FA 让多状态共享 \(w\) ⇒ 错的加权放大 到所有共享 \(w\) 的状态 ⇒ 发散。每条独立 都不放大,三条凑齐才放大-不收敛。
Step 2. 误差界 \((1-\gamma)^{-1}\) 因子的几何意义。线性 FA 的"最佳逼近" \(\Pi v_\pi\) 在正交投影 意义下离 \(v_\pi\) 最近;但 TD 的不动点 \(\Phi w^*\) 不是 \(\Pi v_\pi\),而是 \(M T_\pi\) 的不动点("先 Bellman 推一步,再投影")。这两个点之间最坏可差 \(\frac{\gamma}{1-\gamma}\) 倍("每过一步贝尔曼,投影偏差被 \(\gamma\) 放缩")。\(\gamma\) 接近 1 时偏差几何累积 ⇒ \((1-\gamma)^{-1}\) 因子。这是"bootstrap 偏差" 在逼近 损耗下的复合效应。
Step 3. Deadly triad 的工程哲学:理论说"三联可发散",但 deep RL 必须 同时用三者(NN + TD + replay buffer 抽样 ≈ off-policy)。怎么办?DQN 的回答是"缓解" 而非"消除":用 target net 削弱 bootstrap 反馈,用 replay 让加权近似 于均匀,用大 batch + 小 lr 让数值上稳。这是实操工程 战胜理论 paranoid 的经典案例 —— 但代价是无理论收敛保证,只有实证(Atari benchmark)。L6-L7 把这套缓解策略逐条展开。

§5 常见误解 / 易混概念

很多人以为:"deadly triad" 是理论 paranoid —— 实操中 NN + Q-learning + replay 跑得很好,所以应该不会真发散。
实际上:Baird 反例是真 发散的实证 —— 7 状态线性 FA + Q-learning,无任何缓解,\(\|w_k\|\) 指数爆炸。deep RL 跑得稳是因为 DQN 额外 加了三道防线(target net + replay + 小 lr),不是 deadly triad 不存在。一旦把这些 trick 关掉(比如直接 NN + 单步 Q-learning + 在线训练),实测训练曲线发散 是常态(Mnih et al., 2015 的 ablation 验证过)。
因为:RL 与监督学习的本质区别:监督学习有 i.i.d. 数据 + 固定 target,SGD 收敛性是泛函分析标准结果。RL 数据由策略 induced(non-iid),target 含当前估计器(non-stationary)。三件事(FA + bootstrap + off-policy)各自破坏一条监督学习的前提,凑齐三条 ⇒ 完全脱离 SGD 收敛理论 ⇒ 发散是默认行为,收敛才是需要工程构造 的现象。
很多人以为:"线性 FA on-policy 收敛到 MSPBE 极小 ⇒ MSPBE 极小就是 MSE 极小 ⇒ \(\hat v(w^*) \approx v_\pi\)。"
实际上:MSPBE 极小不 是 MSE 极小。两个最优点在族错配(\(v_\pi \notin\) 线性族)时不重合,差距由 (8.32) \(\frac{1}{1-\gamma}\min \sqrt{J_E}\) 控制。\(\gamma\) 接近 1 时差距可很大。L2 已用数值例子展示:\(d=2\) 时 MSE 极小 \(w^* = (-9.5, 1.95)\),MSPBE 极小 \(w^* = (-10.2, 2.05)\),两者差 0.5+。这不是 bug,是 TD-FA 的结构特征。
因为:TD 用 bootstrap target,目标函数是 \(J_{\text{PBE}} = \|\hat v - \Pi T_\pi \hat v\|_D^2\) —— "先贝尔曼一步再投影" 的不动点。这与"正交投影 \(v_\pi\)" 在族错配时不重合。MC 用真回报,目标是 \(J_E = \|\hat v - v_\pi\|_D^2\),收敛点恰是正交投影。两者在 \(v_\pi \in\) 族 时重合。
很多人以为:"experience replay 是为了 data efficiency(每条样本用多次)。"
实际上:data efficiency 只是副效益;主效益 是打散样本时间相关性。原始 RL 数据 \((s_t, a_t, r_t, s_{t+1})\) 在时间上高度相关(连续步同一轨迹),违反 SGD 的"独立样本" 假设;replay buffer 均匀抽样 ≈ i.i.d.,把数据加权近似 拉到均匀分布。这一近似 on-policy 让 deadly triad 的"off-policy 加权" 那条缓解(虽不严格消除)。data efficiency 只是抽样多次的顺带 好处。
因为:L7 详细展开。目标函数 (8.37) 假设 \((S, A) \sim \text{Uniform}\),behavior policy 数据不 是均匀;replay buffer + 均匀抽样把"样本路径分布" 撬到"近似 buffer 自身分布",而 buffer 是很多 episode 的混合 ≈ 均匀。这是 mathematical 的"把 (8.37) 期望近似算对",不是工程巧思。

§6 自检 + 视频 + 下一节

Q1. 写出 deadly triad 的三条,并解释"缺一不发散" 的因果链。

A. 三条:(1) 函数近似(共享参数 \(w\));(2) bootstrap target(\(r + \gamma \hat v(s'; w)\),不是 MC 回报);(3) off-policy(数据加权 \(D_b\) 不是 \(D_\pi\))。缺一不发散:去掉 FA ⇒ 表式,每格独立无放大;去掉 bootstrap ⇒ MC,target 不依赖当前 \(w\) 无反馈;去掉 off-policy ⇒ on-policy,\(d_\pi^\top P_\pi = d_\pi\) 引理让 \(A \succ 0\)。三条凑齐 ⇒ \(A\) 不再正定 + 共享 \(w\) 放大错的加权 + bootstrap 自反馈 ⇒ 权重指数爆炸(Baird 反例)。

Q2. 教材 (8.32) 给出 \(\|\Phi w^* - v_\pi\|_D \le \frac{1}{1-\gamma}\min_w \sqrt{J_E}\)。\(\gamma=0.99\) 时 \(\frac{1}{1-\gamma}=100\),为什么实操不爆?

A. (8.32) 是最坏情况 上界,实测远松。原因:(i) \(\min_w J_E\) 在合理 \(\phi\) 下很小(实测 grid-world 上 \(d=10\) 多项式 \(\min J_E \approx 0.1\),100×0.1 = 10 仍合理);(ii) 上界证明用了几次三角不等式 + Cauchy-Schwarz,常数放大 严重;(iii) 实际 \(\hat v(w^*)\) 在常见 状态(\(d_\pi\) 高密度)拟合好,只在边角 误差大。所以 (8.32) 是结构警告(γ→1 + 族容量小 ⇒ 危险),不是实操精确预测。

Q3. 为什么experience replay 缓解 deadly triad?它消除 了 deadly triad 吗?

A. Replay buffer 把多个 episode 的样本混合存储,均匀抽样 ⇒ 经验分布接近buffer 平均(很多策略历史的混合,近似均匀)。这让 (8.37) 中"(S, A) ~ Uniform" 的假设近似成立,在数学上把"off-policy 严重加权失配" 缩成"轻微加权失配"。但不消除:buffer 仍由 behavior policy 写入(off-policy 仍在);bootstrap target 仍含 \(\hat q(s', a; w)\)(target net 减轻但未消);FA 仍是 NN(共享 \(w\) 仍在)。三条都还在,只是每条都被减弱,合力让实操稳。这就是 DQN 工程哲学:"缓解三条都让发散变缓,慢到能在训练 episode 数内收敛"。
