L6: Q-learning Introduction (max 不 sample — 直接学 q*)

§1 Why this lesson

Sarsa(L4)与 Expected Sarsa(L5)在 \(\delta_t\) 中"下一状态 value" 用当前 policy(sampled 或 expected)的 \(Q\)。本节(教材 §7.4.1-7.4.2)把这一项第三种处理:\(\max_a Q(s_{t+1}, a)\)。(1) 算法名 Q-learning。(2) 学到的是 \(q^*\)(最优 \(q\)),不是 \(q_\pi\) ⇒ off-policy:behavior policy(采数据)与 target policy(被估 = greedy w.r.t. \(Q\))分离。(3) 这是 deep RL 时代 DQN 与一族经典算法的种子。

核心 takeaway:(a) Q-learning \(\delta_t = r_{t+1} + \gamma \max_a Q(s_{t+1}, a) - Q(s_t, a_t)\)。(b) "max" 把 \(\delta_t\) 中的"下一动作 value" 计算成"greedy policy 上的值",独立于 behavior policy ⇒ off-policy。(c) 收敛点 = Bellman 最优算子 \(T^*\) 的固定点 = \(q^*\)。(d) 三种"下一动作处理" 视觉化:Sarsa(sampled)、Expected Sarsa(weighted by π)、Q-learning(max)。

§2 核心定义(直觉先行)

  • Q-learning 更新(教材 (7.31)):\(Q(s_t, a_t) \leftarrow Q(s_t, a_t) + \alpha_t [r_{t+1} + \gamma \max_a Q(s_{t+1}, a) - Q(s_t, a_t)]\)。结构与 Sarsa 一字差,只把 sampled \(a_{t+1}\) 改成 \(\arg\max_a Q(s_{t+1}, a)\)。
  • Q-learning \(\delta_t\):\(\delta_t = r_{t+1} + \gamma \max_a Q(s_{t+1}, a) - Q(s_t, a_t)\)。等价于"从下一状态起,假设之后总走 greedy policy" 时的 Bellman 最优残差。
  • off-policy 含义:behavior policy(采数据用,如 \(\epsilon\)-greedy)与 target policy(被估的策略,即 greedy w.r.t. \(Q\))分离。Q-learning 学的是 target policy 的 \(q\),不论 behavior 是什么。
  • Bellman 最优方程的固定点:\(q^*(s,a) = \mathbb E[R + \gamma \max_{a'} q^*(s', a') | s, a]\)。Q-learning 的 \(\delta_t\) 期望正 是 \(g(Q)(s,a) = Q(s,a) - (T^* Q)(s,a)\),根 \(Q = q^*\)。
  • 与 Sarsa 的关键单点对比:Sarsa 用 \(Q(s_{t+1}, a_{t+1}^{\text{sampled}})\);Q-learning 用 \(\max_a Q(s_{t+1}, a)\)。sampled 是 \(\pi\) 上的随机抽,max 是 \(\pi_*\) 上的确定挑。"谁负责挑下一动作" 决定 on/off-policy。
  • off-policy 的工程价值:可复用旧数据(replay buffer)、从 demonstrations 学、同时探索与利用。这是 DQN/Rainbow/SAC 等深度 RL 算法的基础。

§3 关键公式与拆解

本节核心是"下一动作处理" 的三种结构对比:Sarsa(sampled)、Expected Sarsa(\(\sum_a \pi\))、Q-learning(\(\max_a\))。

公式:Q-learning vs Sarsa vs Expected Sarsa 的 \(\delta_t\)

"下一动作 value" 三种处理的可视化对比(动作空间 \(\mathcal A = \{a_1, a_2, a_3, a_4\}\) 在 \(s_{t+1}\) 处的 \(Q\) 表):

Step 1. "max" 让 Q-learning 不依赖 behavior policy。Sarsa 的 \(a_{t+1}\) 与 Expected Sarsa 的权重 \(\pi(a|s_{t+1})\) 都涉及"当前 \(\pi\)"。Q-learning 的 \(\max_a\) 只用当前 \(Q\),不引用任何 \(\pi\) ⇒ 即使 behavior 是纯随机,target 仍朝向 \(q^*\)。所以同一份样本数据,可在不同 behavior 下复用 ⇒ off-policy 性。
Step 2. Q-learning 收敛到 \(q^*\) 的 RM 视角。\(g(Q)(s,a) = Q(s,a) - (T^* Q)(s,a)\),其中 \(T^*\) 是 Bellman 最优算子(Ch.3 已熟)。\(\tilde g_t = Q(s_t, a_t) - [r + \gamma \max_a Q(s_{t+1}, a)] = -\delta_t\)。代入 RM:\(Q \leftarrow Q + \alpha \delta_t\)。三条件 + 每 \((s,a)\) 无穷访问 ⇒ \(Q \to q^*\)(Watkins-Dayan 1992,套 Theorem 6.3 多变量 RM)。L7 详细给伪代码与"每 \((s,a)\) 无穷访问" 的 behavior 设计。
Step 3. maximization bias 警告。\(\max_a Q(s, a)\) 是有偏估计 \(\max_a q^*(s,a)\) 的:\(\mathbb E[\max_a \hat Q] \ge \max_a \mathbb E[\hat Q]\)(Jensen),即使 \(\hat Q\) 无偏,\(\max\) 仍把正噪声 放大。这导致 Q-learning 早期阶段过高估 \(q^*\),收敛慢。Double Q-learning(Hado van Hasselt 2010)用两套 \(Q\) 解耦"选 action" 与"估 value" 修这一 bias。L7 / Ch.8 DDQN 处理。

§4 Grid-world 例子:Q-learning vs Sarsa cliff-walking 经典对比

5×5 grid,target=(3,2),"悬崖式" forbidden = {(1,1),(2,1),(2,2),(1,3),(3,3)}(踩入 \(r=-10\))。\(\epsilon\)-greedy 探索 \(\epsilon=0.1\)。

解读:Q-learning 学到的 greedy policy 走"贴边 cliff 旁的最短路"(因 target = greedy,不计 \(\epsilon\) 探索风险);Sarsa 学到的稍远 cliff 的"安全路"(因 target = sampled,把"探索误入 cliff" 的 \(-10\) 计入)。同一 ε-greedy behavior,两个算法学到不同 policy —— 这是"评估当前 policy vs 评估最优 policy" 的结构后果。L7 详细做 cliff-walking。

§5 常见误解 / 易混概念

很多人以为:"Q-learning 既然学 \(q^*\),就不 需要 behavior 探索。"
实际上:Q-learning 仍需每 \((s,a)\) 无穷访问,这必须 由 behavior 探索保证。target 的 \(\max_a Q(s_{t+1}, a)\) 是 greedy,但要让 Q-learning 在 \((s,a)\) 上更新 \(Q\),必须实际访问到 \((s,a)\)。若 behavior 也 greedy ⇒ 永远不试其他 action ⇒ 某些 \(Q\) 永不更新 ⇒ 不收敛。所以 Q-learning 需要 \(\epsilon\)-greedy 等有探索的 behavior。
因为:"off-policy" 不等于"免探索"。off-policy 只是把"采数据的 policy" 与"被估的 policy" 分开,前者仍要充分覆盖状态-动作空间。这是 RM 第二条件(每分量步长两条件 + 每分量更新无穷次)在 RL 上的结构要求。
很多人以为:"\(\max_a Q\) 的 max 让 Q-learning 估计偏高,Q-learning 不可靠。"
实际上:maximization bias 是真实问题,但有结构化解:Double Q-learning(L7 / Ch.8 DDQN)把"选 action" 与"估 value" 解耦,用两套 \(Q_1, Q_2\) 互相估,bias 大幅消减。tabular Q-learning 在小动作空间上偏差有限,实操问题不大;deep RL(动作空间大、网络拟合)bias 严重 ⇒ DDQN 是 DQN 改进的核心一笔。
因为:"max 偏高" 是有限样本 现象,渐近来看 \(Q \to q^*\) 仍成立(Watkins-Dayan 收敛定理)。但有限样本下 偏差让 deep RL 高估 \(Q\) ⇒ policy 选错 action ⇒ 性能恶化。Ch.8 DDQN 详谈。

§6 自检 + 视频 + 下一节

Q1. 写出 Q-learning 的 \(\delta_t\),并指出"max 不 sample" 让它从 Sarsa 的 on-policy 变成 off-policy 的根本原因。

A. \(\delta_t = r_{t+1} + \gamma \max_a Q(s_{t+1}, a) - Q(s_t, a_t)\)。"max" 让 target 计算"下一动作 value" 不引用任何 \(\pi\),只用当前 \(Q\),等价于"用 greedy(Q) 这一固定 target policy 评估" ⇒ behavior(采数据 policy)可任意,target policy 锁定为 greedy(Q)。分离 ⇒ off-policy。Sarsa 的 sampled \(a_{t+1}\) 来自当前 \(\pi\) ⇒ behavior 与 target 锁住 ⇒ on-policy。

Q2. Q-learning、Sarsa、Expected Sarsa 三者在 \(s_{t+1}\) 处对"下一动作 value" 的处理各是什么?学到的固定点分别是什么?

A. Sarsa: \(Q(s_{t+1}, a_{t+1}^{\text{sampled}})\) — 单点采样(\(a_{t+1} \sim \pi\));固定点 \(q_\pi\)(on-policy)。Expected Sarsa: \(\sum_a \pi(a | s_{t+1}) Q(s_{t+1}, a)\) — \(\pi\) 加权期望;固定点 \(q_\pi\)(on-policy)。Q-learning: \(\max_a Q(s_{t+1}, a)\) — greedy 选;固定点 \(q^*\)(off-policy,target = greedy(Q))。
