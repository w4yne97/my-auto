L2: TD Algorithm Introduction (TD 算法 + δ_t 灵魂表达式)

§1 Why this lesson

L1 把 (7.3) 写成"估 \(\mathbb E[R + \gamma v(X')]\)" 的递推种子。本节(教材 §7.1.2)把它正式落成 TD(0) 算法,并引入整个 Ch.7-Ch.10 最常被引用的单一表达式:TD error \(\delta_t = r_{t+1} + \gamma V(s_{t+1}) - V(s_t)\)。后续 Sarsa、Expected Sarsa、Q-learning 都只是 \(\delta_t\) 中"下一状态 value 怎么算" 的变奏 —— 所以 \(\delta_t\) 在本节定义一次,Ch.7-Ch.8 反复复用。

核心 takeaway:(a) TD(0) 更新公式:\(V(s_t) \leftarrow V(s_t) + \alpha_t \delta_t\)。(b) \(\delta_t\) 是带噪 Bellman 残差:\(\mathbb E[\delta_t | S_t = s] = v_\pi(s) - V(s) - 0\)?不,准确说 \(\mathbb E[\delta_t | S_t = s, V] = (T_\pi V)(s) - V(s)\),即当前估计与 Bellman 算子一次作用后的差。\(\delta_t \to 0\) ⇔ \(V = T_\pi V\) ⇔ \(V = v_\pi\)。(c) 算法 update 等价于"把 \(V(s_t)\) 往 target \(r_{t+1} + \gamma V(s_{t+1})\) 移动 \(\alpha_t\) 比例"。(d) 与 MC 的对照:MC 用 \(G_t\) 取代 target,target 无 bias 但需等 episode。

§2 核心定义(直觉先行)

  • TD(0) 算法(教材 (7.5)/(7.6)):每收一条样本 \((s_t, r_{t+1}, s_{t+1})\),做 \(V(s_t) \leftarrow V(s_t) + \alpha_t [r_{t+1} + \gamma V(s_{t+1}) - V(s_t)]\)。未访问到的状态不更新。
  • TD target(教材 §7.1.2):\(\bar V_t \doteq r_{t+1} + \gamma V(s_{t+1})\)。"一步 reward + 下一状态当前估计"。这是 Bellman 期望右端的一次单样本估计。
  • TD error \(\delta_t\)(全章灵魂):\(\boxed{\delta_t \doteq r_{t+1} + \gamma V(s_{t+1}) - V(s_t)}\)。即当前 target 与当前估计的差。4 个组件:\(r_{t+1}\)(即时 reward)、\(\gamma V(s_{t+1})\)(打折后的下一状态值)、\(V(s_t)\)(当前估计)、\(\delta_t\) 是它们的代数差。
  • 性质 1:零均值条件。\(\mathbb E_\pi[\delta_t | S_t = s] = (T_\pi V)(s) - V(s)\)。当 \(V = v_\pi\) 时,\(T_\pi v_\pi = v_\pi\) ⇒ \(\mathbb E[\delta_t] = 0\)。所以 \(\delta_t\) 是"偏离 \(v_\pi\) 的带噪测度"。
  • 性质 2:更新方向。\(\delta_t > 0\) ⇒ target 比当前估计大 ⇒ \(V(s_t)\) 上调;\(\delta_t < 0\) ⇒ 下调。这与 SGD 沿负梯度的更新结构相同(L3 揭示具体 \(g\))。
  • "未访问状态不动"。TD 是状态局部 更新。每步只改 \(V(s_t)\) 一格。这与 Ch.4 DP 的同步全表更新 形成结构对比 —— 本质是用样本接触状态空间。

§3 关键公式与拆解

本节核心是\(\delta_t\) 的定义:整个 Ch.7-Ch.8 反复复用,值得一次性把 4 个组件画清。

公式:TD error \(\delta_t\) 与 TD(0) 更新

Step 1. 把 TD(0) 写成 RM 形态。L1 §3 已铺好:令 \(g(V)(s) \doteq V(s) - (T_\pi V)(s) = V(s) - \mathbb E[r_{t+1} + \gamma V(s_{t+1}) | S_t = s]\)。带噪估计 \(\tilde g_t \doteq V(s_t) - [r_{t+1} + \gamma V(s_{t+1})] = -\delta_t\)。代入 RM:\(V(s_t) \leftarrow V(s_t) - \alpha_t \tilde g_t = V(s_t) + \alpha_t \delta_t\)。这正 是 TD(0)。所以 TD(0) ≡ RM 在 Bellman 算子的固定点上。
Step 2. \(\delta_t\) 的条件零均值。给定 \(S_t = s\) 与当前 \(V\),\(R_{t+1}\) 与 \(S_{t+1}\) 由环境分布 \(p(s',r|s,\pi(s))\) 生成。\(\mathbb E[\delta_t | S_t = s] = \mathbb E[r + \gamma V(s') | s] - V(s) = (T_\pi V)(s) - V(s)\)。当 \(V = v_\pi\) 时 \(T_\pi v_\pi = v_\pi\) ⇒ 零均值。所以 \(\delta_t\) 是"偏离 \(v_\pi\)" 的带噪诊断量,RM 第三条件天然满足。
Step 3. \(\delta_t\) 在 Ch.7 全章是名词。后面所有讨论"哪种 \(\delta\)" 时,只需指明下一状态 value 怎么算:Sarsa \(= Q(s_{t+1}, a_{t+1})\)、Expected Sarsa \(= \sum_a \pi(a|s_{t+1}) Q\)、Q-learning \(= \max_a Q\)、TD(0) \(= V(s_{t+1})\)。4 个算法,4 个 \(\delta\) 变体,1 个共享骨架。这是 Ch.7 给学习者最大的"认知压缩"。

§4 数值演示:TD(0) 单步更新如何推动 V 向 v_π 走

本节不用 grid-world(纯算法骨架,空间布局不增信息)。用 iteration-stepper 把 6 步 \((s_t, r_{t+1}, s_{t+1})\) 样本逐帧展示,每帧给出 \(\delta_t\) 与 \(V\) 表更新,直观看 \(V\) 收敛到 \(v_\pi\) 的过程。

§5 常见误解 / 易混概念

很多人以为:"TD(0) 收敛快是因为更新次数多。"
实际上:真正原因是信息的局部传播。每步只改 \(V(s_t)\) 一格,但 \(V(s_t)\) 一动,下次 \(s_{t-1}\) 经过时它的 target(\(r + \gamma V(s_t)\))就更新了 —— bootstrap 让已学到的信息沿轨迹反向接龙。MC 没有这种接龙(target 只用真采样),因此 long horizon 任务里 TD 信息利用率几倍于 MC。
因为:"更新次数" 是表象;"每次更新带的信息含量" 才是本质。TD 的 \(\delta_t\) 把已经学到的下一状态估计 也用上了,等于"每步用的不止当前 reward,还有过去所有步的累积学习"。这是 RL 中"样本效率" 的结构来源。
很多人以为:"\(\delta_t = 0\) 就代表算法收敛。"
实际上:需要对所有 \(s\) 经常性\(\delta_t \to 0\)。单条样本 \(\delta_t = 0\) 可能只是巧合(reward 噪声平衡)。算法收敛要求条件期望 \(\mathbb E[\delta_t | S_t = s, V] \to 0\) 对所有 \(s\) —— 即 \(V = T_\pi V\) ⇔ \(V = v_\pi\)。这是 Bellman 算子固定点的同义 表述,L3 用 RM 严证。
因为:"样本 \(\delta\)" 与"期望 \(\delta\)" 不同。前者带噪,后者无噪;算法看的是长期平均。这条区分是 RM 第三条件\(\mathbb E[\eta|\mathcal H]=0\)的本质要求,Ch.6 L7 已强调。

§6 自检 + 视频 + 下一节

Q1. 写出 \(\delta_t\) 的 4 个组件,并说明 \(\delta_t > 0\) 时 \(V(s_t)\) 的变化方向。

A. \(\delta_t = r_{t+1} + \gamma V(s_{t+1}) - V(s_t)\)。组件:即时 reward \(r_{t+1}\);打折下一估计 \(\gamma V(s_{t+1})\);当前估计 \(V(s_t)\)。\(\delta_t > 0\) ⇒ target 大于当前估计 ⇒ \(V(s_t)\) 通过 \(V \leftarrow V + \alpha\delta\) 上调。反之 下调。

Q2. 把 TD(0) 写成 RM 形态:找出 \(g\)、\(\tilde g\)、\(\eta\)。

A. \(g(V)(s) = V(s) - (T_\pi V)(s) = V(s) - \mathbb E[r + \gamma V(s') | s]\),其根 \(V^* = v_\pi\)。\(\tilde g_t = V(s_t) - [r_{t+1} + \gamma V(s_{t+1})] = -\delta_t\)。噪声 \(\eta_t = \tilde g_t - g(V)(s_t) = (T_\pi V)(s_t) - [r_{t+1} + \gamma V(s_{t+1})]\),条件零均值。RM:\(V \leftarrow V - \alpha\tilde g = V + \alpha\delta\)。
