L2: Robbins-Monro Algorithm — Introduction (Robbins-Monro 算法:看不见全貌也能解方程)

§1 Why this lesson

L1 把均值估计写成 \(w_{k+1} = w_k - \alpha_k(w_k - x_k)\),并暗示这是解 \(g(w) = w - \mathbb E[X] = 0\) 用带噪样本。本节(教材 §6.2.1,pp. 103-105)把这个观察做正式定义:给定一个未知函数 \(g\),只能拿到带噪观测 \(\tilde g(w, \eta) = g(w) + \eta\),如何用迭代算法找根 \(g(w^*) = 0\)?Robbins-Monro 算法 (RM,1951) 给出标准答案,并奠定整个随机近似 学科。

关键洞察(教材 §6.2 + (6.5)):(a) "看不见 \(g\),只能在你递入 \(w\) 时拿到带噪输出" —— 这是黑箱 root-finding(Figure 6.2)。(b) RM 算法 \(w_{k+1} = w_k - a_k \tilde g(w_k, \eta_k)\)(教材 (6.5)) —— 与 L1 (6.4) 同结构,但 \(g\) 不再限定为 \(w - \mathbb E[X]\),而是任意 满足条件的函数。(c) RM 不需要 \(g\) 的解析表达、不需要梯度、不需要 \(g\) 的概率分布 —— 只需要"递入 \(w\) 取一次带噪输出" 的能力。这是 RM 比传统 root-finding(Newton 法、bisection)更强 的地方。(d) 教材 \(g(w) = w^3 - 5\) 的例子(Figure 6.3):真根 \(5^{1/3} \approx 1.71\),起点 \(w_1 = 0\),用 \(a_k = 1/k\) 跑 50 步,\(w_k\) 在噪声中平稳逼近 1.71。(e) 朴素直觉(教材 \(g = \tanh(w-1)\) 例,Figure 6.4):\(w_k > w^*\) ⇒ \(g(w_k) > 0\) ⇒ 更新让 \(w\) 下降;\(w_k < w^*\) ⇒ \(g(w_k) < 0\) ⇒ \(w\) 上升。这是"负反馈" 的最朴素形式。(f) "为什么能扛住噪声" 是 L3 RM 收敛性 + Dvoretzky 定理的工作 —— 本节先不证,先把动力学直觉 立住。

§2 核心定义(直觉先行)

  • Stochastic approximation (随机近似,教材 §6.2 开头):一类用带噪样本迭代解 root-finding 或优化问题 的算法的总称。共同结构:\(w_{k+1} = w_k - a_k \cdot \text{(关于 } g(w_k) \text{ 的带噪估计)}\)。RM 是它的开山之作(1951),SGD 是它的明星特例(L4-L7)。
  • RM problem statement (教材 §6.2):已知方程 \(g(w) = 0\),其中 \(w \in \mathbb R\)、\(g: \mathbb R \to \mathbb R\) 但函数表达未知。可观测:递入任意 \(w\),拿到 \(\tilde g(w, \eta) = g(w) + \eta\)(\(\eta\) 是观测噪声)。求 \(w^*\) 使 \(g(w^*) = 0\)。
  • RM algorithm (教材 (6.5)):\(w_{k+1} = w_k - a_k \tilde g(w_k, \eta_k),\ k = 1, 2, 3, \dots\)。\(w_k\) 是第 \(k\) 步估计;\(\tilde g(w_k, \eta_k)\) 是用 \(w_k\) 取一次 带噪观测;\(a_k > 0\) 是步长(coefficient)。注意:RM 用的是当前 \(w_k\) 的观测,不是固定的初始观测 —— 这是它能"跟着估计走" 的关键。
  • RM 不需要什么(教材 §6.2 中段):(i) 不需要 \(g\) 的解析表达(可以是黑箱、神经网络);(ii) 不需要 \(\nabla g\)(grad-free);(iii) 不需要 \(\eta\) 的分布(non-Gaussian 也行);(iv) 不需要 \(\eta\) 与 \(w_k\) 独立(允许 \(\eta\) 依赖历史 \(\mathcal H_k\),L3 给条件)。需要的只是"递入 \(w\) 取输出" 的接口。
  • 把优化转成 root-finding(教材 §6.2):若目标是 \(\min_w J(w)\),设 \(g(w) \doteq \nabla_w J(w)\) ⇒ 最优解 \(w^* = \arg\min J\) 满足 \(g(w^*) = 0\)。RM 用"带噪梯度估计"\(\tilde g(w, x) = \nabla_w f(w, x)\)(\(J(w) = \mathbb E_X[f(w, X)]\)) ⇒ 直接退化成 SGD(L4 详谈)。
  • "负反馈" 朴素直觉(教材 Figure 6.4,\(g = \tanh(w-1)\)):\(g\) 单增,\(g(w^*) = g(1) = 0\)。设无噪声 \(\eta_k \equiv 0\),起点 \(w_1 = 3\):\(g(3) = \tanh(2) \approx 0.96\) ⇒ \(w_2 = 3 - 1 \cdot 0.96 \approx 2.04\);\(g(2.04) \approx \tanh(1.04) \approx 0.78\) ⇒ \(w_3 = 2.04 - 0.5 \cdot 0.78 \approx 1.65\)…逐步逼近 1。无噪声版的 RM = 普通 fixed-point iteration。
  • RM 与梯度下降的关系(教材 §6.2 末段提示):若 \(g(w) = \nabla J(w)\)、\(\tilde g\) 是 \(\nabla J\) 的带噪估计,则 RM ≡ SGD。反过来不真:RM 可以解非梯度 的根问题(如 \(g(w) = w^3 - 5\),无明显的"对应优化目标")。RM ⊃ SGD(集合包含)。

Step 1. 无噪声版 RM = fixed-point iteration(教材 Figure 6.4 直观)。设 \(\eta_k \equiv 0\),则 \(w_{k+1} = w_k - a_k g(w_k)\) 是常微分方程 \(\dot w = -g(w)\) 的离散化(Euler 法)。\(g\) 单增 ⇒ \(w^*\) 是稳定不动点(\(g'(w^*) > 0\) ⇒ 线性化系统稳定)。所以无噪声版 RM 的收敛性就是 ODE 的稳定性,不需要"新数学"。新数学要等 \(\eta\) 来添麻烦后才登场。
Step 2. 带噪声为什么不会"放大"?关键在 \(a_k \to 0\)。每步带噪扰动量 \(= a_k \eta_k\),\(a_k \to 0\) ⇒ 新加入的噪声越来越小。同时 \(\sum a_k = \infty\)(L3 第二条件)保证累积学习能力不被早期消耗光。"越走越稳 + 累积能力够" 这两个矛盾诉求被 \(\sum a = \infty, \sum a^2 < \infty\) 同时满足 —— 这是 \(a_k = 1/k\) 这类调度的魔法所在。L3 用 Dvoretzky 定理形式化。
Step 3. RM 算法的黑箱 视角(教材 Figure 6.2)。把 \(g\) 看成不可见的黑盒:你递入 \(w\),它返回 \(g(w) + \eta\)。RM 不查看 \(g\) 的内部结构,只调用 它的输出。这与梯度下降 不同 —— 后者要求 \(\nabla J\) 已知或可解析求得。RM 的黑箱 性质让它在两个场景特别强:(i) \(g\) 来自真实物理 / 随机系统(无解析式);(ii) RL —— 我们不知道 \(p(s'|s,a)\),但能"取样"\(s'\),正是 RM 的调用接口。

§4 RM 算法的轨迹(用 \(g = \tanh(w-1)\) 演示)

本章不用 grid-world。我们用教材 Figure 6.4 的具体例子 \(g(w) = \tanh(w-1)\),\(w^* = 1\),\(w_1 = 3\),\(a_k = 1/k\),\(\eta_k \equiv 0\)(简化,只看 RM 的动力学骨架)。逐步跟踪 \(w_k\) 的演化。

关键观察:(a) 远处更新大、近处更新小,自动调节,无需手动调参;(b) 步长 \(a_k = 1/k\) 既保证"不跑飞"(\(\sum a^2 < \infty\),后期噪声衰减),又保证"能持续学"(\(\sum a = \infty\),累积步长无界);(c) 本节没证 收敛 —— L3 用 Dvoretzky 定理给严证。但动力学 已经直观了。

§5 常见误解 / 易混概念

很多人以为:"RM 算法不就是梯度下降?\(\tilde g\) 就是带噪梯度。"
实际上:RM 比梯度下降更广。RM 解的是 \(g(w) = 0\) 的任意 root-finding 问题,\(g\) 不必是某个 \(J\) 的梯度。例如 \(g(w) = w^3 - 5\) 没有"对应的标量目标 \(J\)"(\(\int g \, dw = w^4/4 - 5w\) 不是优化目标,无下界)。\(g(w) = w - \mathbb E[X]\) 在 L1 中也是"定义出来的" 而非"梯度"。RM ⊃ SGD(集合包含):SGD 是 RM 在"\(g = \nabla J\) + \(J = \mathbb E[f]\)" 这个特例上的表现。
因为:"解方程" 与"优化" 本是不同问题:前者要找"0 点",后者要找"极小值"。两者关联但不等价 —— 凸优化的极小值是梯度的零点,但解方程 不需要凸结构。Ch.7 TD 学习就是 RM 的非梯度 应用 —— Bellman 算子的固定点 \(v_\pi\) 是 \(g(v) = v - T_\pi v = 0\) 的根,这不是 任何 \(J\) 的梯度的根(原始 TD 学习并没有 一个全局损失函数,这是 deadly-triad 问题的源头之一,Ch.8 详谈)。
很多人以为:"RM 既然是 1951 年 的老算法,直接换成现代 Newton 法或 Adam 不就全面更优?"
实际上:RM 在黑箱 设定下仍是 SOTA。Newton 法要 \(g\) 与 \(\nabla g\) 都已知,不能用于"只能取一次带噪输出" 的场景。Adam / RMSProp 等 自适应步长 算法是SGD 的改进(优化梯度方差),仍属于 RM 大家族(\(\tilde g\) = 带噪梯度),只是把固定 \(a_k\) 换成逐参数自适应 的步长。它们和 RM 本质不冲突,而是RM 的工程优化。
因为:RM 的价值 在于"定义了正确的接口与收敛条件",而不是"给出最快的算法"。Adam 算法的收敛证明 仍然要走 RM 框架(在原 paper 中通过 Dvoretzky-style 论证)。Ch.8 deep RL 用 Adam,但能用 是因为 RM 框架兜底。
很多人以为:"RM 必须用 \(a_k = 1/k\),其他步长都不行。"
实际上:RM 接受任意 满足两条件的步长(L3)。\(a_k = 1/k\) 是经典默认,但 \(c/k\)、\(c/k^\beta\)(\(\beta \in (0.5, 1]\))、\(c/(k+k_0)\) 都满足条件。常步长 \(a_k \equiv 0.1\) 违反 \(\sum a^2 < \infty\),不能严格 收敛到 \(w^*\),但能"稳定在 \(w^*\) 附近振荡",L1 §4 (C) 已演示。实操:(i) 平稳分布 + 求精确 \(w^*\) ⇒ 用衰减 \(a_k\);(ii) 非平稳 + 跟踪移动 target ⇒ 用常 \(a_k\)(Q-learning、SARSA)。
因为:"收敛到固定点" 与"跟踪移动目标" 是对立 诉求(L1 §5 已述)。RM 的两条件只保证前者;放弃前者换得后者,是 RL 实操的常态。L3 给的是下界,不是实操菜单。

§6 自检 + 视频 + 下一节

Q1. RM 算法 \(w_{k+1} = w_k - a_k \tilde g(w_k, \eta_k)\) 与 fixed-point iteration \(w_{k+1} = T(w_k)\) 的关系?

A. RM 是 fixed-point iteration 的带噪 + 步长 推广。设 \(T(w) \doteq w - g(w)\),则无噪声版 RM(\(\eta = 0\))在 \(a_k \equiv 1\) 时变成 \(w_{k+1} = w_k - g(w_k) = T(w_k)\),即 fixed-point iteration。RM 的两个新东西:(i) 步长 \(a_k\) 可变(衰减或常数);(ii) 用带噪 \(\tilde g\) 替代 \(g\)。这两个推广让 RM 能解 fixed-point iteration 不能解的问题(噪声采样、未知函数)。Ch.4 VI 就是纯 fixed-point iteration(\(v_{k+1} = T v_k\)),Ch.7 TD 就是带噪 + 步长 的版本(\(V \leftarrow V + \alpha[T V - V]_\text{sample}\))—— 后者本质是 RM。

Q2. 若 \(g\) 是下降 函数(\(\nabla g(w) < 0\) 而非 \(> 0\)),RM (6.5) 还能用吗?如何改?

A. 不能直接用。\(g\) 下降时 \(w_k > w^*\) 推出 \(g(w_k) < 0\),则更新 \(-a_k g(w_k) > 0\),\(w\) 反而远离 \(w^*\) —— "正反馈" 发散。修正:把 \(g\) 替换成 \(-g\),即用 \(\tilde g' \doteq -\tilde g\) 跑 RM,等价于把更新写成 \(w_{k+1} = w_k + a_k \tilde g(w_k, \eta_k)\)(注意符号翻转)。教材 §6.2.2 第一条件直接要求 \(\nabla g > 0\),就是为了避免这个歧义 —— 实操中遇到下降函数,人工翻号即可。这就像梯度下降 vs 梯度上升:本质是同一算法,符号约定不同。

Q3. 教材 \(g(w) = w^3 - 5\) 例子(Figure 6.3)说"初始猜 \(w_1\) 必须 properly selected"。为什么?\(\tanh(w-1)\) 例(Figure 6.4)却没这条要求?

A. 关键差别在\(\nabla g\) 是否全局有界。\(g = \tanh(w-1)\):\(\nabla g = \text{sech}^2(w-1) \in (0, 1]\),全局上下有界,满足 L3 第一条件 \(0 < c_1 \le \nabla g \le c_2\) ⇒ 任意 初始 \(w_1\) 都能收敛。\(g = w^3 - 5\):\(\nabla g = 3w^2\),在 \(w \to \pm \infty\) 时 \(\nabla g \to \infty\),不 满足 L3 第一条件 ⇒ 若 \(w_1\) 太远(如 \(w_1 = 100\)),则 \(\tilde g(w_1) \approx 10^6\),\(a_1 \tilde g \approx 10^6\) 把 \(w_2\) 弹到极远处,数值上"炸开"。实操:(i) 用合理初始(如 \(w_1 = 0\)),让 \(g(w_1)\) 有界;(ii) 加 gradient clipping(\(\tilde g\) 截断到 \([-G, G]\)),把"不满足 L3 第一条件" 改造成"满足"。后者是 deep RL 标配技术。

下一节预告

L2 给了 RM 算法的形态 与动力学直觉,但没证 它真的能收敛 —— 直观上"负反馈 + 步长衰减" 听起来对,但严格证明要回答:哪些 \(g\) 可以用 RM?哪些 \(\{a_k\}\) 是合法步长?哪种噪声 \(\eta_k\) 能扛?L3(教材 §6.2.2 + §6.3) 给出标准答案 —— Robbins-Monro 收敛定理(Theorem 6.1)的三条件:(a) \(0 < c_1 \le \nabla g \le c_2\)(单调有界);(b) \(\sum a_k = \infty\) 且 \(\sum a_k^2 < \infty\)(步长两条件);(c) \(\mathbb E[\eta_k|\mathcal H_k] = 0,\ \mathbb E[\eta_k^2|\mathcal H_k] < \infty\)(噪声零均值方差有界)。证明依靠Dvoretzky 收敛定理(Theorem 6.2),它进一步给出"任意 \(\Delta_{k+1} = (1-\alpha_k)\Delta_k + \beta_k \eta_k\) 形式的随机过程" 的收敛性。L3 把"步长两条件" 的几何意义讲透 —— \(\sum \alpha = \infty\) 防止"太早停步",\(\sum \alpha^2 < \infty\) 防止"噪声累积"。这是 Ch.7 TD 收敛证明的核心工具,也是 Ch.8 DQN deadly triad 分析的起点。
