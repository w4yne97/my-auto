L4: SGD Algorithm — Introduction (随机梯度下降:RM 在期望优化上的化身)

§1 Why this lesson

L1-L3 把 RM 算法 \(w_{k+1} = w_k - a_k \tilde g(w_k, \eta_k)\) 的形态(L2)+ 收敛性(L3)讲透。本节(教材 §6.4,pp. 114-116)把视角切到机器学习与深度学习每天都在用的Stochastic Gradient Descent (SGD),并给出 Ch.6 整章的核心命题之一:SGD 是 RM 的特例 —— 不是"类似",而是"把 RM 中的 \(g\) 替换成 \(\nabla J\)、把 \(\tilde g\) 替换成带噪梯度" 的字面套用。

关键洞察(教材 §6.4 + Box 6.1):(a) SGD 解的是期望优化:\(\min_w J(w) = \mathbb E_X[f(w, X)]\)(教材 (6.10))。(b) 普通梯度下降 \(w_{k+1} = w_k - \alpha_k \mathbb E[\nabla_w f(w_k, X)]\)(教材 (6.11))在分布未知 时不可用 —— 求不出期望。(c) SGD 用单样本梯度 替代:\(w_{k+1} = w_k - \alpha_k \nabla_w f(w_k, x_k)\)(教材 (6.13))。(d) 与 RM 同构(教材 Box 6.1):令 \(g(w) \doteq \nabla_w J(w) = \mathbb E[\nabla_w f(w, X)]\),则 SGD 求的是 \(g(w) = 0\) 的根;\(\tilde g(w_k, x_k) = \nabla_w f(w_k, x_k) = g(w_k) + \eta_k\),其中 \(\eta_k \doteq \nabla_w f(w_k, x_k) - \mathbb E[\nabla_w f(w_k, X)]\)。\(\eta_k\) 由 \(x_k\) i.i.d. 抽样保证条件零均值(教材 (6.13) 后段)。(e) SGD 收敛证明(Theorem 6.4)直接套 RM 三条件:\(c_1 \le \nabla^2 f \le c_2\)(强凸 + 光滑)、\(\sum\alpha=\infty,\sum\alpha^2<\infty\)、i.i.d. 样本。(f) "L1 均值估计" 是 SGD 在 \(f(w, X) = \|w - X\|^2/2\) 上的特例(L5 详谈) —— 整个 Ch.6 因此首尾闭环:从均值估计出发,经过 RM 抽象,回到均值估计作 SGD 第一例。

§2 核心定义(直觉先行)

  • 期望优化问题(教材 (6.10)):\(\min_w J(w) \doteq \mathbb E_X[f(w, X)]\)。\(w\) 是参数(标量或向量),\(X\) 是随机变量。典型场景:监督学习里 \(X = (\text{input}, \text{label})\),\(f\) 是单样本损失;RL 里 \(X\) 是 trajectory 样本,\(f\) 是某种 surrogate loss。
  • 普通梯度下降 (Gradient Descent,GD,教材 (6.11)):\(w_{k+1} = w_k - \alpha_k \nabla_w J(w_k) = w_k - \alpha_k \mathbb E_X[\nabla_w f(w_k, X)]\)。问题:期望要么无法解析求(分布未知),要么太贵(数据集巨大,每步算 \(\nabla J\) 要扫一遍全数据)。
  • 批量梯度下降 (Batch GD,BGD,教材 (6.12)):用 \(n\) 个 i.i.d. 样本 \(\{x_i\}_{i=1}^n\) 估期望:\(w_{k+1} = w_k - \frac{\alpha_k}{n}\sum_{i=1}^n \nabla_w f(w_k, x_i)\)。当 \(n\) 大时 接近真梯度,但每步成本 \(O(n)\),数据巨量时仍贵。
  • 随机梯度下降 (SGD,教材 (6.13)):\(w_{k+1} = w_k - \alpha_k \nabla_w f(w_k, x_k)\)。每步只用一个样本 \(x_k\),成本 \(O(1)\)。"stochastic" 名字来源:\(x_k\) 是从 \(X\) 分布随机抽样 一次。
  • SGD = RM 在 \(g = \nabla J\) 上跑(教材 Box 6.1):把 SGD 的 \(\nabla_w f(w_k, x_k)\) 拆成 \(\mathbb E[\nabla_w f(w, X)] + (\nabla_w f(w_k, x_k) - \mathbb E[\nabla_w f(w_k, X)]) = g(w_k) + \eta_k\)。代入 SGD 得 \(w_{k+1} = w_k - \alpha_k[g(w_k) + \eta_k]\),正是 RM (6.5)。所以SGD ⊂ RM。
  • \(\eta_k\) 在 SGD 中的具体形式:\(\eta_k = \nabla_w f(w_k, x_k) - \mathbb E_X[\nabla_w f(w_k, X)]\)。条件零均值:\(\mathbb E[\eta_k|\mathcal H_k] = \mathbb E_{x_k}[\nabla_w f(w_k, x_k) | w_k] - \mathbb E_X[\nabla_w f(w_k, X)] = 0\)(因为 \(x_k \sim X\) i.i.d. 与 \(\mathcal H_k\) 独立)。方差有界:若 \(|\nabla_w f(w, x)| < \infty\) 对所有 \(w, x\),则 \(\mathbb E[\eta^2|\mathcal H] < \infty\)。RM 第三条件自动满足。
  • SGD 收敛定理(Theorem 6.4):在 (a) \(0 < c_1 \le \nabla_w^2 f(w, X) \le c_2\)(\(f\) 关于 \(w\) 强凸 + 光滑)、(b) \(\sum\alpha_k = \infty,\sum\alpha_k^2 < \infty\)、(c) \(\{x_k\}\) i.i.d. 下,\(w_k \xrightarrow{a.s.} w^* \doteq \arg\min J\)。证明:Box 6.1 把 SGD 写成 RM 形式,直接套 Theorem 6.1。

Step 1. 从 GD 到 SGD 的"实用主义" 推导(教材 §6.4 中段)。GD (6.11) 要 \(\mathbb E[\nabla_w f(w_k, X)]\),分布未知时无解。方案 A:用 \(n\) 样本估期望 (6.12) BGD,代价 \(O(n)\) 每步。方案 B:每步只用一个 样本估期望 (6.13) SGD,代价 \(O(1)\) 每步。SGD 的每步估计有偏 / 高方差,但多步累积 趋向真梯度(类似 L1 (6.2) 用 (1/k) 累积)。这就是"用大量便宜的不准估计 替代少量昂贵的准确估计" 的设计哲学,大数据时代 SGD 主导 ML 的根本原因。
Step 2. SGD 收敛的三步证明(教材 Box 6.1)。(i) 建立 \(g(w) \doteq \nabla J(w),\ \tilde g(w_k, x_k) \doteq \nabla_w f(w_k, x_k) = g(w_k) + \eta_k\)。(ii) 验 RM 三条件:(a) \(\nabla g = \nabla^2 J\) 在 \([c_1, c_2]\) ⇐ Theorem 6.4 (a);(b) \(\sum \alpha = \infty, \sum \alpha^2 < \infty\) ⇐ Theorem 6.4 (b);(c) \(\mathbb E[\eta|\mathcal H] = 0\)(i.i.d. 抽样)、\(\mathbb E[\eta^2|\mathcal H] < \infty\)(\(\nabla f\) 有界)⇐ Theorem 6.4 (c)。(iii) 套 Theorem 6.1:\(w_k \to w^*\) a.s.。整个证明 < 1 页 —— 这是 RM 框架的力量:抽象代价付一次,所有特例自动收益。
Step 3. "单样本梯度" 不等于 "差梯度"(教材 §6.4 末段直觉,L6 详谈)。SGD 看似"每步只用一个样本" 估计很差,但当 \(w_k\) 远离 \(w^*\) 时,真梯度 \(\nabla J(w_k)\) 本身就大,单样本梯度的相对噪声小 ⇒ SGD 此时几乎与 GD 等效快速下降。当 \(w_k\) 接近 \(w^*\) 时 真梯度小,单样本梯度的相对噪声大 ⇒ SGD 在 \(w^*\) 附近抖动 不收敛到精确点。这就是 L6 要详谈的 SGD"远处快、近处慢" pattern,与"常步长 SGD 不严格收敛" 是同一根。

§5 常见误解 / 易混概念

很多人以为:"SGD 只是近似 梯度下降,损失精度。"
实际上:SGD 在无穷数据极限 与 GD 收敛到同一 \(w^*\)(Theorem 6.4)。短期看 SGD 单步嘈杂,长期看样本均值无偏。具体:\(\mathbb E[\nabla_w f(w_k, x_k) | w_k] = \nabla J(w_k)\),所以 SGD 的更新方向期望 就是 GD 的更新方向 —— 单步有偏差,但整体路径无偏。这是 Robbins-Monro 1951 年的核心洞察:用大量便宜带噪估计累积,等效于少量昂贵精确估计。
因为:这是大数定律 在迭代算法上的具体化。\(\sum_k \alpha_k \nabla f(w_k, x_k) \approx \sum_k \alpha_k \mathbb E[\nabla f(w_k, X)]\)(Cesàro 类平均),只要 \(\alpha_k\) 满足 RM 第二条件,SGD 累积梯度就等于 GD 累积梯度。在数据规模上,SGD 比 GD 节省 \(O(n)\) 倍计算 ⇒ 大数据时代主导地位。
很多人以为:"SGD 是新算法,与 RM 是独立 的两个东西。"
实际上:SGD ⊂ RM,两者是包含关系(Box 6.1 严证)。SGD 是 RM 在"\(g = \nabla J\) 且 \(J = \mathbb E[f]\)" 这个特例上的具体形态。RM 比 SGD 更广 —— 可以解非梯度 根问题(\(g(w) = w^3 - 5\) 没"对应优化目标")、可以处理不可微 的 \(g\)(只要单调有界)、可以处理固定点方程(\(g(w) = w - T(w)\) 不一定是某个 \(J\) 的梯度,如 Ch.7 TD 的 Bellman 算子)。
因为:RM 是 1951 年提出的统计 root-finding 框架,SGD 是 ML 后续在 RM 上的命名 + 应用。两者历史顺序也反向(RM 早 SGD 半世纪)。教学顺序上先讲 RM 再讲 SGD,是因为从抽象到具体 比从具体到抽象 教学效率高 —— 你看一次 RM 的收敛证明,就能免费 推出 SGD、TD、Q-learning、policy gradient 等十几个算法的收敛性。
很多人以为:"SGD 配 \(\alpha_k = 1/k\) 才严格收敛,深度学习用常数 lr 是不严谨。"
实际上:常数 lr SGD 不严格收敛到 \(w^*\),但稳定在 \(w^*\) 附近邻域,这是设计选择(L1/L3 已多次提)。深度学习目标不是"严格找到 \(w^*\)",而是"找到泛化好的 \(w\)"。常数 lr 下 \(w_k\) 在邻域内"探索",有助于跳出尖锐局部最优(generalization vs convergence trade-off)。Adam、RMSProp 等都属于"常 lr + 自适应" 流派,本质仍是 RM 框架内,但优化的是"实操速度" 而非"渐近收敛"。
因为:"渐近收敛"(\(k \to \infty\))是数学家关心的;"有限步效率" 是工程师关心的。RM 给前者保证,常步长是为后者妥协。两者不矛盾 —— 都在 RM 大伞下。Ch.8 deep RL 默认 lr=3e-4 等常数是工程实操,不影响收敛理论的正确性。

§6 自检 + 视频 + 下一节

Q1. 写出 \(f(w, X) = (w - X)^2/2\) 的 \(\nabla_w f\),代入 SGD (6.13),验证它就是 L1 (6.4)。

A. \(\nabla_w f(w, X) = w - X\)(对 \(w\) 求导,\(X\) 当常数)。代入 SGD:\(w_{k+1} = w_k - \alpha_k(w_k - x_k)\),与 L1 (6.4) 完全一致。再看 \(g(w) = \mathbb E[\nabla_w f(w, X)] = \mathbb E[w - X] = w - \mathbb E[X]\),\(g\) 的根 \(w^* = \mathbb E[X]\) 即真均值。所以"L1 均值估计" 是 SGD 在 \(f = (w-X)^2/2\) 上的具体化,Ch.6 故事首尾闭环。

Q2. SGD (6.13) 与 RM (6.5) 的条件 对应:RM 第一条件 \(c_1 \le \nabla g \le c_2\) 对应 SGD 的什么?

A. SGD 中 \(g(w) = \nabla J(w)\),所以 \(\nabla g(w) = \nabla^2 J(w)\) 是Hessian(\(J\) 二阶导)。RM 第一条件 \(c_1 \le \nabla g \le c_2\) 对应 \(c_1 I \le \nabla^2 J \le c_2 I\)(矩阵不等式),即 \(J\) 强凸(\(c_1 > 0\))+ 光滑(\(c_2 < \infty\))。这是凸优化的标配假设。Theorem 6.4 (a) 直接写 \(c_1 \le \nabla_w^2 f \le c_2\)(对 \(f\) 而非 \(J\)),由 \(J = \mathbb E[f]\) 与求导线性 \(\nabla^2 J = \mathbb E[\nabla^2 f]\),\(f\) 强凸光滑 ⇒ \(J\) 强凸光滑。实操:深度学习里 \(f\) 通常非凸,Theorem 6.4 严格不适用;但局部 仍可应用("局部凸 + warm start"),收敛性需要更精细分析(Bottou-Curtis-Nocedal 等)。

Q3. "SGD ⊂ RM" 的反向是否成立 —— 任意 RM 都能找到对应 \(J\) 把它写成 SGD?

A. 不一定。要把 RM 写成 SGD,需要找 \(J\) 使 \(g(w) = \nabla_w J(w)\) ⇔ \(g\) 是梯度场。在 \(\mathbb R^1\) 总能找(\(J(w) = \int_0^w g(u) du\)),但是否有意义 依赖 \(g\)。例:\(g(w) = w^3 - 5\) ⇒ \(J(w) = w^4/4 - 5w\),无下界,不是合法的"极小化目标" —— 这个"SGD" 不会停。在 \(\mathbb R^d\)(向量),\(g\) 是梯度场要求Jacobian 对称(\(\partial g_i / \partial w_j = \partial g_j / \partial w_i\)),一般 \(g\) 不满足。典型反例:Ch.7 TD 算法 \(g(V) = V - T_\pi V\) 在线性 FA 下不是 任何 \(J\) 的梯度(deadly triad 的根)—— 所以 TD 是 RM 但不是 SGD,Ch.8 详谈。结论:RM 严格更广,SGD 是它的"梯度场特例"。
