L7: SGD Convergence Theorem & Chapter Summary (SGD 收敛性证明与全章总结)

§1 Why this lesson

L1-L6 把 RM 与 SGD 的形态、收敛证明、动力学、变种 全讲完。本节(教材 §6.4.5-6.6,pp. 121-124)做三件事:(1) Theorem 6.4 SGD 收敛性的严证(教材 Box 6.1)—— 用 Box 6.1 把 SGD 归约到 RM,直接套 Theorem 6.1,不到一页 完成;(2) Ch.6 全章梳理 —— 从 (1/n) 均值递推到 RM 抽象到 SGD 特例,四个最重要的取舍;(3) Ch.7 TD 学习预告 —— RM 框架将直接套用在 Bellman 算子的固定点上,生出第一族真正能在线运行的 RL 算法。

核心 takeaway(不是 复述全章六节,而是抓4 个最关键的):[T1] RM 是"用样本估期望" 这件事的统一骨架:\(w_{k+1} = w_k - \alpha_k \tilde g_k\) 一行公式覆盖均值估计、SGD、TD、Q-learning、policy gradient ——差别只在 \(\tilde g\) 怎么从数据采。[T2] 步长两条件 \(\sum\alpha=\infty\) 与 \(\sum\alpha^2<\infty\) 是张力:\(\sum\alpha=\infty\) 防"太早停",\(\sum\alpha^2<\infty\) 防"停不下来"。两者必须同时。\(\alpha_k = c/k\) 是经典默认,常 \(\alpha\) 不严格收敛但能跟踪非平稳 target(deep RL 默认)。[T3] SGD ⊂ RM,Theorem 6.4 几乎免费:把 SGD 写成 RM 形态(Box 6.1)即得收敛。这套"归约 + 套定理" 思路 在 Ch.7-Ch.10 反复出现,Q-learning 收敛证明就是把它套到 Bellman 算子上。[T4] BGD/SGD/MBGD 是同一族:m=1 SGD,m=n BGD,中间 MBGD。"样本预算 vs 更新频率 vs 方差 vs 泛化" 四维 trade-off 由 m 调控,深度 RL 黄金区间 \(m \in [32, 256]\)。[Ch.7 桥接] RM 给"迭代 + 采样估期望" 的工具;Ch.7 TD 直接把这套用到 Bellman 算子的固定点 \(v_\pi\):\(g(v) = v - T_\pi v\),\(\tilde g(v_t, s_t, s_{t+1}, r_{t+1}) = V(s_t) - [r_{t+1} + \gamma V(s_{t+1})]\),套 RM 即得 TD(0)。SARSA、Q-learning 是 TD(0) 在 \(q\)-value 上的同构变种。这是 RL 算法演化"从 model-based 到 model-free + incremental" 的第一颗果实。

§2 核心定义(直觉先行)

  • Theorem 6.4 (SGD 收敛性,教材 §6.4.5,p. 121):在 (a) \(0 < c_1 \le \nabla_w^2 f(w, X) \le c_2\)(\(f\) 强凸 + 光滑)、(b) \(\sum \alpha_k = \infty,\sum \alpha_k^2 < \infty\)、(c) \(\{x_k\}\) i.i.d. 下,SGD \(w_{k+1} = w_k - \alpha_k \nabla_w f(w_k, x_k) \xrightarrow{a.s.} w^* = \arg\min \mathbb E[f(w, X)]\)。
  • 证明大纲(教材 Box 6.1,p. 122):(i) 写 \(g(w) \doteq \nabla_w J(w) = \mathbb E[\nabla_w f(w, X)]\),\(\tilde g_k = \nabla_w f(w_k, x_k) = g(w_k) + \eta_k\)。(ii) 验 RM 三条件:\(\nabla g = \nabla^2 J = \mathbb E[\nabla^2 f] \in [c_1, c_2]\) ✓;\(\sum\alpha = \infty,\sum\alpha^2 < \infty\) ✓;\(\mathbb E[\eta_k|\mathcal H_k] = 0\)(i.i.d.)、\(\mathbb E[\eta^2|\mathcal H] < \infty\)(\(\nabla f\) 有界)✓。(iii) 套 Theorem 6.1。
  • Ch.6 takeaway [T1] —— RM 是统一骨架:所有"用样本估期望" 的迭代算法都是 \(w_{k+1} = w_k - \alpha_k \tilde g_k\) 一行公式的特例。差别在 \(\tilde g\) 的采样方式:均值估计 \(\tilde g = w - x\);SGD \(\tilde g = \nabla_w f(w, x)\);TD \(\tilde g = V(s) - [r + \gamma V(s')]\);Q-learning \(\tilde g = Q(s,a) - [r + \gamma \max_{a'} Q(s', a')]\);policy gradient \(\tilde g = -\nabla_\theta \log \pi_\theta(a|s) \cdot \tilde q(s, a)\)。
  • Ch.6 takeaway [T2] —— 步长两条件的张力:\(\sum\alpha = \infty\) 与 \(\sum\alpha^2 < \infty\) 必须同时成立。\(\alpha_k = c/k^\beta\) 满足两条件 ⇔ \(\beta \in (0.5, 1]\)。常 \(\alpha\) 违反第二条件,严格不收敛,但能跟踪非平稳 target(RL 必备)—— 这是 deep RL 用 Adam lr=3e-4 而非衰减步长的根。
  • Ch.6 takeaway [T3] —— "归约 + 套定理" 思路:把新算法 写成 RM 形态(找到 \(g\)、\(\tilde g\)、验三条件),即得收敛证明。Theorem 6.4 是模板,Ch.7 Q-learning 收敛、Ch.8 TD with FA 收敛(线性 case)、Ch.10 actor-critic 收敛都用这一思路。
  • Ch.6 takeaway [T4] —— m 调控的多维权衡:m=1 SGD 单步便宜噪声大;m=n BGD 单步贵噪声 0;中间 MBGD 平衡。但"泛化" 这第四维让小 batch SGD 在深度学习中胜过 BGD(逃尖谷)。Deep RL 实操 m 多在 32-256。
  • Ch.7 链条(教材 §6.6 第二条 Q&A):RM 给"迭代 + 采样估期望",Ch.7 TD 直接套到 Bellman 固定点 \(v_\pi\)。"RL 算法都是 RM 在不同 \(\tilde g\) 上的具体化" —— 这是 Ch.6 在全书中作为数学中心 的真正含义。

公式 2:RM 骨架在不同算法的具体化(全书统一视图)

Step 1. "把新算法 写成 RM 形态" 三步。(i) 找方程:你的算法在解什么 \(g(w) = 0\)?(均值估计:\(g = w - \mathbb E[X]\);SGD:\(g = \nabla J\);TD:\(g = V - T_\pi V\)。)(ii) 找带噪估计:你能从数据里采到 \(\tilde g(w, \xi) = g(w) + \eta\)?(均值估计:\(\tilde g = w - x\);TD:\(\tilde g = V(s) - [r + \gamma V(s')]\)。)(iii) 验三条件:\(\nabla g\) 有界?\(\alpha_k\) 满足两条件?\(\eta\) 条件零均值方差有界?三步通过 ⇒ 收敛。这就是 Ch.7-Ch.10 算法收敛证明的母骨架。
Step 2. "为什么 RM 第三条件是条件期望"(Ch.6 收尾再强调)。\(\mathbb E[\eta_k|\mathcal H_k] = 0\) 而非 \(\mathbb E[\eta_k] = 0\)。差别:前者允许 \(\eta_k\) 依赖当前 \(w_k\),只要在给定 \(w_k\) 下零均值;后者要求无条件 零均值,过强。RL 中 \((s_t, a_t, r_{t+1}, s_{t+1})\) 由当前策略生成 ⇒ 数据分布依赖 当前 \(w\),无条件期望不为零;但给定 \(w\),环境分布 \(p(s',r|s,a)\) 由 MDP 固定 ⇒ 条件期望为零成立。这是 RM 框架适配 RL 的关键技术细节,Theorem 6.3 的多变量推广进一步给 Q-learning 收敛。
Step 3. Ch.6 的第一性原理 总结。"用样本估期望" 这件事的极简实现 是:\(w_{k+1} = w_k - \alpha_k \tilde g_k\)。能收敛 的条件是:(a) 函数结构(单调 + Lipschitz);(b) 步长调度(两条件);(c) 噪声性质(条件零均值方差有界)。三者缺一不可。这套理论让 1951 年的 RM 算法在 70 多年后的 deep RL 时代仍然是基础 —— 当前 deep RL 算法的所有"新" 部分都可以拆成"RM 框架 + 工程优化",未变 的是 RM 的数学骨架。

§5 常见误解 / 易混概念

很多人以为:"Ch.6 是数学预备章,可以跳过直接学 Ch.7 TD。"
实际上:跳过 Ch.6 会导致 Ch.7-Ch.10 收敛证明都看不懂。Ch.7 Q-learning 收敛证明直接引用 Theorem 6.3(多变量 Dvoretzky)。Ch.8 linear FA 收敛(Tsitsiklis-Van Roy)是 Ch.6 RM 在向量空间 + 加权 norm 上的扩展。Ch.10 actor-critic 双时间尺度收敛(Borkar)是 Ch.6 RM 的两层并行版本。Ch.6 不是预备,是核心。
因为:RL 算法多样性不在 "不同的更新公式",在 "不同的 \(\tilde g\) 来源"。但所有"更新公式" 都共享 RM 框架,所有收敛证明 都共享 Theorem 6.1/6.3 模板。学完 Ch.6,后续每章只需问"g 是什么?\(\tilde g\) 怎么采?三条件验证?" 三步就能给出收敛分析。这是教材作者把 Ch.6 放在全书 chapter map 中央(Figure 6.1)的结构原因。
很多人以为:"deep RL 用 Adam / 常 lr,所以 Ch.6 RM 框架不适用。"
实际上:Adam 仍是 RM,只是 \(\alpha_k\) 改成逐参数自适应。Adam 的更新是 \(w_{k+1} = w_k - \alpha_k m_k / \sqrt{v_k + \epsilon}\),其中 \(m_k\) 是梯度一阶矩(EMA),\(v_k\) 是二阶矩。整体仍是 \(w_{k+1} = w_k - \alpha'_k \tilde g'_k\) 形态,\(\alpha'_k = \alpha_k / \sqrt{v_k + \epsilon}\) 是有效 步长,\(\tilde g'_k = m_k\) 是有效 梯度估计。Adam 收敛证明(Reddi et al. 2018)走 RM 路线,只是条件更复杂(因 \(v_k\) 也是 stochastic)。RM 是母,Adam 是子。
因为:"常 lr" 在 deep RL 的真正含义是"不衰减但有自适应"。Adam 通过 \(v_k\) 给"每个参数自动调步长",在不同坐标上行为不同。这不违反 RM,只是把"标量 \(\alpha\)" 推广到"对角矩阵 \(\text{diag}(\alpha')\)",收敛分析需要更精细但仍在 RM 框架内。Ch.8 deep RL 用 Adam 的工程实操,完全建立在 Ch.6 数学基础上。
很多人以为:"Ch.6 是静态 数学,与 RL 的动态 在线学习有距离。"
实际上:RM 从 1951 年起就是为在线学习 设计的。Robbins-Monro 原 paper 题目是"A stochastic approximation method",场景是"每来一个样本调整一次估计",正是 RL 的核心需求。第三条件的条件期望 \(\mathbb E[\eta|\mathcal H]=0\) 允许"样本依赖历史",直接对应 RL 数据流(策略改变 ⇒ 数据分布改变)。Theorem 6.3 多变量版进一步处理"多个状态各自 incremental",对应 Q-learning。RM 框架本质上是为在线 / 序贯 / 非平稳 学习设计,与 RL 是天作之合,不是事后凑数。
因为:"统计学习" 与"RL" 的边界很模糊 —— RL 是带反馈循环的统计学习。Robbins-Monro 1951 不知道有"RL" 一词,但他设计的框架直接覆盖了 RL 50 年后的需求,这是数学的普适性 在闪光。Ch.6 让我们看清:RL 不是新 数学,是RM 在 MDP 结构上的具体化。

§6 自检 + 视频 + 下一章

Q1. 复述 Box 6.1 SGD 收敛证明的 3 步,只用一两句话。

A. (i) 把 SGD 改写成 RM:令 \(g(w) = \nabla J(w) = \mathbb E[\nabla_w f(w, X)]\),\(\tilde g_k = \nabla_w f(w_k, x_k) = g(w_k) + \eta_k\)(单点梯度 = 真梯度 + 零均值噪声)。(ii) 验 RM 三条件:\(\nabla g = \mathbb E[\nabla^2 f] \in [c_1, c_2]\)(由 Theorem 6.4 (a));\(\sum\alpha=\infty,\sum\alpha^2<\infty\)(由 (b));\(\mathbb E[\eta|\mathcal H]=0\)(由 i.i.d.)。(iii) 套 Theorem 6.1:\(w_k \to w^*\) a.s.。核心:把"新" 算法归约到"已知收敛的母算法",这是数学证明的分而治之 套路,后续 RL 算法收敛全靠这招。

Q2. Ch.6 4 个 takeaway 中,对 RL 实操影响最大 的是哪个?为什么?

A. 见仁见智,我选[T2] 步长两条件的张力。理由:每个 deep RL 算法都需选 lr,而 [T2] 直接告诉我们"衰减 lr → 严格收敛" vs "常 lr → 跟踪非平稳" 的结构性 选择。Q-learning、DQN、PPO、SAC 都默认常 lr + 自适应优化(Adam),这是 [T2] 在非平稳 RL 数据流 下的实操选择。理解 [T2] 让你明白 lr scheduler 何时需要(收敛任务)、何时不需要(在线任务)。次选 [T3]"归约 + 套定理" —— 它是读论文 时最有价值的工具(看新算法的收敛证明,套这个模板就能找出"新点" 在哪)。

Q3. Ch.6 的 \(g\) 与 Ch.7 TD 的"Bellman 残差" 怎么对应?

A. Ch.7 TD 解的是 Bellman 方程 \(v_\pi = r_\pi + \gamma P_\pi v_\pi\),即固定点方程 \(v = T_\pi v\) 的根。改写成根 finding:\(g(v) \doteq v - T_\pi v = 0\)。带噪估计:在 \((s_t, r_{t+1}, s_{t+1})\) 这条单步样本 上,\(\tilde g(v, s_t, r_{t+1}, s_{t+1}) \doteq V(s_t) - [r_{t+1} + \gamma V(s_{t+1})] = g(V)|_{s_t} + \eta_t\)(因 \([r + \gamma V(s')]\) 是 \(T_\pi V\) 在 \(s\) 处的带噪估计)。代入 RM:\(V(s_t) \leftarrow V(s_t) - \alpha_t \tilde g_t = V(s_t) - \alpha_t[V(s_t) - r_{t+1} - \gamma V(s_{t+1})] = V(s_t) + \alpha_t \delta_t\)(\(\delta_t\) 是 TD error)—— 正是 TD(0) 更新公式。所以 TD(0) ≡ RM 在 Bellman 算子的固定点上,Ch.7 的故事就是把这个观察展开。

下一章预告

到这一节为止,Ch.6 完成:RM 给了我们"迭代 + 采样估期望" 的统一工具,SGD 是它的明星特例,BGD/MBGD/SGD 三联是 m 调控的同族。但 Ch.6 没有给任何具体的 RL 算法 —— 它是数学工具章。Ch.7 Temporal-Difference Methods 把这套工具直接套用 在 RL 的核心结构 —— Bellman 方程的固定点 —— 上,生出第一族真正能在线运行 的 RL 算法:TD(0)(估 \(v_\pi\))、SARSA(估 \(q_\pi\) on-policy)、Q-learning(估 \(q^*\) off-policy)。关键洞察:Ch.5 MC 要"等 episode 收完算 \(\hat G\)",方差大;Ch.7 TD 用"\(r + \gamma V(s')\) 当一步 bootstrap target",每步就能更新,方差降但引入 bias。"方差 vs bias" 这个 trade-off 是 Ch.7 全章主线。Ch.7 还讲 \(n\)-step TD(MC 与 TD(0) 之间的连续光谱)与 TD(λ)(经典)。Q-learning 收敛证明直接套 Ch.6 Theorem 6.3 多变量 Dvoretzky 定理,Ch.6 投资在 Ch.7 立刻收回。
