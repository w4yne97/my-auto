L3: RM Convergence — Robbins-Monro Theorem & Step-Size Conditions (RM 收敛定理与步长两条件)

§1 Why this lesson

L2 立了 RM 算法 \(w_{k+1} = w_k - a_k \tilde g(w_k, \eta_k)\) 的形态,演示了"负反馈 + 步长衰减" 的直觉,但没证 它真的会收敛。本节(教材 §6.2.2-6.3,pp. 105-114)给出 Ch.6 的数学中心:Robbins-Monro 定理(Theorem 6.1) 的三条件 + Dvoretzky 定理(Theorem 6.2) 作证明工具。这把"带噪迭代解方程" 这件事的充分条件 锁死,Ch.7 TD 学习收敛证明几乎"免费拿来用"。

关键洞察(教材 §6.2.2 + §6.3):(a) RM 定理三条件(Theorem 6.1,p. 106):(i) \(0 < c_1 \le \nabla g(w) \le c_2\)(\(g\) 全局单增 + Lipschitz);(ii) \(\sum_{k=1}^\infty a_k = \infty\) 且 \(\sum_{k=1}^\infty a_k^2 < \infty\)(步长两条件);(iii) \(\mathbb E[\eta_k|\mathcal H_k]=0\) 且 \(\mathbb E[\eta_k^2|\mathcal H_k]<\infty\)(噪声零均值方差有界,不需 Gaussian)。则 \(w_k \xrightarrow{a.s.} w^*\)。(b) 步长两条件的几何意义(教材 §6.2.2 中段 + Eq 6.6):\(\sum a^2<\infty\) ⇒ \(a_k \to 0\) ⇒ "新加入的噪声 \(a_k \eta_k\) 越来越小",防止振荡;\(\sum a=\infty\) ⇒ 累积步长无界 ⇒ 算法有能力 从任意初始点走到 \(w^*\)。(c) 经典 \(a_k = 1/k\) 满足:\(\sum 1/k = \infty\)(harmonic),\(\sum 1/k^2 = \pi^2/6 < \infty\)(Basel)。(d) Dvoretzky 定理(Theorem 6.2) 是工具:任意"\(\Delta_{k+1} = (1-\alpha_k)\Delta_k + \beta_k \eta_k\)" 形式的过程,在 \(\sum\alpha=\infty,\sum\alpha^2<\infty,\sum\beta^2<\infty,\mathbb E[\eta|\mathcal H]=0,\mathbb E[\eta^2|\mathcal H]\le C\) 下,\(\Delta_k \xrightarrow{a.s.} 0\)。(e) 用 Dvoretzky 证 RM(教材 §6.3.3):\(w_{k+1} - w^* = w_k - w^* - a_k[g(w_k) - g(w^*) + \eta_k] = (1 - a_k\nabla g(w'_k))(w_k - w^*) - a_k \eta_k\),正是 Dvoretzky 形态。(f) 反过来当条件不全满足时:常步长 \(a \equiv 0.1\) 违反 \(\sum a^2 < \infty\),\(w_k\) 在 \(w^*\) 附近稳态振荡 不收敛;\(a_k = 1/k^2\) 违反 \(\sum a = \infty\),累积步长有限 ⇒ 若 \(|w_1 - w^*| > b\)(教材 (6.6))算法陷入 远处不可及。(g) Theorem 6.3(教材 §6.3.4) 把 Dvoretzky 推广到多变量、最大范数情形,Q-learning 收敛证明直接用它。

§2 核心定义(直觉先行)

  • Robbins-Monro 定理(Theorem 6.1,p. 106):RM 算法 (6.5) 在三条件下 \(w_k \xrightarrow{a.s.} w^*\)。"a.s." (almost surely):概率 1 收敛 —— 比"依概率收敛" 强,接近"每次跑都收敛" 的实操语义。
  • 第一条件(\(g\) 单增 + Lipschitz):\(0 < c_1 \le \nabla g(w) \le c_2\) 对所有 \(w\)。解释:(i) \(c_1 > 0\) 保证 \(g\) 严格单增 ⇒ \(g(w)=0\) 根唯一(否则RM 不知该往哪个根走);(ii) \(c_2 < \infty\) 保证不爆炸(否则 \(w\) 远处时 \(g\) 太大,一步弹飞)。对应优化场景:\(g = \nabla J\) ⇒ \(c_1 \le \nabla^2 J \le c_2\) ⇒ \(J\) 强凸且光滑(经典凸优化设定)。
  • 第二条件(步长两条件):\(\sum a_k = \infty\) 且 \(\sum a_k^2 < \infty\)。第一:\(\sum a^2<\infty\) ⇒ \(a_k \to 0\) ⇒ 后期"新加入扰动 \(a_k \eta_k\) 衰减",算法稳得下来;第二:\(\sum a=\infty\) ⇒ 累积步长无界,算法"有能力" 走任意远(教材 (6.6) 给反证 —— 若 \(\sum a<\infty\) 则 \(|w_1 - w_\infty| \le \sum a_k|\tilde g| \le b\) 有限,初始 \(w_1\) 离 \(w^*\) 大于 \(b\) 时不可达)。
  • 第三条件(噪声零均值方差有界):\(\mathbb E[\eta_k|\mathcal H_k]=0\),\(\mathbb E[\eta_k^2|\mathcal H_k]<\infty\)。\(\mathcal H_k = \{w_k, w_{k-1}, \dots\}\) 是历史。关键:不要求 \(\eta_k\) 是 i.i.d. 或 Gaussian —— 允许 \(\eta_k\) 依赖 历史 \(w_{条件零均值 即可。这让 RM 适用于 RL —— 在 RL 中,样本 \(s_{t+1}\) 依赖 \(w_t\)(策略改变 → 轨迹改变),不是 i.i.d.,但条件零均值常成立。
  • Dvoretzky 定理(Theorem 6.2,p. 109):对随机过程 \(\Delta_{k+1} = (1-\alpha_k)\Delta_k + \beta_k \eta_k\),若 \(\sum \alpha=\infty, \sum \alpha^2<\infty, \sum \beta^2<\infty\)(uniformly a.s.),\(\mathbb E[\eta_k|\mathcal H_k]=0, \mathbb E[\eta_k^2|\mathcal H_k]\le C\),则 \(\Delta_k \xrightarrow{a.s.} 0\)。这是 RM 收敛的"母定理",证明用 quasimartingale convergence(教材 Appendix C)。
  • "\(\sum 1/k=\infty\) 且 \(\sum 1/k^2<\infty\)"(教材 §6.2.2 末,p. 108):harmonic series 与 Basel problem 的共生 让 \(a_k = 1/k\) 恰好满足 RM 第二条件。更一般:\(a_k = c/k^\beta\) 满足两条件当且仅当 \(\beta \in (0.5, 1]\):\(\beta \le 0.5\) 时 \(\sum a^2 = \infty\)(噪声不衰);\(\beta > 1\) 时 \(\sum a < \infty\)(走不远)。\(\beta=1/2\) 是临界;\(\beta \in (0.5, 1]\) 是合法窗口。
  • Theorem 6.3 多变量推广(教材 §6.3.4):把"\(\Delta_k\) 是标量" 推广到"\(\Delta_k(s)\) 在有限集 \(\mathcal S\) 上,用 \(\|\cdot\|_\infty\) 度量"。Q-learning 的 \(Q(s,a)\) 收敛证明就用这个 —— 把每个 \((s,a)\) 看一个 \(\Delta(s,a)\),应用 Theorem 6.3 给出 \(\|Q_k - Q^*\|_\infty \to 0\)。这是 Ch.7 Q-learning 收敛证明的钥匙。

Step 1. "\(\sum a^2 < \infty\) 必要" 的反证(教材 §6.2.2 中段,反例 \(a_k \equiv c\))。设 \(a_k \equiv c > 0\) 常数,则 \(\sum a^2 = \infty\) 违反 RM 第二条件。具体看:\(\Delta_{k+1} = (1 - c\nabla g)\Delta_k - c\eta_k\)。设 \(\nabla g \equiv 1\),\(\eta_k\) i.i.d. 零均值方差 \(\sigma^2\)。稳态方差 \(\text{Var}[\Delta_\infty] = c^2 \sigma^2 / (1 - (1-c)^2) = c\sigma^2/(2-c)\) > 0(几何级数和)。即使取 \(c\) 很小,\(\Delta_\infty\) 方差非零,\(w_k\) 不收敛到 \(w^*\) 而是稳态振荡。实操含义:常步长 SGD(包括 deep RL 的 Adam 配 lr=0.001)只能让 \(w_k\) "停在 \(w^*\) 附近一定半径内",不能严格收敛。这不是 bug,是设计选择(L1 §5 已讨论"跟踪 vs 收敛" trade-off)。
Step 2. "\(\sum a = \infty\) 必要" 的反证(教材 (6.6),反例 \(a_k = 1/k^2\))。\(\sum 1/k^2 = \pi^2/6 < \infty\) 违反 RM 第二条件第一项。教材 (6.6):\(|w_1 - w_\infty| = |\sum a_k \tilde g(w_k, \eta_k)| \le \sum a_k |\tilde g| \le b\)(若 \(|\tilde g|\) 有界且 \(\sum a < \infty\))。所以从 \(w_1\) 出发,\(w_\infty\) 离 \(w_1\) 不超过 \(b\)。若 \(|w_1 - w^*| > b\),则 \(w_\infty \ne w^*\)。具体例:\(g(w) = w\)、\(w^* = 0\)、\(w_1 = 100\)、\(\eta = 0\),\(a_k = 1/k^2\) ⇒ \(w_2 = 100 - 1 \cdot 100 = 0\)?计算错:\(w_2 = w_1 - a_1 g(w_1) = 100 - 1 \cdot 100 = 0\) ✓ 但若 \(a_k \tilde g\) 不爆,实际数值 \(b = \sum_{k=2}^\infty (1/k^2)|\tilde g_k|\) 衰减极快,从远处到不了原点。结论:\(\sum a = \infty\) 是"从任意初始点都能到 \(w^*\)" 的必要条件。
Step 3. 用 Dvoretzky 证 RM 的归约 (教材 §6.3.3)。RM 算法 \(w_{k+1} = w_k - a_k[g(w_k) + \eta_k]\)。减去 \(w^*\):\(w_{k+1} - w^* = w_k - w^* - a_k[g(w_k) - g(w^*)] - a_k \eta_k\)(用 \(g(w^*) = 0\))。中值定理:\(g(w_k) - g(w^*) = \nabla g(w'_k)(w_k - w^*)\),\(w'_k \in [w_k, w^*]\)。代入:\(\Delta_{k+1} = [1 - a_k \nabla g(w'_k)] \Delta_k + a_k(-\eta_k) \doteq (1 - \alpha_k) \Delta_k + \beta_k \eta'_k\),其中 \(\alpha_k = a_k \nabla g(w'_k)\)、\(\beta_k = a_k\)、\(\eta'_k = -\eta_k\)。验证 Dvoretzky 三条件:(i) \(\sum\alpha = \sum a_k \nabla g(w'_k) \ge c_1 \sum a_k = \infty\);\(\sum\alpha^2 \le c_2^2 \sum a^2 < \infty\) ✓;(ii) \(\sum\beta^2 = \sum a^2 < \infty\) ✓;(iii) \(\mathbb E[\eta'|\mathcal H] = 0\),\(\mathbb E[\eta'^2|\mathcal H] < \infty\) ✓。Dvoretzky 给出 \(\Delta_k \to 0\) a.s.,即 \(w_k \to w^*\) a.s.。QED。

§4 四种步长调度的收敛比较

本节不用 grid-world(纯抽象数学)。我们用 \(g(w) = w - 1\)、\(w^* = 1\)、\(w_1 = 5\)、\(\eta_k\) i.i.d. \(\mathcal N(0, 1)\),跑四种 \(a_k\):(A) 常 \(a_k \equiv 1\)(违反两条件);(B) 经典 \(a_k = 1/k\)(满足);(C) 慢衰 \(a_k = 1/\sqrt{k}\)(满足 \(\sum a = \infty\),违反 \(\sum a^2 < \infty\));(D) 快衰 \(a_k = 1/k^2\)(违反 \(\sum a = \infty\))。

解读:(A) 常步长:噪声每步注入 \(1 \cdot \eta_k\) 不衰减,稳态方差 \(\sigma^2 / (2-1) = 1\) 量级 ⇒ \(|w - w^*|\) 稳定在 \(\approx 1\),不收敛。(B) 1/k:满足两条件,严格收敛,速率 \(O(1/\sqrt{k})\)(中心极限定理结果)。(C) 1/√k:\(\sum a = \infty\) ✓ 但 \(\sum a^2 = \sum 1/k = \infty\) ✗ ⇒ 后期噪声衰减不够,误差稳定在 \(\sigma\sqrt{a_k}\) 量级,看起来在缓慢减小但不严格 收敛到 0(真实是 \(O(\sigma/k^{1/4})\) 慢得多)。(D) 1/k²:\(\sum a = \pi^2/6 \approx 1.64\) 有限。从 \(w_1 = 5\) 出发,累积可达距离 \(\le 1.64 \cdot |\tilde g| \approx 6.6 \cdot 1 = 6.6\) 看似够,但每步衰减太快,前几步把"累积预算" 消耗大半后,后续 \(a_k\) 太小走不动,稳定在 \(\approx 3.4\) 处陷入 不可达。这就是 RM 第二条件为什么要两条同时成立。

注:本节图为 schematic — 数值取自典型 \(\sigma=1\) 实验的量级估计,严格曲线需要 Monte Carlo 数百次平均。但排序 与稳态值 是严证可推。

§5 常见误解 / 易混概念

很多人以为:"\(\sum a = \infty\) 与 \(\sum a^2 < \infty\) 是孤立两条件,任意一条单独成立都足够。"
实际上:两条件必须同时 成立(§4 (C)/(D) 已演示)。缺第一(\(\sum a < \infty\))⇒ 累积步长有限,远初始点不可达(教材 (6.6))。缺第二(\(\sum a^2 = \infty\))⇒ 后期噪声不衰,稳态振荡(§4 (A))。两条件构成"能到 \(w^*\)" 与"到了能停下来" 的张力 —— 步长不能太小(否则走不到)、不能太大(否则停不住)。\(a_k = c/k^\beta, \beta \in (0.5, 1]\) 是合法窗口;\(\beta=1\) 最经典,\(\beta=2/3\) 等也行。
因为:这是"勘测 vs 收敛" 的本质张力,跨整个 RL 都成立。Ch.5 ε-greedy 的"大 ε 探索 / 小 ε 利用" 是策略层 的同种张力;Ch.6 RM 的步长两条件是更新层 的体现;Ch.7-Ch.10 算法都同时 处理这两层。
很多人以为:"RM 定理只对i.i.d. 噪声 成立,RL 中数据 non-i.i.d.,所以 RM 不适用。"
实际上:RM 第三条件是条件期望:\(\mathbb E[\eta_k|\mathcal H_k] = 0\),允许 non-i.i.d.。\(\eta_k\) 可以依赖 \(w_k, w_{k-1}, \dots, \eta_{k-1}, \dots\),只要在给定历史 下零均值即可。RL 中 \((s, a, r, s')\) 由当前策略生成,确实 non-i.i.d.,但 \(r + \gamma V(s') - V(s)\) 在 \(\mathcal H_k\) 下条件零均值(若 \(V\) 接近 \(V_\pi\)) ⇒ 满足 RM 第三条件。这就是 Ch.7 TD 收敛证明能套 RM 的关键。
因为:RM 框架被 1951 年 Robbins-Monro 设计时已经 考虑了 RL-style non-i.i.d. 场景(虽然 1951 年还没有 RL 一词)。条件期望 \(\mathbb E[\cdot|\mathcal H]\) 是 measure-theoretic probability 的标准工具,处理"带历史的随机过程" 的零均值约束。Theorem 6.3 多变量推广(§6.3.4)进一步把"有限状态空间 + max norm" 加入,恰好是 Q-learning 的设定。
很多人以为:"Dvoretzky 定理是 RM 证明的装饰,直接看 RM 三条件证明就行。"
实际上:Dvoretzky 是更通用 的工具,RM 是它的具体应用。Dvoretzky 处理"\(\Delta_{k+1} = (1-\alpha_k)\Delta_k + \beta_k\eta_k\) 形态" 的所有 随机过程,不限于 RM。Q-learning(Ch.7)、stochastic approximation 在线性 FA(Ch.8)、TD(λ) 等都套这个母定理。教材 §6.3.4 还给了 Theorem 6.3 多变量版,直接服务 Q-learning。所以学 Dvoretzky 一次,所有 RM-style 算法的收敛证明都会读,这是投资回报最高 的数学工具之一。
因为:RM 是"用样本估期望" 这件事的迭代形态;Dvoretzky 是"带噪几何衰减过程" 的统一收敛工具。前者具体,后者抽象。把 RM 归约到 Dvoretzky 之后,Q-learning、TD-Gammon、deep Q-network、policy gradient 的收敛证明都是同一套数学的不同适配。这是 Ch.6 在全书中作为"数学中心" 的核心价值。

§6 自检 + 视频 + 下一节

Q1. 验证 \(a_k = c/k^\beta\) 满足 RM 第二条件当且仅当 \(\beta \in (0.5, 1]\)。\(\beta = 1\) 与 \(\beta = 0.6\) 谁收敛更快?

A. \(\sum 1/k^\beta\) p-series:\(\beta > 1\) 收敛、\(\beta \le 1\) 发散。所以 \(\sum a = \infty\) 要 \(\beta \le 1\);\(\sum a^2 = \sum 1/k^{2\beta} < \infty\) 要 \(2\beta > 1\) 即 \(\beta > 0.5\)。合法窗口 \(\beta \in (0.5, 1]\)。速率:\(\beta = 1\) ⇒ 收敛速率 \(O(1/\sqrt{k})\)(标准 RM 中心极限);\(\beta = 0.6\) ⇒ \(O(1/k^{0.2})\) 慢得多(衰减太慢的代价)。实操:\(\beta = 1\) 是经典最优;\(\beta < 1\) 让"后期步长更小" 衰减误差更慢。但 \(\beta\) 太接近 1 时有限步前期收敛慢(因 \(c/k\) 太小),工程常用 \(a_k = c/(k+k_0)\) 加 warmup 延迟到 \(k_0\) 后再衰。

Q2. 教材 \(g(w) = w^3 - 5\) 例子(L2 Q3 已述)不 满足 RM 第一条件 \(\nabla g \le c_2\),但 Figure 6.3 仍然收敛。如何解释?

A. 第一条件 \(c_1 \le \nabla g \le c_2\) 是充分 不必要 条件 —— 实际收敛域比定理保证的更宽。具体:\(g = w^3 - 5\),\(\nabla g = 3w^2\),在 \(w \in [0, 3]\) 上 \(\nabla g \in [0, 27]\) 有界,只要 \(w_k\) 不跑出 这个区间就行。教材选 \(w_1 = 0\)(adequately selected),配 \(a_k = 1/k\),\(w_k\) 始终留在 \([0, 3]\) 区间内,Theorem 6.1 在局部 仍适用。若选 \(w_1 = 100\),则 \(\tilde g(100) \approx 10^6\),\(w_2 = 100 - 1 \cdot 10^6\) 弹到极远,数值上炸开。实操推论:对不满足全局 Lipschitz 的 \(g\),配合合理初值 + gradient clipping 仍可用 RM —— deep RL 标配技术。

Q3. 把 \(\Delta_{k+1} = (1-\alpha_k)\Delta_k + \beta_k \eta_k\) 中 \(\alpha_k\) 全替换为常数 \(\alpha\)、\(\beta_k = \alpha\)、\(\eta_k\) i.i.d. \(\sigma^2\)。计算稳态方差 \(\text{Var}[\Delta_\infty]\)。这与 §4 (A) 实验值一致吗?

A. 写 \(\Delta_{k+1} = (1-\alpha)\Delta_k - \alpha \eta_k\)(注意 RM 中 \(\eta\) 前是减号)。两端方差(用 \(\Delta\) 与 \(\eta\) 独立、稳态 \(\text{Var}[\Delta_{k+1}]=\text{Var}[\Delta_k]=V\)):\(V = (1-\alpha)^2 V + \alpha^2 \sigma^2\) ⇒ \(V = \alpha^2 \sigma^2 / [1 - (1-\alpha)^2] = \alpha^2 \sigma^2 / [\alpha(2-\alpha)] = \alpha\sigma^2/(2-\alpha)\)。代入 \(\alpha=1, \sigma=1\):\(V = 1/(2-1) = 1\),稳态标准差 \(\sqrt{V} = 1\)。§4 (A) 图上 \(|w_k - w^*|\) 稳定在 \(\approx 1\),完全一致 ✓。含义:常步长 RM 不收敛但有稳态分布,稳态标准差 \(\sigma\sqrt{\alpha/(2-\alpha)}\) —— 这正是 Q-learning 等 deep RL 算法在常 lr 下的"误差稳态" 的来源。

下一节预告

L1-L3 把 RM 框架完整搭好:形态(L2)+ 收敛证明(L3)。L4(教材 §6.4) 把视角切到机器学习 主流话题 —— Stochastic Gradient Descent (SGD)。SGD 解的是 \(\min_w J(w) = \mathbb E_X[f(w, X)]\) 这类期望优化:把"梯度"\(\nabla J(w) = \mathbb E[\nabla_w f(w, X)]\) 用单样本\(\nabla_w f(w_k, x_k)\) 替代,得 \(w_{k+1} = w_k - \alpha_k \nabla_w f(w_k, x_k)\)。关键洞察:这恰好是 RM 在 \(g(w) = \nabla J(w) = \mathbb E[\nabla_w f(w, X)]\) 这个未知函数 上跑的算法!\(\tilde g(w_k, x_k) = \nabla_w f(w_k, x_k) = g(w_k) + (\nabla_w f(w_k, x_k) - g(w_k))\),后者就是零均值噪声 \(\eta_k\)。所以 SGD 直接套 RM 收敛证明(教材 Theorem 6.4)。L4 先把 SGD 的定义 + 与 RM 关系 讲清,后面 L5-L7 看 SGD 的具体例子、性质、与 BGD/MBGD 对比。
